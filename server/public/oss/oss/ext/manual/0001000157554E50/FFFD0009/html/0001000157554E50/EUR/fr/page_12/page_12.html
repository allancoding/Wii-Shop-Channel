Menu principal


Mode libre
Sélectionnez le mode libre si vous voulez faire une partie rapide (bien sûr, vous avez également la possibilité de changer les paramètres du jeu dans ce mode).

Tournoi
Le mode Tournoi est un mode de jeu un peu plus poussé. Le joueur devra réaliser un certain nombre de défis pour atteindre des objectifs bien particuliers et remporter des récompenses, le but ultime étant de devenir un expert du UNO™ !

Multijoueur
Choisissez cette option si vous voulez jouer à UNO™ avec des amis. Vous pouvez jouer avec vos amis sur une seule console Wii ou en ligne avec un ami, n'importe où dans le monde !

Options (icône Engrenage)
Réglez les options du jeu et changez le profil du joueur.
