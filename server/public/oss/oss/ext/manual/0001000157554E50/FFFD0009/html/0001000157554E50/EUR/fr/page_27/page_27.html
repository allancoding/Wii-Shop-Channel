Créer partie


Mondial
Ce mode de jeu vous permet d'héberger une partie en personnalisant les règles et de jouer avec des joueurs du monde entier. Vous devez d'abord choisir les règles et les options avec lesquelles vous voulez jouer, puis un salon de jeu sera créé. Vous pourrez y attendre d'autres joueurs ayant eux aussi choisi ces mêmes règles et options ou inviter vos amis. Vous pouvez aussi commencer à jouer immédiatement. Si tel est le cas, la partie commencera avec des CPU.

Partie entre amis
Ce mode fonctionne comme le mode Mondial, mais vous ne pouvez jouer qu'avec vos amis. Tout comme en mode Mondial, vous devez créer une partie en fonction de vos préférences, puis un salon de jeu s'ouvrira. Vous pourrez y attendre vos amis ou les inviter à jouer avec vous. Vous pourrez également commencer une partie avec des CPU.

Liste d'amis


La liste d'amis est une liste de vos amis ajoutés dans le jeu.

Au début, cette liste sera vide. Vous devrez donc entrer le code ami de chacun de vos amis pour les enregistrer. Cette liste vous permettra de voir les Mii des amis que vous avez ajoutés et qui vous ont également déjà ajouté comme ami.

Un "?" vous indique qu'un ami n'a pas encore rentré votre code ami, tandis qu'un smiley vous indique qu'un ami est actuellement en train d'héberger une partie. Vous pouvez pointer un Mii avec la télécommande Wii pour voir un code ami.

Vous pouvez utiliser l'icône "Supprimer" à côté du nom de votre ami pour le supprimer de votre liste.

Code ami

Sur l'écran de saisie du code ami, vous devez entrer le bon code en utilisant le clavier numérique à l'écran. Assurez-vous d'entrer le bon code ou votre ami ne sera pas enregistré dans votre liste d'amis.
