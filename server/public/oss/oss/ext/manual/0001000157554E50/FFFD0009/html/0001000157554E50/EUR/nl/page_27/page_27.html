Create Play (Spel creëren)


Worldwide (Wereldwijd)

In deze modus kun je een spel hosten met behulp van aangepaste instellingen van spelers uit de hele wereld. Eerst kies je de regels en spelopties, vervolgens wordt er een lobby gecreëerd. Je kunt wachten op spelers die willen spelen volgens jouw regels of je kunt vrienden uitnodigen om mee te spelen. Je kunt ook direct beginnen met spelen. Als je hiervoor kiest, dat start het spel met computertegenstanders (CPU opponents).

Friends Game (Spel met vrienden)

Deze modus is net als de ‘Worldwide’ (Wereldwijd)-modus, behalve dan dat je hier alleen met vrienden kunt spelen. Net als de in ‘Worldwide’ (Wereldwijd)-modus moet je een spel creëren met jouw voorkeuren en vervolgens wordt er een lobby gecreëerd. Je kunt wachten op vrienden die willen meespelen of je kunt vrienden. Je kunt ook het spel starten met computertegenstanders (CPU opponents)

Friend Roster (Vriendenoverzicht)


Het Vriendenoverzicht is de lijst met vrienden die je hebt toegevoegd aan het spel.

Eerst zal je overzicht leeg zijn, dus ga je naar het ‘Friend Code’ (Vriendencode) invoerscherm om een vriend(in) te registreren. Het overzicht toont de Mii-personages van alle vrienden die je hebt toegevoegd en die jou hebben toegevoegd.

Een ‘?’ geeft aan dat je vriend(in) jouw vriendencode nog niet heeft ingevoerd en een ‘smiley’ geeft aan welke vriend(in) op dat moment een spel aan het hosten is. Richt met de Wii-afstandsbediening op een Mii-personage om je vriendencode te bekijken.

Gebruik het ‘delete’ (verwijderen) –pictogram naast het vakje van je vriend(in) om die vriend(in) te verwijderen uit het overzicht.

Friend Code (Vriendencode)

In het ‘Friend Code’ (Vriendencode) invoerscherm, moet je de juiste code invoeren via het numeriek toetsenbord op het scherm. Zorg dat je de code goed overneemt anders wordt je vriend(in) niet geregistreerd in je overzicht.
