5
操作方法

このゲームでは、4種類の操作スタイルで操作できます。この取扱説明書では、横持ちスタイルの操作で説明を行います。

● 横持ちスタイル
Wiiリモコンを横に持って使います。

● クラシック・スタイル
Wiiリモコンにクラシックコントローラを接続すると、クラシックコントローラでプレイできます。

クラシックコントローラの操作
／
カーソルの移動、キャラの移動
決定ボタン、ブロックを掘る
キャンセル
（ゲーム中に）ポーズメニューの表示
HOMEボタンメニュー

● ヌンチャク・スタイル
Wiiリモコンにヌンチャクを接続し、2つを組み合わせて操作するスタイルです。

Wiiリモコンとヌンチャクの操作
カーソルの移動、キャラの移動
決定ボタン、ブロックを掘る
キャンセル
（ゲーム中に）ポーズメニューの表示
HOMEボタンメニュー

● ゲームキューブコントローラ
ゲームキューブコントローラも使用できます。

※ゲームキューブコントローラの振動機能は使用しません。

ゲームキューブコントローラの操作
／

カーソルの移動、キャラの移動
決定ボタン、ブロックを掘る
キャンセル
START/PAUSE
ボタン

（ゲーム中に）

ポーズメニューの表示
