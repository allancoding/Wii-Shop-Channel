Game Setup
Stage Select Screen
You can select and decide a stage to play. Each stage's BEST TIME is also displayed.

Number of Players Screen
You can decide the number of players to play with. Each player's Wii Remote must be connected to the Wii console in order to play.

Stage Details Screen
Here you can read about a person's problem and their request for help. When you are ready to play, everyone must press the onscreen 'START' button.
