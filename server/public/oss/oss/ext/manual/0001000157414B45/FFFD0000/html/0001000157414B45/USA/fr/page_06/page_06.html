Vous jouez à ce jeu pour la première fois

 

Dans ce jeu, vous incarnez un détective en collaboration avec l’agence de détectives ACME. La mission d’ACME est de débusquer les bandits que sont Carmen Sandiego et ses sous-fifres de V.I.L.E.

 

Votre travail est d’enquêter sur des crimes commis par V.I.L.E. et de réunir assez de preuves pour arrêter Carmen Sandiego et ses bandits.

 

Avant de commencer votre première partie, vous devez créer votre personnage de jeu.

 

 

  1. Pour cela, placez le curseur sur le bouton Gestion de l'agent (Agent Manager) dans le menu du jeu, puis appuyez sur .

     

  2. Sur l'écran Gestion de l'agent (Agent Manager), placez le curseur sur Nouveau (New) et appuyez sur .

     

  3. Sur l'écran Création d'agent (Agent Creation), placez le curseur sur le champ Saisir votre nom (Enter Your Name) et appuyez sur .

     

  4. Tapez votre nom en utilisant le clavier virtuel. Placez le curseur sur la coche verte et appuyez sur lorsque vous avez terminé.

     

  5. Choisissez Homme (Male) ou Femme (Female). Faites défiler les personnages jusqu'au personnage désiré. Placez ensuite le curseur sur Créer (Create) et appuyez sur .

 

Une fois que vous avez créé votre personnage, vous pouvez éditer votre nom avec le bouton Éditer l'agent (Edit Agent).

 

Utilisez le menu Gestion de l'agent (Agent Manager) pour créer, supprimer ou jouer avec d'autres personnages.

 

Maintenant que vous avez créé votre personnage de jeu, vous pouvez utiliser toutes les options dans le menu du jeu. Vous voilà prêt à jouer!
