Controles

 

 

Para usar el Wii Remote para jugar, apunta a la pantalla.

 

Puedes usar para:

  • Caminar a un lugar específico. Coloca el cursor sobre el lugar donde quieres ir y luego oprime .
  • Escuchar lo que tienen que decir los otros personajes.
  • Recoger objetos y Hint Coins (Monedas para sugerencias).
  • Contestar el ACME Phone (Teléfono ACME).
