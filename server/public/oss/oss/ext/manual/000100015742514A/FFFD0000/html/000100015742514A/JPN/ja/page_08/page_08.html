ゲームの始めかた
タイトル画面で決定ボタンを押すと、ゲームセレクト画面が表示されます。ポインターまたはなどでモードを選び、決定ボタンで決定してください。
WiiがWi-Fiコネクションに接続されていないと、
Wi-Fiランキングに登録したり、Wi-Fiランキングを見ることはできません。

※初回起動時のみ、Wi-Fiランキングの登録地域を設定する必要があります。選択肢の中からお住まいの都道府県を選択してください。

モード
Game Start
2分および5分でスコアを競います。
Fast Shot Mode
ボタン連打の早さを競います。
Records
ハイスコアおよびWi-Fiランキングを見ることができます。
Option
オート連射機能の設定および
Wi-Fiランキングの登録地域を変更できます。
Credit
StarSoldierRの制作スタッフを見ることができます。
