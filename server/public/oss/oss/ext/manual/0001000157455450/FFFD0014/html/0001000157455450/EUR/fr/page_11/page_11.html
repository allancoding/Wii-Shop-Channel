Règles de base

Les joueurs agissent à tour de rôle pour avancer vers la case d’arrivée.

 

Les actions pendant votre tour

Suivez les étapes ci-dessous pour agir lorsque votre tour arrive.

 

Note: vous pouvez appuyer sur pour faire défiler les messages plus rapidement lorsque c’est à l’ordinateur de jouer. Seul peut être utilisé lorsque l’ordinateur joue.

 

Choisir une carte ou une pièce

Tout d’abord, choisissez d’obtenir une carte ou une pièce selon ce qui vous semble être le mieux dans la situation donnée.

 

Sélectionner une carte à utiliser

Pointez la carte que vous désirez utiliser et appuyez sur . Les cartes se divisent en deux types: les cartes action et les cartes déplacement. Vous pouvez utiliser autant de cartes action que vous voulez pendant votre tour, mais vous ne pouvez utiliser qu’une seule carte déplacement, qui est celle vous permettant d’avancer (voir la section 14 "Types de cartes"). Votre tour se termine après l’utilisation d’une carte déplacement.

 

Note: vous pouvez aussi terminer votre tour sans utiliser de carte déplacement en appuyant sur .

 

Activer une case

Chaque case produit un certain effet lorsqu’un joueur s’arrête dessus. Vous pourrez voir la teneur de cet effet après qu’un joueur est passé sur une case donnée pour la première fois. Les cases ne s’activent pas si vous atterrissez dessus en conséquence d’une activation du grimoire (voir la section 15 "Types de cases").
