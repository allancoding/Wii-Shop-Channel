Ending the Game

To end the game

Press to display the HOME Menu, then select Wii MENU.

 

To suspend the game

If you’ve set the number of rounds to two or more, you can suspend the game after clearing a map. To resume from suspended data, select CONTINUE on the Main Menu Screen.

 

Note: You cannot suspend the game while still on a map.

 

To suspend the game: after clearing the map, point at the arrow on the bottom left of the screen and press .

Point at YES and press to suspend the game and return to the Main Menu Screen.

To delete player data

You can delete all data for PictureBook Games: A Pop-Up Adventure by deleting its file in the Wii System Memory. For details, refer to the Wii Settings and Data Management section of the Wii Operations Manual - Channels and Settings. You can also delete individual player data from the Select Players Screen (see “9. Starting a Game”).

 

Note: Deleted data cannot be recovered. Please be careful when deleting data.

About Saving

Performing any of the following operations while using the save function may result in loss of save data or damage to the Wii console:

● Pressing the POWER Button on the Wii Remote or the POWER Button or RESET on the Wii console while data is being saved.

● Opening the HOME Menu and selecting Wii MENU or RESET while data is being saved.

● Disconnecting the Wii Power Supply from the DC Input Connector or removing the AC Adapter from the mains.

Lost save data cannot be recovered, so please be careful.
