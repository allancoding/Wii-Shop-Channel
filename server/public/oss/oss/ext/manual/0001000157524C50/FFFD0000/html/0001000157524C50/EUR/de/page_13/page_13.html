Herausforderungen

Im Herausforderungs-Modus können 3 Sets mit jeweils 7 Herausforderungen gespielt werden.


Am Anfang können nur die 7 einfachsten Herausforderungen gespielt werden. Sobald du alle Herausforderungen in einem Set geschafft hast, wird das nächste freigeschalten.

Arten von Herausforderungen
Es gibt eine Vielzahl von verschiedenen Herausforderungen mit jeweils unterschiedlichen Zielvorgaben. In einer Sprung-Herausforderung musst du eine bestimmte Sprungweite erreichen. In Zeit-Herausforderungen hingegen musst du innerhalb des Zeitlimits entweder das Ziel erreichen oder mehrere Runden fahren.
