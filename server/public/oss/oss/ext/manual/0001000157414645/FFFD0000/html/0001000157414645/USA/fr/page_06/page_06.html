Jouer comme un expert

Si la couleur de la porte est assortie à celle de l´avion, vous recevez un bonus Combo!

Vous gagnez un bonus Combo en faisant décoller ou atterrir plusieurs avions sur la même piste.

Utilisez l'escale pour rendre les avions plus confortable pendant qu'ils attendent.

Vous pouvez mettre en attente plusieurs actions sur un avion e puis annuler ces actions, avec le bouton B.

Si vous assignez un avion à un emplacement occupé, il attendra que celui-ci se libère.

Les points gagnés peuvent servir aux achats dans le magasin. Plus de point = plus de pouvoir d´achat!
