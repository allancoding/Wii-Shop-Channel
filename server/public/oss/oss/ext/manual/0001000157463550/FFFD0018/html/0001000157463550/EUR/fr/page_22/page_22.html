Téléchargements
●Téléchargements
Vous pouvez acheter des contenus additionnel pour « Family Card Games » à l’intérieur même du jeu. Pour acheter des contenus additionnel, vous aurez besoin de Wii Points. (*Frais supplémentaires)

●Acheter des contenus additionelss (re-télécharger)
Sélectionnez le niveau que vous souhaitez acheter ou re-télécharger, et sélectionnez soit le bouton « Télécharger », soit le bouton « Re-télécharger ». Une fois que vous avez acheté un niveau, vous pourrez le télécharger de nouveau gratuitement, même si vous le supprimez.

Si vous n’avez pas assez de Wii Points, veuillez vous la Chaîne Boutique Wii ou le site internet officiel de Nintendo et suivez les instructions pour apprendre comment acheter des contenus additionnel.

●Supprimer des contenus téléchargés
Vous pouvez supprimer les contenus additionnel que vous avez téléchargés sur la console Wii. Les contenus supprimés ici peuvent être re-téléchargés gratuitement.

●contenus additionnel disponibles
Wii Points nécessaires : 100 Wii Points
※Pour chaque paquet de cartes

Nombre de blocs: 2 blocs

Description du contenu:
De nouveaux jeux de cartes peuvent être ajoutés à « Family Card Games ». Il y en a plein de différents ! Achetez ceux qui vous plaisent !


※Avertissement
Les contenus additionnel sont des contenus additionelss pour « Family Card Games ». Vous ne pouvez pas jouer avec uniquement les contenus additionnel.
