Steuerung
Spielanleitung
●Zeiger
Mit dem Zeiger auf der Wii-Fernbedienung bewegst Du den Cursor.
●A-Knopf
Der A-Knopf dient zur Auswahl.
●B-Knopf
Drücke den B-Knopf, um zum vorherigen Bildschirm zurückzukehren.

Steuerung bei Daifugo
＜Vertikale Einstellungen der Wii-Fernbedienung＞
●Zeiger
Mit dem Zeiger auf der Wii-Fernbedienung bewegst Du den Cursor.
●A-Knopf
Zeige mit dem Cursor auf die Karten und drücke den A-Knopf, um die gewünschte Karte an-/abzuwählen.
●B-Knopf
Drücke den B-Knopf, um die gewünschte Karte abzuwählen.
●- Knopf
Zeigt das Pausenmenü an.
Bei einem Nintendo WFC Kampf können Name und Titel geändert werden.

＜Horizontale Einstellungen der Wii-Fernbedienung＞
●Steuerkreuz
Bewegt den Cursor.
●1-Knopf
Wählt die ausgewählte Karte ab.
●2-Knopf
Bewege den Cursor auf eine Karte und drücke den 2-Knopf, um die Karte an-/abzuwählen.
●- Knopf
Das Pausenmenü wird angezeigt.
Bei einem Nintendo WFC Kampf können Name und Titel geändert werden.

※Einstellungen, wie man die Wii-Fernbedienung hält, können durch "Einstellungen" im "Hauptmenü" eingestellt werden.

Steuerung bei Speed
●Zeiger
Mit dem Zeiger auf der Wii-Fernbedienung bewegst Du den Cursor.
●A-Knopf
Platziere den Cursor auf den Karten und drücke den A-Knopf, um die Karte zu nehmen. Drücke noch einmal den A-Knopf, um die Karte loszulassen.
●- Knopf
Zeigt das Pausenmenü an.
Einstellungen bei Memory
●Zeiger
Bewege den Cursor mit dem Zeiger der Wii-Fernbedienung.
●A-Knopf
Bewege Deinen Zeiger auf die Karte und drücke den A-Knopf, um die Karte umzudrehen.
●- Knopf
Zeigt das Pausenmenü an.
