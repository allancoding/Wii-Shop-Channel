User Select
In the "Main Menu", if you select "Nintendo WFC Daifugo" it will take you to the "User Select" screen.

Select User
If you select a user that has already been registered, it will move on to the "Nintendo WFC Menu".

Registering a User
If you are going to register a new user, click "Not registered yet". Enter their name and select a character and complete registration. When the user is registered, it will move on to the "Nintendo WFC Menu".

※You can register up to 4 players.

※If you are going to delete a user, please do so from "User Settings".
