8
Commandes du jeu

SAUTER - appuyez sur le bouton 2 pour sauter.
Maintenez le bouton 2 pour sauter à plusieurs reprises.

PLANER - maintenez haut sur la croix directionnelle pour planer.
Vous pouvez utiliser le planeur dans les airs uniquement.

VITESSE - maintenez droite sur la croix directionnelle pour courir.
Sautez en maintenant droite sur la croix directionnelle pour effectuer un "saut en longueur".

BERSERK - maintenez bas sur la croix directionnelle pour vous faire un chemin à travers les obstacles.
Dans les airs, maintenez bas sur la croix directionnelle pour tomber rapidement.
Sautez en maintenant bas sur la croix directionnelle pour effectuer un "mini saut".

REMONTER LE TEMPS - appuyez à gauche sur la croix directionnelle en passant devant l'horloge pour remonter le temps.
