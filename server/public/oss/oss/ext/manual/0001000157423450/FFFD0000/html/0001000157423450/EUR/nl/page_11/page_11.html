Besturing in spelmenu’s
Wii-afstandsbediening:
Richt met de Wii-afstandsbediening op het pictogram dat je wilt selecteren.
A-knop: Selecteren.
Plusknop: Menu sluiten
B-knop: Menu sluiten
HOME: Opent het HOME-menu.
Wii Zapper:
Richt met de Wii Zapper op het pictogram dat je wilt selecteren.
A-knop: Selecteren.
Plusknop: Menu sluiten
Z-knop: Menu sluiten
HOME: Opent het HOME-menu.
