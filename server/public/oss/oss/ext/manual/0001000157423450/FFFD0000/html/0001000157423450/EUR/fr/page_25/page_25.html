Epreuves spéciales

Le géant Indien ⇨ Il vous attrapera et vous
secouera comme un prunier.

 

Mode de commande télécommande Wii  :

Secouez la télécommande Wii pour vous en défaire plus rapidement.

 

Mode de commande Wii Zapper  :

S’il vous attrape, faites pivoter le stick directionnel pour stopper son attaque plus rapidement.

 

 

 

Le savant fou ⇨ S’il parvient à vous approcher, il va vous enfumer.

 

Mode de commande télécommande Wii  :

Secouez la télécommande Wii pour faire disparaître la fumée plus vite.

 

Mode de commande Wii Zapper  :

S’il vous atteint, faites pivoter le stick directionnel pour stopper son attaque plus rapidement.

 

 

 

Le Fossoyeur ⇨ Ne le laissez pas vous approcher
ou il vous assommera à coups de pelles.
