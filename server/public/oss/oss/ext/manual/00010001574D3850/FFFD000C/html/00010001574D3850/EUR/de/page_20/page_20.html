Speichern
Dieses Spiel verfügt über eine automatische Speicher-Funktion: Daten werden automatisch gespeichert, wenn ein Kampf beendet oder Einstellungen geändert wurden. Wenn du Daten löschen möchtest, kannst du das auf dem Datenverwaltungsbildschirm des Wii-Menüs tun. Achtung: Nach dem Löschen sind die Daten unwiederbringlich verloren! Du kannst die Daten nicht auf eine SD Card kopieren.

Speicherfunktion

Es kann zum Verlust von Speicherdaten kommen, wenn während des Speichervorgangs eine der folgenden Aktionen ausgeführt wird. Hinweis: Speicherdaten, die auf diese Weise verloren gehen, können nicht wiederhergestellt werden.

● Wenn der Power-Schalter oder der RESET-Knopf an der Wii-Konsole oder auf der Wii-Fernbedienung betätigt wird, während Daten gespeichert werden.

● Wenn im HOME-Menü die Optionen „Wii-Menü“ oder „Reset“ ausgewählt werden, während Daten gespeichert werden.

● Wenn der Netzstecker aus der Steckdose gezogen wird, während die Wii-Konsole eingeschaltet ist.
