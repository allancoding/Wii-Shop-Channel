Controls
Here is a basic overview of controls. Instructions on how to use each button will be displayed at various Menu screens, so use them accordingly.

 

Using a Wii Remote
Hold the Wii Remote sideways.

Shaker Items

Shaker Item effects are triggered by shaking the Wii Remote up and down.

 

Using a Nintendo GameCube Controller
The Nintendo GameCube Controller's Rumble Feature will not work in this game.
