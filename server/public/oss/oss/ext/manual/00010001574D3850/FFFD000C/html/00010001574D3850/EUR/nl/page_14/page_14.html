Vriend instellen
Als je Vriend instellen kiest, krijg je toegang tot de volgende twee vriendencodemenu's.
Vriendencode Kies deze optie om je eigen vriendencode te bekijken. Je vriendencode wordt alleen weergegeven als je verbinding hebt met Nintendo Wi-Fi Connection. Selecteer Wi-Fi strijd om verbinding te maken.
Vriendenschema Je kunt lege slots kiezen om vriendencodes die je vrienden je hebben doorgegeven te registreren.

Wat is een "vriendencode"?

Een "vriendencode" is een 12 cijferige code die automatisch wordt gemaakt als je verbinding maakt met Nintendo Wi-Fi Connection. Wissel vriendencodes uit met mensen ver weg en registreer ze in je Vriendenschema. Zo kun je een Wi-Fi strijd met deze mensen aangaan.
