Sobre Batalla Wi-Fi
La Batalla Wi-Fi te permite jugar con amigos a distancia mediante la Conexión Wi-Fi de Nintendo.

¿Qué es la Conexión Wi-Fi de Nintendo?

La Conexión Wi-Fi de Nintendo es un modo fácil, seguro y gratis de jugar con gente de todas partes por internet gracias a Nintendo. Es el primer sistema de este tipo. Usa tu conexión inalámbrica o LAN para combatir, jugar aventuras o aprender de otras culturas con usuarios de todo el mundo.

 

Respeta a los demás en las Batallas Wi-Fi

● Cuando uses la conexión Wi-Fi de Nintendo durante el combate, no apagues la consola Wii, no la desconectes intencionadamente ni causes problemas a otros jugadores.

● La Conexión Wi-Fi de Nintendo puede denegar el acceso a usuarios con apodos que resulten ofensivos. Los apodos se cambian en Información de usuario.

 

Observaciones sobre la Batalla Wi-Fi

● Si hay varias consolas conectadas a una única conexión inalámbrica, esta última puede ser lenta o inestable.

● Si juegas una Batalla Wi-Fi con una conexión a internet pueden surgir problemas como lapsos de conexión, caminar sobre bombas u otros efectos raros. Son solo problemas de pantalla, y el juego no se ve afectado si ocurren.

● La Batalla Wi-Fi con la Conexión Wi-Fi de Nintendo pueden jugarla hasta 8 personas. Pueden participar hasta 4 jugadores por consola Wii, y ningún jugador IA.

 * Este software también puede usarse con "Bomberman" para Batallas Wi-Fi, aún sin comercializar.

 * “Bomberman" para Wii es compatible con WiiWare Bomberman Blast para Batallas Wi-Fi. Ten en cuenta que en la actualidad (Otoño de 2008), “Bomberman” para Wii solo está disponible en ciertos países.
“Bomberman” para Wii tiene más Poses de acción disponibles; es posible que encuentres algunas que no pueden usarse con la versión WiiWare.
