User Information
Users can be registered here for Wi-Fi battles, and battle records can be checked.

 

Users
Select an empty slot here to register a new user, or change the settings for a user already registered.
Check/Edit Check or edit data for registered users.
Delete Delete selected user data.

 

User Data
When selecting Check/Edit at the Users screen, the following menus are displayed.
Character This can be selected once the Mii capability is unlocked. Select whether or not you will use a Mii.
Nickname Create or change the user's nickname.
Action Pose Set an Action Pose to be triggered during battle by pressing .
Action Message Action Messages can be set here. These messages can only be displayed during a Friend Battle in Wi-Fi Battle mode. Four different messages can be set in total, and be displayed in three situations: during battle, when winning a battle, or when losing a battle. Press during the game to display an Action Message, or hold down and press to change to a different message.
Battle Records Check your win/loss record in Normal Battle or Wi-Fi Battle modes.
Leaderboard Check your rank in Wi-Fi Battle mode.
