Configuración Batalla Wi-Fi
Modo de iniciar una Batalla Wi-Fi.

 

Elige usuario
Selecciona un usuario para la Batalla Wi-Fi.

 

Menús
Selecciona "Batalla Wi-Fi" en la pantalla de menú.

 

Menús Wi-Fi
Selecciona "Batalla amigos" o "Comb. clasif.".

Elige el número de jugadores.

Unirse solo Aquí seleccionas 1 jugador para la Batalla Wi-Fi.
Unirse multi. Aquí seleccionas 2-4 jugadores para la Batalla Wi-Fi.

 

Jugadores
Selecciona “Unirse multi.” y pulsa derecha o izquierda para seleccionar jugadores a los que unirte. El número de usuarios que juegan solo hace falta registrarlo para "Batalla amigos".

 

 

● Jugar a Batalla amigos
Personajes
Al seleccionar "Unirse multi.", elige el usuario(s) con el que quieres jugar de entre los registrados.
Salas
Aparecerá una lista de salas de batallas. Selecciona una sala, o selecciona la opción Crear sala e invita a otros a jugar. En la sala de batalla
amarilla hay una batalla en curso. Puede pasar un tiempo antes de poder unirte.
El anfitrión puede decidir si reclutar miembros o no en una batalla con amigos. Cuando entra un nuevo miembro, hay que cancelar la batalla actual para que empiece una nueva.

 

● Jugar Combate clasif.
Elegir zona
Selecciona la localización de los jugadores con los que quieres combatir.
Continental Juega una batalla Wi-Fi con alguien de Europa.
Mundial Juega una Batalla Wi-Fi con alguien de algún lugar del mundo. El color de la ventana Estado durante una Batalla Wi-Fi será diferente en función del lugar de procedencia del jugador participante. Será blanca si es de Japón, roja para Norteamérica, y azul para Europa.

 

Puede pasar un tiempo hasta que puedas unirte. Juega una partida de "Esquivar bombas" mientras esperas. Deberás esquivar las explosiones para sobrevivir.

 

 

Reglas / Escen.
Primero aparecerá la pantalla de Reglas. Si algunos miembros han seleccionado reglas diferentes, se elegirá de entre las reglas seleccionadas. A continuación, aparece la pantalla de Fases, así que, de igual forma, selecciona una fase para jugar en el tiempo permitido.

● Jugar Batalla amigos
En la pantalla Salas, los que seleccionen Crear sala, pueden ajustar la configuración y hacer una comprobación final.

● Jugar Combate clasif.
La configuración se fija como sigue.

● Tipo combate: Puntuación

● Rondas: 2

● Tiempo: 2 min.

● Punto inicial: Aleatorio

● Bloques de presión: SÍ

● Venganza: NO

● Paneles objeto: Ignífugo

● Calaveras: SÍ

 



 

Resultados y ganador
La pantalla de resultados aparece cuando la batalla ha acabado. Si juegas un combate clasificatorio,
aparecen niveles de Bomber. Cada jugador puede ver las puntuaciones detalladas. Después de jugar el número de rondas fijado, se anuncia el ganador.
Salir Salir modo Batalla Wi-Fi.
Seguir Volver a la pantalla de Reglas y volver a intentar la batalla.
