Friend Setup
When selecting Friend Setup, the following two friend code menus can be accessed.
Friend Code Select this option to view your own friend code. Your friend code will only be displayed when connected to Nintendo Wi-Fi Connection. Make sure to select Wi-Fi Battle to connect.
Friend Roster You can select empty slots here to register friend codes taught to you by your friends.

What is a "friend code"?

A "friend code" is a 12-digit code that is automatically created when connecting to Nintendo Wi-Fi Connection. Exchange friend codes with people who live far away, then register them at the Friend Setup menu. This will allow you to play those people in Wi-Fi battles.
