Wi-Fi Battle Settings
A Wi-Fi battle can be started as follows.

 

Select User
Select a user to play in the Wi-Fi battle.

 

Menus
Select "Wi-Fi Battle" from the Menus screen.

 

Wi-Fi Menus
Select "Friend Battle" or "Ranked Battle."

Then choose the number of players.

Single Entry Select this to enter 1 player for a Wi-Fi battle.
Multi Entry Select this to enter 2-4 players for a Wi-Fi battle.

 

Players
When selecting "Multi Entry," press right or left to select players to join. The number of users playing only have to be registered for "Friend Battle."

 

 

‚óè Playing a Friend Battle
Characters
When selecting "Multi Entry," choose the user(s) you want to play with from among those registered.
Rooms
A list of battle rooms will be displayed. Select one from among them, or select the Create Room option and invite guests to play.
A yellow battle room has a battle in progress, and it might be awhile before you can join.
The host of a Friend Battle decides whether or not to recruit additional members. When additional members have joined, the current battle must be cancelled for a new one to start.

 

‚óè Playing Ranked Battle
Select Area
Select the location of players you want to battle.
Continental Play a Wi-Fi battle with someone in Europe
World Play a Wi-Fi battle with someone anywhere in the world. The color of each Status Window shown during a Wi-Fi battle will differ according to what part of the world a player is participating from. White is used for Japan, red for North America, and blue for Europe.

 

It might be awhile before you can join. Please enjoy a game of "Dodge Bomb" while waiting, in which you must dodge bomb blasts to survive.

 

 

Rules / Stages
First the Rules screen will be displayed. If any members have selected a different set of rules, one set will be chosen from among them. Next the Stages screen will be displayed, so in the same way, select a stage you want to play in before time runs out.

‚óè Playing a Friend Battle
At the Rooms screen, players who have selected Create Room can adjust Settings and perform a Final Check.

Results & Winner Announced
The Results screen is displayed once a battle has ended. When playing a Ranked Battle, Bomber Grades are displayed. Each player can see a detailed run-down of the scores at this time. Once the designated number of rounds have all been played, the final winner is announced.
