Controles

Para avanzar, pulsa el Botón A.

Para frenar, pulsa el Botón Z.

Para utilizar una bonificación, pulsa el Botón B.

Para derrapar, inclina el Nunchuk hacia la misma dirección del giro.

Para saltar, mueve rápidamente el Wii Remote hacia arriba o pulsa el Botón C.

Para detener el juego, pulsa el Botón -.

Para dirigir tus ataques, utiliza la palanca de control del Nunchuk (arriba/abajo).

Para dirigir el carro, utiliza la palanca de control del Nunchuk (derecha/izquierda).

Para utilizar la vista en primera persona, pulsa arriba en la cruz de control.

Para utilizar la vista posterior, pulsa abajo en la cruz de control.
