Mode centre aquatique
Vous pouvez laisser ou prendre un poisson, ou voir la base de données des poissons.
Sélectionner un poisson

Sélectionnez le poisson que vous désirez mettre dans l'aquarium. Vous ne pouvez pas mettre de poisson si l'aquarium est plein ou trop petit.
Vous ne pouvez pas sélectionner un poisson "???".
Vous obtenez de nouveaux poissons au cours du jeu.

Vous pouvez aussi obtenir des poissons en acquérant du contenu additionnel.

La jauge de poids se remplit chaque fois que vous ajoutez un poisson. Lorsqu'elle est pleine, vous ne pouvez plus ajouter de poisson. Vous devez donc retirer un poisson si vous désirez en ajouter un nouveau.
Sortir un poisson

Retirez un poisson pour le renvoyer au centre aquatique. Leur croissance sera remise par défaut.

Base de données des poissons
Regardez les caractéristiques et les informations des différents poissons.
Vous ne pouvez pas consulter les informations sur les poissons "???".
