Menu
To access the various customization options for your Rabbid, point at the Menu icon on the main screen and press the A Button.
Tools
In this part of the lab, you'll find a vise that you can use to make your Rabbid's head thinner or flatter. Check the tools page of the Online Manual to understand how to use it better.
Accessories
Here you'll find two accessories in seasonal colors that you can place on your Rabbid's head. Check the accessories page of the Online Manual if you need more information about how to use them.
Painting
This is where things start getting serious! Using the many different painting tools available, learn how to turn your Rabbid into a unique, colorful character. Let your imagination run wild! Check out the Painting section of this Online Manual and learn how to master all the possibilities that this module has to offer!
