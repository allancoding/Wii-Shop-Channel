◇ラッキーパズルとは？◇
長方形を７つに分割したピースを動かして、シルエットの形と一致するようにはめこむ……基本ルールはわずかこれだけ！ピースを組み合わせて、シルエットの形を見つける過程の中で、柔軟な思考力と図形を見る目が試されます。シンプルながらも、挑戦するとつい誰もが夢中になってしまう魅力があるのです。

歴史あるパズル
「シルエットパズル」とも呼ばれるこの分野のパズルの歴史は古く、原型となるものは古代ギリシャの時代に生まれたといいます。日本では１８世紀に記された書物の中に「清少納言の知恵の板」という名称で紹介されています。遊ぶ人たちによって次々と新しいシルエットが作られ世界中の人々に楽しまれた、歴史あるパズルなのです。 「ラッキーパズル」の愛称でも親しまれている「ハメコミ　LUCKY PUZZLE®」は数ある「シルエットパズル」の中でも日本でとても人気のあるパズルです。
