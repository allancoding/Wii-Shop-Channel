Game Modes


Select game mode:
You can select from 3 different game modes.



Duel Mode: Be the first to reach the selected number of wins! The difficulty level can be selected - affects attack time and computer strength.



Time Mode: Knockout more chicks than your opponent within a set amount of time! Knocked out chicks are replaced. The difficulty level can be selected - affects attack time and computer strength.



Pro Modus: Keep going for as long as you can! Knocked out chicks will only be replaced on your opponent's side. The longer the battle lasts, the stronger your computer opponent will become. Difficulty level can't be selected.

One of 3 arenas (City, Ship and Haunted wood) can be selected for each mode or one can allow it to be selected at random.
