  「チキンバトル！チクチクブーン」
は、5匹のチキンにより構成される
チームを率いて、
ライバルのチキンチームと
バトルするゲームです。

アタックやディフェンスを駆使して、
ライバルチームのライフを先に０にすれば勝ち、
逆にライバルチームより先に、
ライフが０になってしまうと負けです。

バトル中は、Wiiリモコンのポインタを使い、
フィールド上やお絵かきエリアに、
絵を描くことができます。

お絵かきエリアに絵を描くことによって、
ライバルに【アタック】（攻撃）することができ、
また、フィールド上にラインを描くことによって、
ライバルのアタックを【ディフェンス】（防御）
したり、さらに、ライバルのディフェンスを
【ブレイク】（妨害）することができます。

瞬時に今の状況を判断して、
アタックをしかけたり、ディフェンスしたり、
最適の行動をとることが、勝利につながります。
