Démarrer une partie

À l'écran-titre, appuyer sur un bouton pour afficher les options suivantes.

Si vous jouez pour la première fois, sélectionnez "Nouvelle partie". Vous devrez sélectionner un emplacement de sauvegarde et décider si vous souhaitez ou non commencer par le didacticiel.

Si vous disposez déjà de fichiers de sauvegarde, sélectionnez "Charger partie" puis choisissez une sauvegarde pour reprendre la partie là où vous l'aviez laissée.

Sélectionner "Acheter contenu supplémentaire" pour télécharger des contenus supplémentaires (voir page 14 pour plus de détails).
