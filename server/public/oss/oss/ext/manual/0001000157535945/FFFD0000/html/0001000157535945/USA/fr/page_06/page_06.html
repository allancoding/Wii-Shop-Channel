Modes de jeu

Aventure (Adventure)
Ce mode de jeu standard amène Turbo sur 50 parcours de distribution répartis dans dix galaxies.

Aventure en coopération
(Co-Op Adventure)
Ce mode vous permet de jouer au mode Aventure avec un ami. Lorsqu'un des compagnons meurt, l'autre a la possibilité de compléter le parcours de distribution.

Course contre la montre (Time Trial)
Améliorez vos compétentes et testez votre vitesse dans une course contre la montre. Choisissez n'importe quel parcours auquel vous avez déjà joué et tentez de battre votre meilleur temps.

Mode Défi (Challenge Mode)
Le mode Défi est rempli d'un nombre infini de niveaux générés aléatoirement et il vous permet de choisir un niveau de difficulté et une vitesse de course. Si vous augmentez le niveau de difficulté, l'ensemble des parcours de distribution sera plus long et il y aura plus d'ennemis et d'obstacles.

Multijoueur (Multiplayer)
Ce mode vous permet de jour avec jusqu'à trois amis. Comme pour le mode défi, le mode multijoueur vous permet de sélectionner un niveau de difficulté et une vitesse de course. Il vous permet également de choisir la manière dont le vainqueur sera choisi -- Confrontation à mort (Death Match), Course (Race), Pointage (Score) ou Course contre la montre (Time Trial) -- et jusqu'à cinq pistes.
