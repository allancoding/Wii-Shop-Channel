ゲームの内容
かつてのゲームソフトにはないほどリアルな
ビリヤードを、個性豊かなキャラクター達と
一緒にプレイしましょう！

とにかくビリヤードをプレイしたくてたまらない！
そんな時にはクイックプレイがお薦めです！
１人プレイまたは友達との２人プレイから選ぶ
ことができます。ストーリーモードでは夢を実現
しようと頑張っているアメリカ各地の凄腕の
ハスラー達の中から、お気に入りのキャラクターを
選んでプレイすることができます。

ビリヤードをやったことがなくても心配は
ありません！チュートリアルをプレイすれば
基本的なルールから、高度なテクニックまで身に
つけることができます。ゲームをプレイした後に
現実のビリヤードの実力が上がっていたとしても
驚かないでくださいね。

もし、あなたがもっと難しいことに挑戦したいと
思っている上級者だとしても大丈夫です。
プロ顔負けのハスラーにも難しいチャレンジを
たくさん用意してあります！

ハマる★ビリヤードでは、ナインボールでも
エイトボール(US/UKルール)でも好きなルールを
選んでプレイすることができます。
コンボショットやマッセ、その他にも様々な
テクニックをWiiリモコンを使いこなすことで
リアルにプレイすることができます。
Wiiリモコンのモーションセンサーが、デリケートな
軽い一突きから強力なブレイクショットまで
しっかりと使い分けることができます。

さぁ、Wiiリモコンを片手に早速ビリヤードを
始めましょう！
