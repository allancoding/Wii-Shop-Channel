Controles

Palanca de control
Adelante para patinar, hacia los lados para dirigir a tu patinadora.

 

Botón A
Presiona repetidamente para patinar a mayor velocidad

 

Botón B
Golpea hacia tu lado derecho

 

Botón Z
Golpea hacia tu lado izquierdo

 

Botón C
Activa tu poder

 

Botón +
Pausa el juego

 

Cruz de control
ARRIBA: Termina el Jam prematuramente
IZQUIERDA: Selecciona la estrategia de PARED
ABAJO: Selecciona la estrategia de ASISTENCIA
DERECHA: Selecciona la estrategia UNO-A-UNO

 

Para ser lanzado hacia el frente por un miembro de tu equipo
Agita el Wii Remote cuando estés detrás de un miembro de tu equipo

 

Para saltar
Agita el Nunchuk hacia arriba

 

Para hacer trucos
Agita el Wii Remote mientras estés en el aire
