Controls

This game uses the Wii Remote in a horizontal position.

* Items in parentheses indicate non-gaming controls.

* On the Classic Controller, use the b (x) Button to fire and the a (y) Button to jump.

The L and R Buttons can be used to change weapons.

* You can change the Fire and Jump buttons in Options.

Return to Default Weapon

Press the A and B Buttons together to return to your regular weapon.

Proto Man's Controls
Sliding

Hold down on the +Control Pad and press the 2 Button to slide. This is useful for accessing narrow passages and avoiding certain attacks.

Proto Man Strike
Hold down the 1 Button and then release to fire a more powerful shot.
