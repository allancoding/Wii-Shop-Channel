Modes


GAME START
Commencer à jouer. Si vous sélectionnez ce mode, le menu suivant s'affichera.

NEW GAME

Démarrer une nouvelle partie.


LOAD

Continuer une partie sauvegardée.


TIME ATTACK

Faire chaque niveau en mode Contre-la-montre.

Les temps seront enregistrés dans la liste des classements.

Vous pouvez sauvegarder vos données de replay de Time Attack.


ENDLESS ATTACK

Disponible en achetant le contenu supplémentaire correspondant.

Il s'agit d'un niveau infini. L'objectif est d'aller le plus loin possible.

Vos records seront enregistrés dans la liste des classements.


CHALLENGES

Petits niveaux avec leurs propres règles. Consultez la liste des challenges dans le jeu principal.


RANKINGS

Voir les classements du joueur.


OPTIONS

Modifier les paramètres du jeu.


ADD-ON CONTENT

Acheter un contenu supplémentaire pour Mega Man 10.
