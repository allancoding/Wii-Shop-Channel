Cómo jugar
●Puntero
Con el puntero en el Wii Remote, mueve el cursor.
●Botón A
Pulsa el Botón A para elegir.
●Botón B
Pulsa el Botón B para volver a la pantalla anterior.

Controles para Daifugo
＜Configuración vertical Wii Remote＞
●Puntero
Con el puntero en el Wii Remote, mueve el cursor.
●Botón A
Pon el cursor sobre las cartas y pulsa el Botón A para elegir/no elegir carta.
●Botón B
Pulsa el Botón B para no elegir la carta.
●Botón -
Muestra el menú de pausa.
Cuando estés en una lucha de CWF de Nintendo, la presentación de nombre y título pueden cambiarse.

＜Configuración horizontal Wii Remote＞
●Cruz de control
Mueve el cursor.
●Botón 1
Descarta la carta que has elegido.
●Botón 2
Mueve el cursor sobre la carta y pulsa el Botón 2 para elegir/no elegir carta.
●Botón -
Muestra el menú de pausa.
Cuando estés en una lucha de CWF de Nintendo, la presentación de nombre y título pueden cambiarse.

※La configuración del modo de agarrar el Wii Remote puede cambiarse desde la opción "Configuración" en el "Menú principal".

Controles para Speed
●Puntero
Con el puntero en el Wii Remote, mueve el cursor.
●Botón A
Pon el cursor sobre las cartas y pulsa el Botón A para tomar la carta. Vuelve a pulsarlo para soltarla.
●Botón -
Muestra el menú de pausa.
Controles para Memory
●Puntero
Mueve el cursor con el puntero del Wii Remote.
●Botón A
Pon el puntero sobre la carta y pulsa el Botón A para darle la vuelta.
●Botón -
Muestra el menú de pausa.
