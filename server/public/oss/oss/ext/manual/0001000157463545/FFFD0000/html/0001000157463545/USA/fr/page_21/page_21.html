Téléchargements
●Téléchargements
Vous pouvez acheter des contenus supplémentaires pour «Family Card Games» à l’intérieur même du jeu. Pour acheter des contenus supplémentaires, vous aurez besoin de Wii Points. (*Frais supplémentaires)

●Acheter des contenus ajoutés (re-télécharger)
Sélectionnez le niveau que vous souhaitez acheter ou re-télécharger, et sélectionnez soit le bouton « Télécharger », soit le bouton « Re-télécharger ». Une fois que vous avez acheté un niveau, vous pourrez le télécharger de nouveau gratuitement, même si vous le supprimez.

Si vous n’avez pas assez de Wii Points, veuillez vous la Chaîne Boutique Wii ou le site internet officiel de Nintendo et suivez les instructions pour apprendre comment acheter des contenus supplémentaires.

●Supprimer des contenus téléchargés
Vous pouvez supprimer les contenus supplémentaires que vous avez téléchargés sur la console Wii. Les contenus supprimés ici peuvent être re-téléchargés gratuitement.

●Contenus supplémentaires disponibles
Wii Points nécessaires : 100 Wii Points
※Pour chaque paquet de cartes

Nombre de blocs: 2 blocs

Description du contenu:
De nouveaux jeux de cartes peuvent être ajoutés à «Family Card Games». Il y en a plein de différentes ! Achetez celles qui vous plaisent !


※Avertissement
Les contenus supplémentaires sont des contenus ajoutés pour «Family Card Games». Vous ne pouvez pas jouer avec uniquement les contenus supplémentaires.
