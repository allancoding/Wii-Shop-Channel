9
Color Cells

Votre objectif consiste à détruire toutes les cellules avant que la plate-forme n'atteigne le bas de l'écran. Tirez vers le haut pour essayer de former des groupes de 3 cellules ou plus de couleur identique. Une fois un groupe formé, il disparaitra. Vous devrez parfois tirer contre les murs pour que les cellules rebondissent vers leurs cibles. La partie prendra fin si les cellules atteignent le bas de l'écran. Des bonus apparaitront quelquefois dans une cellule, essayez de les récupérer en l'intégrant à un groupe de 3 pour la détruire.

Commandes:

  • Tenez la manette Wii Remote horizontalement.
  • Visez avec la et tirez avec .


Bonus:

  • Vie supplémentaire - vous remportez une vie supplémentaire.
  • Haut - vous disposez d'une munition spéciale vous permettant de faire remonter toutes les cellules de 2 rangées.
  • Rangée - vous disposez d'une munition spéciale vous permettant de détruire toute une rangée de cellules.
  • Bombe - vous disposez d'une munition spéciale vous permettant de détruire toutes les cellules avoisinantes.
