Écran de jeu et règles

● Écran de jeu
Ce mode d’emploi électronique utilise comme exemple l’écran du mode Normal (classique). L'écran apparaissant dans le jeu peut être légèrement différent de l'image ci-dessous, en fonction du mode de jeu choisi.

● Règles
Manœuvrez votre keydron à travers les ouvertures dans le mur. Si une partie de votre keydron touche le mur, le keydron se détruira et vous le perdrez. Vous disposez d'un nombre limité de keydrons.

Note : Dans le mode Endless (sans fin) vous avez l’occasion de remplacer vos keydrons perdus : cherchez le champignon vert dans un long tunnel et attrapez-le!

 ・GAME OVER (fin de partie)
Si vous rencontrez une des conditions ci-dessous, la partie se terminera. Dans les modes Endless (sans fin) et Trick Challenge (défi de figures), l'écran des résultats (voir p. 14) s'affichera.

Conditions de fin de partie :
Normal
(classique)

・Tous les keydrons disponibles sont détruits.

・Vous n’atteignez pas l'arrivée dans le temps imparti.

Endless
(sans fin)

・Tous les keydrons disponibles sont détruits.
Trick Challenge
(défi de figures)

・Le keydron est détruit.

・Vous ne parvenez pas à réaliser la figure (voir p. 13).

À propos du menu pause

Si vous appuyez sur lors d'une partie, vous mettrez le jeu en pause et le menu pause s’affichera.

Retry
(réessayer)

Réessayer depuis le début avec les mêmes paramètres.
Back to Menu
(retourner au menu)

Revenir à l’écran des paramètres du mode.
