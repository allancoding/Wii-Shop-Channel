9
Pour commencer

● Commencer la partie
Si vous jouez pour la première fois, vous aurez d’abord le choix entre commencer une partie ou voir le tutoriel. Une fois à l'écran titre, vous pourrez commencer une partie, consulter le Leaderboard (classements) ou regarder les informations de How to Play (comment jouer – voir p. 11). Si vous avez des données sauvegardées, ThruSpace les lira automatiquement et avancera directement à l’écran titre.

● Écran titre
Appuyez sur n’importe quel bouton après avoir démarré le jeu pour afficher l’écran titre et sélectionner parmi les options suivantes :

Start
(commencer)

Afficher l’écran de sélection du mode de jeu (voir p. 10) pour choisir entre les différents modes de jeu.

Leaderboard
(classements)

Ceci vous permet de consulter soit les High Scores (meilleurs pointages) soit le World Leaderboard (classement mondial).
How to Play
(comment jouer)

Ceci vous permet de réviser les commandes de votre keydron pour chaque style de commandes. Faites défiler les différents styles en appuyant vers la droite ou la gauche sur ou .
Back
(retour)

Retourner à l’écran titre.

Note : Si vous avez décidé de regarder le tutoriel au début du jeu, les classements ne s’afficheront que lorsque vous aurez terminé la Lesson 2 (leçon 2) dans le Keydron Tutorial (tutoriel keydron).

 ・À propos des High Scores (meilleurs pointages) et du World Leaderboard (classement mondial)
Vous pouvez consulter les classements des différents modes de jeu et types de keydrons dans High Scores (meilleurs pointages) et dans World Leaderboard (classement mondial). Pour sélectionner le classement que vous désirez consulter, appuyez vers le haut ou le bas sur ou pour sélectionner Keydron, Mode et Stage (étape) et modifiez les paramètres de chacun en appuyant vers la droite ou la gauche sur ou .

High Scores
(meilleurs
pointages)

Vous pouvez enregistrer et consulter vos propres meilleurs pointages pour chaque mode. De plus, vous pouvez vous connecter à la connexion Wi-Fi Nintendo en sélectionnant Submit Score (enregistrer pointage) pour enregistrer votre propre pointage dans le classement mondial.

Note : Vous ne pouvez pas enregistrer deux fois le même pointage.

World
Leaderboard
(classement
mondial)

Vous pouvez consulter les dix meilleurs pointages pour chaque étape ou mode. Lorsque vous jouez pour la première fois, sélectionnez Update Leaderboard (mise à jour du classement) et connectez-vous à la connexion Wi-Fi Nintendo pour obtenir les meilleurs pointages en ligne. Si vous avez enregistré vos meilleurs pointages dans le classement mondial, vous pourrez voir où vous vous situez dans celui-ci.

À propos de la sauvegarde

Les données des meilleurs pointages ou de votre progression dans le jeu sont sauvegardées automatiquement. Cependant, les données des reprises vidéos peuvent être sauvegardées en appuyant sur lorsque vous visualisez une reprise vidéo (voir p. 14).
