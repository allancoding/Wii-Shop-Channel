Control parental
La función de control parental permite limitar la adquisición de ciertos contenidos y el acceso a ellos. Sigue los pasos que se detallan a continuación para configurar el control parental para todos los títulos.
Nota: El texto que se verá en pantalla podría diferir del de las ilustraciones dependiendo de la versión del menú de Wii que tengas instalada en tu consola Wii.
Configuración del control parental
  1. Selecciona el icono de Wii situado en la parte inferior izquierda del menú de Wii.
  2. Selecciona CONFIGURACIÓN DE Wii, a continuación, CONTROL PARENTAL y establece una contraseña y una pregunta de seguridad si no lo has hecho aún. Selecciona CONTROL PARENTAL Y CONTRASEÑA para establecer la clasificación de juegos más alta que se permitirá. También puedes seleccionar OTROS LÍMITES y navegar usando ATRÁS y ACEPTAR para restringir contenidos concretos.
  3. En OTROS LÍMITES, selecciona RESTRINGIR para limitar el uso de Wii Points™ en el Canal Tienda Wii.
