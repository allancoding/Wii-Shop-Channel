Note : le système USK n’est utilisé qu’en Allemagne.

Informations importantes concernant la classification USK.

Le produit que vous avez acheté a été évalué et classifié par le Unterhaltungssoftware SelbstKontrolle (USK) selon le paragraphe 14 de la loi allemande sur la protection des mineurs (§ 14 JuSchG). Pour les titres WiiWare, la classification se trouve à l'écran de description (qui apparaît une fois que vous avez sélectionné un titre) et dans le mode d'emploi électronique du titre.
