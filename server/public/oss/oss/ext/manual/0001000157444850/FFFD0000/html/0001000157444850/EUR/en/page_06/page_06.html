● What is Rotohex?

Rotohex is a puzzle game that uses a hexagon motif. Use the cursor (Rotohex) to create hexagons (HEX) by moving the triangle-shaped panels that fall. At first, you will probably play the game by trial and error, trying to figure out how you can build HEX. But gradually, you’ll learn to move instinctively!

● How to Create a HEX

Various colours of triangle panels will fall on the playing field. Collect six panels of the same colour and create hexagons (HEX). You can do this by moving the Rotohex to any place on the field and rotating the Rotohex (and the panels within it) clockwise or anticlockwise. The basic game play and goal are the same in every mode: keep completing HEX to make them disappear. If the playing field gets completely filled with panels, the game is over (see "10. How to Play").
