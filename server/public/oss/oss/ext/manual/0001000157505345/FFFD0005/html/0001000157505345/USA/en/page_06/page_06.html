How to Use the Controller

This game can be played by 1-4 players simultaneously. In addition to the Wii Remote™ controller, the Classic Controller™ and  Nintendo GameCube™ Controller can also be used. However, this manual will focus on the controls available using the Wii Remote.

 

* For more information on using controllers other than the Wii Remote when playing solo, please refer to p. 17: Playing Solo Using a Controller Other Than a Wii Remote.

 

Wii Remote

Please hold the Wii Remote horizontally.

 

Other Controllers

If you are using a Classic Controller, connect it to the Wii Remote. If you are using a Nintendo GameCube Controller, connect it to the Wii™ console. Then proceed to Multiplayer Settings (see p. 17: Beginning Co-op Play).
