Controles

Pueden jugar de 1 a 4 jugadores. Además del Wii Remote™, también se pueden usar el Classic Controller™ o el control de Nintendo GameCube™. Ahora bien, las instrucciones de este manual se centran en los controles del Wii Remote.

 

Nota: Para obtener información sobre cómo jugar con los otros controles en el caso de un solo jugador, consulta “Instalaciones para el Juego en equipo” en la página 17.

 

Wii Remote

Sujeta el Wii Remote en posición horizontal.

 

Otros controles

Para usar el Classic Controller, conéctalo al Wii Remote. Para usar el control de Nintendo GameCube, conéctalo a la consola Wii™. Para obtener más información, consulta “Instalaciones para el Juego en equipo” en la página 17.
