Pause Menu

Press to display the Pause Menu. From the Pause Menu you can return to the Title Menu or Terminal, or take a picture. Of course, while the Pause Menu is open, the game is paused.

 

* The Pause Menu that appears in the Terminal is different from the Pause Menu that appears in the Stages and the Battle Royale.

* If you’re in a Stage and you use the Pause Menu to return to the Terminal, any befriended Pokémon and  you collected in that Stage will be lost.

 

Continue

Continue the game.

Go to Title Menu

Go to the Title Menu. This option only appears in the Terminal.

To the Terminal

Return to the Terminal. This option only appears in a Stage or in the Battle Royale.

 

* Attention: If you return to the Terminal from the Pause Menu, you will forfeit all befriended Pokémon and  you’ve earned in the Stage so far.

Take a Picture

Take a picture of the game screen. You can save pictures you’ve taken to an SD Card (sold separately), but not to Wii system memory.

SD Card

You can delete all Pokémon Rumble pictures saved to the SD Card.

Multiplayer

Play co-operatively with up to 4 players (see p. 17: Facilities for Co-op Play).

 

 

More about Taking Pictures

If you select Take a Picture, a picture will be taken of the game screen. Once you’ve checked the picture, you can select Yes to save the picture to an SD Card (as a JPEG file) and the file name and file folder will be displayed.

 

* The Pause Menu will not appear in pictures.

* You can view your pictures in the Photo Channel or on your home PC.

* The copyrights to all pictures taken belong to The Pokémon Company and the respective rights owners. The pictures are for your enjoyment only and may not be used for any commercial purpose.

 

Deleting Pictures

To delete the pictures saved to the SD Card, insert the SD Card into the Wii console and select (SD Card) → (Delete Pictures).

If you select (Delete Pictures), all images saved to the SD Card will be deleted, not just the pictures from this game. It is not possible to delete images one by one.
