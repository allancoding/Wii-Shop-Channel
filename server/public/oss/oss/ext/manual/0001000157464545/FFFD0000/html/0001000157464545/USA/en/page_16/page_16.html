USEFUL NEWSPAPERS


In the store there are some newspapers that will be very helpful: they have some clues of how to solve a few problems in the game.


LOAD AND SAVE


To go to the menu, you only need to press the + Button, or select a symbol in the upper-left corner of the screen. Then the following options will appear: Back to game, Save game, Load game (only if you have saved a game before).
