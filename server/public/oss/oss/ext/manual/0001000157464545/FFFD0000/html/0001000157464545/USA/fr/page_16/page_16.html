LA PRESSE

 

Dans la boutique, vous trouverez également différents journaux, si l´on peut appeler comme ça ces feuilles de chou dépareillées et antédiluviennes. Ces journaux vous donneront de précieuses informations sur les problèmes que vous aurez à résoudre au cours de l'aventure.
                                         

SAUVEGARDE ET CHARGEMENT

 

Pour accéder au menu, appuyez sur le bouton +. Les options suivantes s'offrent alors à vous:
Revenir au jeu, Sauvegarder, Charger (cette option n'est disponible que si vous avez préalablement effectué une sauvegarde).
