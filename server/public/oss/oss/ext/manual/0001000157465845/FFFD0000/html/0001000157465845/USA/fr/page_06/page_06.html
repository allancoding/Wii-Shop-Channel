Menu principal

Voici le menu principal.

(1) Mode majordome (automatique) : active ce point du menu pour que les nouvelles bûches s'ajoutent automatiquement ; tu peux regarder tranquillement.

(2) Feu non-stop : le feu brûle indéfiniment sans que les bûches ne se consument.

(3) Sélection Cheminée : change de cheminée avec les flèches gauche et droite.

(4) Nouveau : démarre un nouveau feu.

(5) Cheminée : passe à l'écran cheminée.

(6) Point d'interrogation : fait défiler le générique.

(7) Statistiques : affiche différentes informations sur le feu :

  • Durée : indique depuis combien de temps la partie est en cours
  • Chaleur : indique la température actuelle du feu
  • Bois : nombre de bûches qui ont été consumées
  • Brûlures : nombre de fois où tu t’es brûlé les doigts en ajoutant du bois.
