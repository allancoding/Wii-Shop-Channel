Main Menu

This is the main menu.

(1) Butler mode: Select this option and logs will be automatically placed in the fire. Sit back and relax!

(2) Fire non-stop: Select this option for the fire to burn eternally without needing new logs.

(3) Fireplace selection: use the left and right arrow icons to scroll through the various fireplaces.

(4) New: start a new fire

(5) Fireplace: switch to the fireplace screen

(6) Question mark: display the credits and other info

(7) Statistics: A range of info on the fire is displayed here:

  • Duration: how long you have been playing
  • Heat: the temperature of the hearth
  • Wood: the number of logs that have been used
  • Burns: the number of times you have burned your fingers when placing a log on the fire
