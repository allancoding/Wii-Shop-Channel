Getting Started
Press the A Button and B Button on the Wii Remote simultaneously on the title screen.
Save
When you first start out the game, you’re required to create a game file in which you‘ll save the game data later. Six empty blocks in the Wii system memory are required to create the game file. As you progress through the game, the data will be automatically saved. A game file can’t be copied over to a SD Memory Card.

You will select a menu.
