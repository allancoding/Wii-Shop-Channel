Function Button
Press “Function Button” like the icon shown on the left to display the menu below.
Handler information View the handler information. You can change the handler name and team names as well.
Dog information View the information of the currently selected dog.
Change dog Return to “Select a dog” to change the current dog.
Friend information View your “Friend Code”, register your friend’s “Friend Code” to your “Friend Roster”.
Ranking View rankings.
