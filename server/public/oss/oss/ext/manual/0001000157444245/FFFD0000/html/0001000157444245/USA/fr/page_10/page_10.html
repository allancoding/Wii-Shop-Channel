Maître-chien
Jusqu’à quatre maîtres-chiens peuvent être créés. Sélectionnez un panneau disponible pour créer un nouveau maître-chien ou sélectionnez un maître-chien existant pour démarrer la partie avec celui-ci.

Informations sur le maître-chien
Vous pouvez paramétrer les éléments ci-dessous.
Handler name
(Nom du maître-chien)		 Nom du maître-chien. Ce nom peut comprendre jusqu’à neuf caractères. Il sera utilisé pour les courses Connexion Wi-Fi Nintendo et les classements.
Handler voice
(Voix du maître-chien)		 Voix du maître-chien. Vous pouvez une voix masculine ou féminine. La voix du maître-chien est audible via le haut-parleur de la Wii Remote.
Team name
(Nom de l’équipe)		 Nom de l’équipe à laquelle appartient le maître-chien. Ce nom peut comprendre jusqu’à six caractères. Il sera utilisé pour les courses Connexion Wi-Fi Nintendo et les classements.

● Lorsque vous utilisez des symboles dans les noms...

Si les noms (maîtres-chiens, équipes, chiens) comportent des symboles, ceux-ci peuvent ne pas apparaître correctement. Pour y remédier, remplacez ces symboles par d’autres caractères.
Le niveau de compétence d’un maître-chien est indiqué par « Level (Lv) » (Niveau). Ce niveau augmente lorsqu’il marque des points lors des entraînements et diminue lorsqu’il perd une course.
