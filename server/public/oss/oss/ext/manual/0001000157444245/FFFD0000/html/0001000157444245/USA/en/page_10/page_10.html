Handler
Up to four handlers can be created. Select any available panel to create a new handler, or select an existing handler to start the game with that handler.

Handler Information
You can set the items shown below.
Handler name The name of a handler. Up to nine characters can be used per name. It’ll be used for Nintendo Wi-Fi Connection Race, and rankings.
Handler voice The voice of the handler. You can select male or female voice. The handler’s voice will be played through the Wii Remote speaker.
Team name The name of the team the handler belongs to. Up to six characters can be used for the name. It’ll be used for Nintendo Wi-Fi Connection races, and rankings.

● When you use symbols in names...

If handler names, team names or dog names contain any symbols, they might not appear properly in some cases. It can be fixed by replacing the symbols with other characters.
The skill level of a handler is indicated by his/her “Level (Lv)”. The level increases when scoring high points in trainings, and decreases when losing a race.
