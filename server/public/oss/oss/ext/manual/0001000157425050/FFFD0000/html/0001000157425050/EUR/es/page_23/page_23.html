Elige tu YO-Fantástico

Cada YO-Fantástico tiene sus puntos fuertes y débiles. 

Algunos tienen ventajas especiales, como por ejemplo la linterna que ilumina el Bosque de los Fantasmas.

 

Apunta con el mando de Wii a la izquierda o la derecha para volar por el espacio y elegir un YO-Fantástico.

 



Inclinar el mando de Wii para verlos más de cerca y pulsar para seleccionar uno.
