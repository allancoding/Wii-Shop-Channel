Commencer et quitter le jeu
  • Commencer le jeu
    À l'écran titre, appuyez sur . Pour commencer le jeu, pointez "START GAME" avec la télécommande Wii ou sélectionnez le avec puis appuyez sur .

  • Écran de sélection du mode
    Vous avez le choix entre deux modes de jeu différents.

    • 1. Sélectionner un personnage
    Sélectionner le personnage avec lequel vous désirez jouer. Il y a 7 personnages différents au choix.

    2. Sélectionner un tableau
    Sélectionner le tableau dans lequel vous désirez jouer. Il y a 3 tableaux différents au choix.

    1. Sélectionner le nombre de joueurs
    Sélectionner le nombre de joueurs qui prendront part au jeu. Un maximum de 4 joueurs peuvent jouer alternativement dans ce mode en utilisant la même télécommande Wii accompagnée du Nunchuk.

  • Quitter le jeu
    Pendant la partie, appuyez sur pour afficher l'écran du menu HOME. Une fois cela effectué, sélectionnez "Menu Wii" pour quitter le jeu.
