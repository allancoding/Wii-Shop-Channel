Game Rules (2)

In this game mode, you start off with 3 hearts of stamina and compete with other players to see how many walls you can pass through.

The character in front of you will pose and run through walls, so as you approach these walls, make the same pose to pass through them.
Use the Wii Remote and the Nunchuk to pose in four different ways.

Up to 4 players can play alternately in this mode.
