Introduction

 

Driift Mania is an awesome top view racing game allowing 8 players to play together in fierce races and crazy game modes...


To start playing, press or on the game splash screen to have access to the game main menu.


On the main menu you will have the choice between 3 items:

 

 

● Championship

Allows you to play a championship. You have the choice between 4 car categories and 5 different cups. There are 3 courses in each normal cup and 4 in a World cup. Your goal is to gain the maximum number of points in each course and win each cup with each category of car.

 

 

 

● Multiplayer

Multiplayer mode is the main mode of the game. It is a multiplayer mode allowing 8 players to play simultaneously in many different game modes. Players have the choice in game settings to enable or disable categories of cars and tracks. The number of cars and tracks available in each mode depends on the number of cars and tracks you have unlocked.

 

 

 

● Options

The options menu allows you to set various general settings like sound and sound effects. You can also have a look at the game credits.
