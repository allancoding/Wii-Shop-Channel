Présentation

 

Driift Mania est un super jeu de course en vue de haut qui permet à 8 joueurs de s'affronter lors de courses folles et dans des modes de jeu incroyables...


Pour lancer le jeu, appuyez sur ou lorsque l'écran d'accueil est affiché afin d'accéder au menu principal du jeu.


Le menu principal vous propose trois modes différents :

 

 

● Championnat

Permet d'effectuer un championnat. Vous avez le choix entre 4 catégories de véhicules et 5 coupes différentes. Chaque coupe standard est constituée de 3 courses, tandis que la coupe Monde en comporte 4. Votre objectif est d'obtenir le maximum de points à chaque course et de remporter chacune des coupes avec chaque catégorie de véhicule.

 

 

 

● Multijoueur

Le mode Multijoueur est le mode principal du jeu. Il s'agit d'un mode multijoueur qui permet à 8 joueurs de s'affronter simultanément dans de nombreux modes de jeu différents. Les joueurs ont la possibilité d'activer ou de désactiver les catégories de véhicules et de circuits dans les paramètres du jeu. Le nombre de véhicules et de circuits disponibles dans chaque mode dépend du nombre de véhicules et de circuits que vous avez débloqués.

 

 

 

● Options

Le menu Options vous permet de régler plusieurs paramètres généraux, tels que la musique et les effets sonores. Vous pouvez également consulter le générique du jeu.
