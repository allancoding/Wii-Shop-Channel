Démarrer une partie


Sur l'écran de titre, appuie sur pour accéder au menu principal.


Pointe la télécommande Wii sur une sauvegarde et appuie sur pour la sélectionner ou pour l'effacer.


Choisis "Nouvelle partie" pour créer une nouvelle sauvegarde. Ensuite, choisis une Mii™ pour ta sauvegarde.

Une fois que tu as choisi ta sauvegarde, le jeu démarre.
