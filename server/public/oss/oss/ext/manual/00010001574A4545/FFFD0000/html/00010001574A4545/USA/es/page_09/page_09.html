Base aérea


La base aérea es el patrullero de Jett. Usando energía solar, la base aérea puede transportar a Jett a cualquier punto de Yoroppa. Cuantas más células fotoeléctricas encuentre Jett por el planeta, más zonas podrá visitar.


Una célula fotoeléctrica holográfica te indicará cuántas células fotoeléctricas necesitas para acceder a una zona nueva. Hay 3 zonas, cada cual con 4 niveles por explorar.


En la siguiente pantalla, apunta a un nivel con el mando de Wii y pulsa para jugarlo. Jett solo puede acceder al primer nivel de una zona recién desbloqueada. Completar un nivel desbloquea otros nuevos.
