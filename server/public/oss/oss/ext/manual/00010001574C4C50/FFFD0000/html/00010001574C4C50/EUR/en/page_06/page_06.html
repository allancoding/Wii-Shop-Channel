Controllers Settings
How to control

This game is played using Nunchuk style.


Menu Screen Operation

To select option tabs
- Point at any option tab and press A Button to confirm selection.
- Alternatively, use +Control Pad to change selection and press A Button to confirm.
- Press B Button to go back to previous menu (if applicable).

In-Game Operation

  • Strategy Mode
    • - Control the cursor by using the Control Stick on the Nunchuk.
    - Choose a unit by moving the cursor under it and press A Button.
    - Cancel the current selection by pressing B Button.
    - To change camera view, press C Button for view from left or press Z Button for view from right.

  • Battle Mode
    • - Control the movement of the Unit by using the Control Stick on the Nunchuk.
    - Press A Button to perform Primary Attack.
    - Press B Button to perform Secondary Attack.
    - Press Z Button to perform Block.

    Press + Button in either mode to display Pause Menu.
    Press HOME in either mode to display Home Menu.
