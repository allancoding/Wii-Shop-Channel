Configuración de los mandos
Cómo controlar

A este juego se juega utilizando el modo Nunchuk.


Funcionamiento de la pantalla de menú

Para seleccionar pestañas de opciones
- Apunta a cualquier pestaña de opciones y pulsa el Botón A para confirmar la selección.
- Como alternativa, puedes utilizar la cruz de control para cambiar la selección y pulsar el Botón A para confirmar.
- Pulsa el Botón B para volver al menú anterior (en caso aplicable).

Funcionamiento del juego

  • Modo Estrategia
    • - Controla el cursor utilizando la palanca de control del Nunchuk.
    - Elige una Unidad situando el cursor debajo de ella y pulsa el Botón A.
    - Cancela la selección actual pulsando el Botón B.
    - Para cambiar la vista de la cámara, pulsa el Botón C para ver desde la izquierda o pulsa el Botón Z para ver desde la derecha.

  • Modo Batalla
    • - Controla el movimiento de la Unidad utilizando la palanca de control del Nunchuk.
    - Pulsa el Botón A para realizar un Ataque principal.
    - Pulsa el Botón B para realizar un Ataque secundario.
    - Pulsa el Botón Z para realizar un Bloqueo.

    Pulsa el Botón + en cualquiera de los modos para mostrar el Menú de pausa.
    Pulsa el Botón HOME en cualquiera de los modos para mostrar el menú HOME.
