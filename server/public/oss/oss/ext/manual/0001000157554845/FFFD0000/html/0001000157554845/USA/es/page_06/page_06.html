Controles básicos

Este juego utiliza el control Wii Remote™ y el Nunchuk™. Cuando elijas opciones en el menú, apunta a la pantalla con el Wii Remote y oprime .

Nota: También puedes mover el cursor oprimiendo .

● Cómo sujetar los controles durante el juego
Cuando juegues Grill-off with Ultra Hand!, tienes que sujetar el Wii Remote y el Nunchuk de forma vertical, de manera que apunten hacia arriba.

Nota: Si sostienes el Wii Remote y el Nunchuk de forma horizontal te resultará muy difícil controlar la Ultra Hand, ¡así que no lo hagas!

Apuntar
A lo largo de este manual, "Apuntar" se refiere a señalar hacia la pantalla con el control Wii Remote y hacer que el cursor se coloque en un punto específico.
