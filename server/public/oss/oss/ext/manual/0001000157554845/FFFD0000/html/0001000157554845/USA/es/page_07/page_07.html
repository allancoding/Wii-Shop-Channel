Cómo guardar

Cuando comienzas el juego, los datos de este se guardarán automáticamente en la memoria de tu consola Wii. Los datos también se guardarán automáticamente cada vez que termines un juego o cuando se guarden tus puntuaciones.

Nota: Este juego necesita por lo menos un bloque libre en la memoria de la consola Wii.

Sobre el guardado de datos
Puede que los datos no se guarden o se pierdan si haces alguna de las dos cosas que se describen a continuación. Una vez perdidos, no podrás recuperar los datos, así que, ¡ten cuidado!
・
Apagar o reiniciar la consola Wii o el Wii Remote mientras se guardan los datos.
・
Desconectar el adaptador de corriente Wii durante la partida.
