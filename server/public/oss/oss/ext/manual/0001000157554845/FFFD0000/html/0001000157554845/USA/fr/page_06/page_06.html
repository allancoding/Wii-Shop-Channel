Commandes de base

Ce jeu se joue avec la manette Wii Remote™ et le Nunchuk™. Lorsque vous souhaitez sélectionner une option dans les menus, pointez la manette Wii Remote vers l'écran, placez le curseur sur l'option voulue et appuyez sur .

Note : Vous pouvez également déplacer le curseur en appuyant sur .

● Comment tenir les manettes durant le jeu
Lorsque vous jouez à Grill-off with Ultra Hand!, vous devez tenir la manette Wii Remote et le Nunchuk à la verticale de façon à ce qu’ils pointent vers le haut.

Note : Vous aurez de la difficulté à contrôler l’Ultra Hand si vous jouez en tenant la manette Wii Remote et le Nunchuk à l’horizontale, alors ne le faites pas!
