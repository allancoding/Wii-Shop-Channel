Commencer un jeu

Choisissez un fichier vide (EMPTY) pour créer un nouveau profil, puis saisissez un nom pour celui-ci. Si des profils ont déjà été créés, sélectionnez celui avec lequel vous souhaitez jouer.

Note : Après avoir créé un profil, vous pourrez accéder aux options du profil, en sélectionnant le profil désiré et en appuyant sur . Vous pourrez ainsi changer le nom du profil (CHANGE NAME) ou l’effacer (DELETE PROFILE). Une fois effacées, les données du profil ne pourront plus être récupérées!

Après avoir sélectionné un profil, vous serez dirigé vers le menu principal (MAIN MENU).
