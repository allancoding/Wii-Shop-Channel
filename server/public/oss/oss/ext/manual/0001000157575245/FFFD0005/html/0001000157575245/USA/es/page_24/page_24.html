WiiConnect24

WiiConnect24 usa tu conexión de internet para conectar tu consola Wii a internet durante las 24 horas del día, inclusive si la consola se encuentra apagada, lo cual permite que una variedad de transmisiones de datos se realicen y que se puedan compartir mensajes entre amigos y familiares.

Para poder usar esta función, primero debes definir tu configuración para la conexión de internet y activar la función de WiiConnect24.

¡Atención!

Si te conectas a WiiConnect24, existe la posibilidad de que el nombre de tu perfil sea visto por muchas personas.

Para obtener mayor información acerca de la configuración de internet y WiiConnect24, consulta el manual de operaciones de la consola Wii.
