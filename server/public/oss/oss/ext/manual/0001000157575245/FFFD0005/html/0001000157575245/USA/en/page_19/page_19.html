Creating Your Own Tracks

  1. Select Track Editor from the main menu.

  1. Select an empty slot and press .

  1. Create a name for your new track (tracks can be renamed later).

  1. New tracks start off as a series of flat tiles that can be modified as you see fit.

Press in the Track Editor screen to access Track Options.

Track Options Menu
Return Return to the Track Editor screen.
Copy Copy the track into an empty slot.
Change Name Change the name of the track.
Delete Delete the track.
Send Track Send the track to one of your Wii Friends (see page 22).
