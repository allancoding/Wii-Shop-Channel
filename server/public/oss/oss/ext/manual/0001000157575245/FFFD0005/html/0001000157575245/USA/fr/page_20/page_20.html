Modification des parcours

Pour modifier les parcours, sélectionnez les dalles à modifier en appuyant sur gauche ou droite sur .

Appuyez sur haut sur , puis sur pour effacer la dalle sélectionnée.

Appuyez sur bas sur pour insérer une dalle devant la dalle sélectionnée.

Appuyez sur pour modifier la dalle sélectionnée.

Pour changer le type de dalle, appuyez sur gauche ou droite sur . Appuyez sur haut ou bas sur pour choisir une autre catégorie de dalle. Appuyez sur pour accepter la dalle sélectionnée.

Appuyez sur pour essayer votre parcours. Vous pouvez l’essayer depuis le début (TEST FROM START) ou à partir du point où vous en êtes (TEST FROM HERE). Confirmez l’option désirée avec .
