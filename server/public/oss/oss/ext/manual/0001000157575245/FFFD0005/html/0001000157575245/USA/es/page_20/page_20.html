Edición de circuitos

 

Al editar los circuitos, selecciona cada tramo al oprimir hacia la izquierda o derecha en .

Oprime hacia arriba en para borrar el tramo seleccionado.

Oprime hacia abajo en para insertar un nuevo tramo al lado del tramo seleccionado.

Oprime para modificar el tramo seleccionado.

Para cambiar un tramo, oprime hacia la izquierda o derecha en . Oprime hacia arriba o abajo en para ir hacia el siguiente tipo de tramo. Oprime para aceptar el tramo seleccionado.

Oprime para probar tu circuito. Puedes probarlo a partir de tu ubicación actual o desde la salida del circuito.
