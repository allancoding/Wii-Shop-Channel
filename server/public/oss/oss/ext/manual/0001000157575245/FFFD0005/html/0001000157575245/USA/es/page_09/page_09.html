Cómo empezar

Para crear un archivo nuevo elige EMPTY (vacío) y escribe tu nombre. Si un archivo ya existe, selecciona el que quieras usar.

Nota: Cualquier archivo se puede borrar o cambiar de nombre en cualquier momento, simplemente oprime en la pantalla SELECT A PROFILE (selección de archivos) para abrir el menú de opciones, a continuación elige DELETE PROFILE (borrar archivo) para borrar el archivo o CHANGE NAME (cambiar nombre) para modificar el nombre del archivo. Cualquier archivo que se borre no se podrá recuperar.

Después de escoger un archivo, pasarás a MAIN MENU (menú principal).
