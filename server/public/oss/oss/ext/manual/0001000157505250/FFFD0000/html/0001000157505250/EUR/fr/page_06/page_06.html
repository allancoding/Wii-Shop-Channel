Bienvenue!

● Qu’est-ce que Cubello?

Cubello est un casse-tête où vous devez détruire les cubes colorés qui forment le "cubello" en lançant des cubes de votre réserve.

● Comment détruire les cubes

Reliez au minimum quatre cubes de même couleur pour les détruire. Les cubes doivent être reliés par au moins une face, que ce soit sur le plan horizontal ou vertical.

● Deux modes de jeu

Ce jeu a deux modes: le mode Stage et le mode Endless (voir "8. Commencer"). Dans les deux modes, le but est de détruire tous les cubes du cubello sans faire de game over (voir "10. Comment jouer").

Note: le mode Endless sera disponible une fois que vous aurez réussi un certain nombre de niveaux dans le mode Stage.
