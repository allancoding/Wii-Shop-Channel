Willkommen!

● Was ist Cubello?

Ziel des Spiels ist es, verschiedenfarbige Würfel zu eliminieren, die eine "Cubello" genannte Würfelkonstruktion bilden. Dazu werden Würfel aus dem eigenen Würfelspeicher auf den Cubello abgeschossen.

● Eliminieren von Würfeln

Würfel verschwinden, wenn mindestens vier Würfeln derselben Farbe mit den Seiten aufeinander treffen. Die Seiten können dabei sowohl horizontal als auch vertikal aneinander liegen.

● Zwei Spielmodi

Es stehen zwei Spielmodi zur Auswahl: STAGE (Stage-Modus) sowie ENDLESS (Endlos-Modus, siehe „8. Spielstart“). In beiden Modi ist es Ziel des Spiels, sämtliche Würfel des Cubello zu eliminieren, bevor das Spiel mit einem Game Over endet (siehe „10. Spielverlauf“).

Hinweis: Der Endlos-Modus kann erst dann ausgewählt werden, wenn eine bestimmte Anzahl Stages im Stage-Modus bewältigt wurde.
