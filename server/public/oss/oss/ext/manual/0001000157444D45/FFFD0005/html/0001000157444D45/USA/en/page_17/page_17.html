Friend Settings

In order to play with friends over Nintendo Wi-Fi Connection, you must exchange and register friend codes. Select the Nintendo WFC mode and then choose Friend Settings. From there, you can confirm your own friend code and register the friend codes of others.

Friend Roster

Check your Friend Roster, where you can register up to 30 friends. Your friends will be listed in the order you last played them. Friends with whom you have only exchanged friend codes will have their friend code as their name.
Register Friend Code

Enter a friend code to add that player to your friend roster. You will need your own friend code to complete this process. Simply connect to Nintendo Wi-Fi Connection to receive a friend code.
Confirm Friend Code

View your friend code.

Note: This is different from your Wii number.

About Friend Codes:

Each save file will receive a 12-digit friend code the first time it is used to access Nintendo Wi-Fi Connection. Friend codes cannot be obtained without connecting to Nintendo Wi-Fi Connection. If you cannot enter a friend code, or a friend code is not displayed, try connecting to Nintendo Wi-Fi Connection to see if this resolves the problem.
