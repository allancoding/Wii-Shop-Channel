Using WiiConnect24

You can use WiiConnect24™ to send a demo version of Dr. Mario; Online Rx to a friend who doesn’t have the original game. This will allow you to play Friend Battles together. Please read the Wii Operations Manual to learn how to set WiiConnect24 to On. The Parental Controls must be set to Off in order to send a demo so if Parental Controls are being used, please check with a parent or guardian to consider temporarily changing the setting to Off.

Note: There is no charge for the demo version.

● Registering a Wii Friend
In order for you to send the demo version, the recipient must be registered as a Wii Friend. Wii Friends are different from friends registered in the game. Instead, Wii Friends are registered through the Wii System Settings menu. Please refer to the Wii Operations Manual for details.

After you have registered each other, the color of your Wii Friend’s name in your Wii address book will change.

Attention

If you are having trouble connecting to the Internet, please check any error codes that appear on-screen and refer to the troubleshooting section of the Wii Operations Manual.
