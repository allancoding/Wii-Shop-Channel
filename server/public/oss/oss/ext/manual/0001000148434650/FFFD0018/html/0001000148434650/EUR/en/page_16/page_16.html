Creating a Message

You can create voice messages and send them to your Wii Friends. You can also attach both a photo and a Mii to your message. Messages can’t be created while you’re in a conversation.

You can even send messages to Wii Friends who do not have the Wii Speak Channel. They will be able to receive your messages via the Wii Message Board.

Sending a Message
  1. In the Conversation Room or Lobby, select to go to the Create Message Screen.
  2. Select . A list of your Wii Friends will be displayed.

  3. Point at and press to put next to the names of the Wii Friends you want to send your message to.

    You can send a message to up to eight Wii Friends at any one time.

    To remove , point at it again and press .

    Select CONFIRM to return to the Create Message Screen.

  4. If you want to attach a Mii, select ATTACH A Mii and then select the Mii you want to use.

  5. Select the grey box under VOICE MESSAGE on the Create Message Screen to record your voice.

  6. Select RECORD and speak clearly in the direction of the microphone. The maximum recording time is 10 seconds.

    Press before recording to change your voice. You can listen back to the message once it’s recorded.

  7. Select OK to return to the Create Message Screen. The VOICE MESSAGE box will change colour to indicate that there’s a recording attached.

  8. To attach a photo, select the grey box under PHOTO.

    Follow steps 2 and 3 in “14. Showing a Photo” to choose a photo, then choose SELECT.

  9. Select OK to go back to the Create Message Screen. The photo will now be attached to your message.

  10. Once you’re ready to send the message, select SEND. It may take a little time before your message arrives.

    Note: You can change the number of addressees, select another photo or Mii, or record your voice again at any point before sending the message.

Receiving Messages

When you receive a message, the Wii Message Board icon on the Wii Menu will blink. You can also change your Wii System Settings so that the Disc Slot on your Wii console blinks when you receive a new message. Please refer to the Wii Operations Manual – Channels & Settings for details.

  1. Select the Wii Message Board icon from the Wii Menu. The Wii Message Board will be displayed.

  2. Select the message you want to hear to play it back. To hear the message again, select BACK to return to the Wii Message Board and reopen the message.

    To create a reply, select START to go to the Wii Speak Channel.

    To view photos, simply point at them and press .

    Note: If the recipient does not have the Wii Speak Channel, REPLY will be displayed in place of START.

If your Wii Message Board no longer has any free memory, you can save messages to an SD Card. For more information on data management using SD Cards, refer to the Wii Settings and Data Management section in the Wii Operations Manual – Channels & Settings.
