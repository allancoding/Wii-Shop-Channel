Este método de control es para aquellos jugadores que tienen un mando clásico. Si lo conectas al mando de Wii, puedes usarlo para jugar a And Yet It Moves.

Pulsa izquierda o derecha en la cruz de control o mueve la palanca izquierda para hacer correr a tu personaje. Pulsa los Botones a o b para que salte. Finalmente, para rotar el mundo, pulsa los Botones L o R. Mientras mantengas pulsado uno de estos Botones, el mundo se congelará y girará. Para detener su rotación, vuelve a soltar el Botón.
