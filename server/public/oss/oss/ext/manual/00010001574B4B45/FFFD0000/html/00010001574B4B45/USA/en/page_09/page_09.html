Pop-Up Party
Players take turns thrusting swords into a barrel, trying to see who can insert the most swords. Thrust a sword into the wrong hole and you'll send the Captain flying, which means it's game over for you.
Select Pop-Up Party from the main menu screen, and then set the number of players and the order in which they'll play. You'll then proceed to the Pop-Up Party game screen.

● The Game Screen

● Pop-Up Party Rules
Up to four players can play simultaneously.
Players take turns selecting holes and inserting swords one by one.
Rotate the barrel to either side by pressing left and right with the +Control Pad. Once you've selected a hole, hold down the A Button and thrust forward with the Wii Remote to insert a sword.

● Setting the Number of Players
Before starting the game, select the number of players and assign them their controllers. Move the on-screen pointer to set each player's controller assignment.
First select one of the icons on the left side of the screen, and then press the A Button to scroll through the following options: Registered Remote 1-4 → COM → NONE (No Crew Member).

*Player 1 can't be assigned a different remote or be set to COM or NONE (No Crew Member).
*At least two players must be set.

○ Adding Wii Remotes
Press the 1 Button and the 2 Button simultaneously on the Wii Remote you'd like to add. After registering a new Wii Remote, you'll be able to select its icon on the left side of the screen.

● Player Order

    Once you've set the number of players, you'll be prompted to set the order in which they'll play. The order is randomized each time you play.
At the bottom of the screen is a sign spinning atop a sword. Press the A Button to stop the sign, or wait three seconds and it will stop on its own. Each player's assigned sword color determines the order in which they'll play; Red: 1st, Blue: 2nd, Green: 3rd, Yellow: 4th.
