Getting started

Creating a savegame

Select "New Game" to create a new savegame. Now select one of the Mii™ Characters saved on your Wii Console.

Main Menu

Select Qualification for a quick introduction into the game's controls. The qualification is mandatory if it has not been completed before.


Select Single Player if you want to play alone. Check out page 9 for more information on Single Player mode.

You can also play together with up to three friends in Multiplayer mode. Page 10 provides details on the Multiplayer mode.

Select Unlockables for a quick overview of your in-game progress.
