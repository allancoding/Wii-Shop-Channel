Le mode Aventure

Le mode Aventure vous propose de découvrir les douze dieux à travers soixante niveaux.

Disponible dès la première partie, il vous permettra de débloquer les autres modes de jeu.

Progression

Chaque mythologie vous fera découvrir trois dieux à travers cinq niveaux, soit quinze niveaux et trois challenges par mythologie. Les pouvoirs des dieux se débloqueront au fur et à mesure de votre progression.

A chaque fois que vous en avez terminé avec un dieu, vous débloquez un challenge disponible dans le mode Challenge.

Pour terminer une mythologie, vous devrez affronter le champion des humains.

Les champions

Il y a quatre champions dans le jeu, un pour chaque mythologie.

Lorsque la tour est presque entièrement construite, le champion entre dans la tour par le premier étage et monte jusqu'au sommet.

S'il atteint le portail, la partie est perdue.

Pour l'empêcher d'atteindre votre demeure, vous devez l'attaquer avec vos pouvoirs offensifs afin de briser son armure.

Condition de victoire

Pour remporter la partie, vous devez détruire l'armure du champion avant qu'il n’atteigne le portail. Il fera alors demi-tour.
