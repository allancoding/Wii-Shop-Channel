Controls

The game Gods vs Humans requires a Nunchuk.

General information – Wii Remote

Select an icon, confirm.

Return to previous screen, cancel.

Pause the game.

Display the Memo.

Use your Powers – Wii Remote + Nunchuk


Display the Powers selection menu.


Move the Nunchuk Control Stick in the required direction to select a Power. When you release the Control Stick, the Power located at the center of the display is automatically selected.


After selecting a Power, release the +Control Pad to use it.

Use the selected Power.

Control the camera – Nunchuk Control Stick


Up: move up a Level
Down: move down a Level

+
Up: move all the way up to the divine portal.
Down: move all the way down to the Foundations.

Move closer to or further away from the Tower.
