Le mode Duel

Ce mode vous permet d'affronter un deuxième joueur.

Chaque joueur choisit son dieu et sa difficulté parmi les éléments qui ont été débloqués en mode Aventure.

Conditions de victoire

Soit vous détruisez les fondations de votre tour ou tous ses étages, soit le portail de votre adversaire a été détruit.
