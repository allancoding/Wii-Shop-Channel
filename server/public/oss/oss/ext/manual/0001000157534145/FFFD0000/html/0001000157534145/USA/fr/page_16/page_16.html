Créatures ennemies
Vous devez combattre de nombreuses créatures dans Onslaught, connues sous le nom d'"insectes cybernétiques". Les insectes cybernétiques sont composés de 3 sections distinctes : le "cœur", la "chair" et le "métal". Chaque partie subit un montant de dégâts différent. Attaquez le cœur pour infliger plus de dégâts.
Chenilles
Ces créatures peuvent cracher un fluide corrosif et leur morsure est mortelle.
Guêpes
Attention ! Ces insectes volants vous attaquent par le haut.
Scarabées
Ces insectes possèdent des attaques longues portées puissantes et leur carapace est dure à percer.
Sauterelles
Ces créatures sautent partout lorsqu'elles attaquent, ce qui les rend difficiles à cibler.

*Vous rencontrerez bien d'autres types de créatures !
