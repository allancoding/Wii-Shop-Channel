Format de l'écran et sauvegarde
Format de l'écran
Vous pouvez changer le format de l'écran depuis le menu Wii.
Veuillez vous référer au mode d'emploi de la Wii pour plus de détails.
Sauvegarde
Ce jeu utilise une fonction de sauvegarde automatique.
Les données sauvegardées peuvent être perdues si l'une des actions ci-dessous est effectuée au cours d'une sauvegarde. Par ailleurs, les données perdues ne peuvent être restaurées.

● Appuyez sur le bouton POWER ou sur le bouton RESET de la console Wii ou de la Wii Remote alors qu'une sauvegarde est en cours.

● Sélectionnez "Menu Wii" ou "Réinitialisation" dans le menu HOME alors qu'une sauvegarde est en cours..

● Débranchez l'adaptateur CA Wii alors que la console Wii est sous tension.
