Sauvegarde
Appuyez sur le bouton Sauvegarde à l'écran pour afficher l'écran de sauvegarde. Sélectionnez un emplacement pour sauvegarder votre partie.

Les données écrasées ne pourront pas être récupérées.
Fonction sauvegarde

Attention ! Les données perdues ne peuvent être récupérées. Afin d'éviter toute perte de données, évitez de :
*Appuyer sur le bouton POWER, le bouton RESET ou les boutons de la télécommande Wii pendant la sauvegarde.
*Débrancher le bloc d'alimentation Wii pendant la sauvegarde.
