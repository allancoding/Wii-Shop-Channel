Interface de Jeu
 

Curseur

La main est votre curseur. Le nombre affiché à l’intérieur indique le joueur (du Joueur 1 au Joueur 4). La main indique où pointe la télécommande Wii.

Attention ! En mode Light Quiz, votre curseur devient un anneau coloré :
*Orange pour le Joueur 1 *Bleu pour le Joueur 2
*Vert pour le Joueur 3 *Rose pour le Joueur 4
 

Temps

Indique le temps restant pour répondre à la question.
 

Catégories de questions

Indique la catégorie de la question à laquelle vous devez répondre.
 

Argent

Indique le montant que vous allez remporter si vous répondez correctement en premier.
 

Question

La question à laquelle vous devez répondre est affichée à cet endroit.
 

Nombre de questions auxquelles vous avez répondu

Indique le nombre de questions auxquelles vous avez déjà répondu sur un total de 7 questions par quiz.
 

Réponses possibles

En Classic Quiz, vous aurez 4 choix de réponses affichés dans 4 cases. Veuillez choisir la réponse que vous croyez exacte.
