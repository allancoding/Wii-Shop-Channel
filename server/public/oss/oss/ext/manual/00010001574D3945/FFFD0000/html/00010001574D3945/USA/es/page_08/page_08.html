Uso del Wii Remote
Mantén el Wii Remote en posición vertical para jugar a este juego.

Apuntar
Apunta a la pantalla con el Wii Remote para mover el cursor y seleccionar. Verás un puntero en la pantalla que señalará dónde estás apuntando.


Golpear a los topos
Para golpear a los topos, apunta a la pantalla con el martillo y mueve el Wii Remote suavemente hacia abajo.
 

Volver
Para volver a una pantalla, pulsa el botón "Volver" en el menú o pulsa .
