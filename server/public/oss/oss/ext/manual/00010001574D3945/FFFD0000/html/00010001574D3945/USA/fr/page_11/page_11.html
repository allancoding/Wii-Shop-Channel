Sauvegarder une partie
Ce jeu sauvegarde automatiquement votre progression à la fin de chaque niveau ou partie multijoueur.

Pour effacer des données, sélectionnez "Effacer données" à l'écran de choix du joueur, choisissez les données à effacer et validez avec le bouton "Oui". Attention : il est impossible de récupérer un fichier de données effacé.

Si vous supprimez dans la chaîne Mii le Mii que vous avez choisi pour ce jeu, vos données ne seront pas effacées mais un Mii invité vous sera automatiquement attribué.

Sauvegarder

Vous risquez de perdre vos données ou d’endommager votre console Wii si vous effectuez les actions suivantes lorsque vous utilisez la fonction de sauvegarde.

● En appuyant sur le bouton POWER de la console ou de la Wii Remote, ou sur le bouton RESET pendant que les données sont sauvegardées.

● En ouvrant le menu HOME et en sélectionnant Menu Wii ou REINITIALISATION pendant que les données sont sauvegardées.

● En débranchant le adaptateur CA Wii alors que la console Wii est allumée.

Faites attention, les données effacées ne peuvent pas être récupérées.
