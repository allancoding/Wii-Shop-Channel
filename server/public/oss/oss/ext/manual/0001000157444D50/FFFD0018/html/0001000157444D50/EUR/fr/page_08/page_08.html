Commandes de Dr. Mario

Vous devez tenir la télécommande Wii à l’horizontale pour jouer à Dr. Mario.

NOTE: vous pouvez utiliser un Nunchuk, une manette classique ou une manette Nintendo GameCube pour jouer à Dr. Mario. Lorsque vous sélectionnez un mode de jeu dans le menu principal, le joueur qui appuie sur le bouton de sa manette pour confirmer devient le joueur 1.

Ce mode d’emploi explique comment jouer avec la télécommande Wii, mais les commandes de base pour les autres manettes sont listées ci-dessous. Pour plus d’informations sur les autres manettes, veuillez vous référer aux icônes affichées dans le jeu.
