Vriendinstellingen

Om met vrienden te kunnen spelen via de Nintendo Wi-Fi Connection, moet je vriendcodes uitwisselen en registreren. Kies in het speltypekeuzescherm voor NINTENDO WFC en vervolgens voor FRIEND SETTINGS. Hier kun je je eigen vriendcode bekijken en de vriendcodes van anderen registreren.

FRIEND ROSTER
(vriendenlijst)

Bekijk je vriendenlijst, waar je maximaal 30 vrienden kunt registreren. De vriend met wie je het laatst hebt gespeeld, staat bovenaan. Bij vrienden met wie je alleen vriendcodes hebt uitgewisseld staat nog geen naam vermeld, maar wel de vriendcode.
REGISTER FRIEND CODE
(vriendcode registreren)

Voer een vriendcode in om een speler aan je vriendenlijst toe te voegen. Om dit proces te voltooien, heb je ook je eigen vriendcode nodig. Je hoeft slechts verbinding te maken met de Nintendo Wi-Fi Connection om een vriendcode te ontvangen.
CONFIRM FRIEND CODE
(vriendcode bekijken)

Bekijk je vriendcode.

Opmerking: je vriendcode is niet hetzelfde als je Wii-nummer.

Over vriendcodes

Ieder spelerbestand met opgeslagen gegevens ontvangt een 12-cijferige vriendcode, als voor het eerst verbinding wordt gemaakt met de Nintendo Wi-Fi Connection. Vriendcodes kunnen niet worden bemachtigd zonder verbinding te maken met de Nintendo Wi-Fi Connection. Als je geen vriendcode kunt invoeren of er geen vriendcode wordt weergegeven, probeer dan verbinding te maken met de Nintendo Wi-Fi Connection om te zien of het probleem hiermee verholpen wordt.
