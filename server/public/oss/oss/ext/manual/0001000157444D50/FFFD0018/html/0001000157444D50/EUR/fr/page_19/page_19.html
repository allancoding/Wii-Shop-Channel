Utiliser WiiConnect24

Vous pouvez utiliser WiiConnect24 pour envoyer une version démo de Dr. Mario & Bactéricide à un ami n’ayant pas le jeu. Cela vous permettra de jouer ensemble via la connexion Wi-Fi Nintendo. Pour plus d’informations sur l’activation de WiiConnect24, veuillez consulter le mode d’emploi Wii – Chaînes et paramètres.

NOTE: pour utiliser cette fonction, le contrôle parental doit être désactivé. La réception de la version démo est gratuite.

● Enregistrer un ami Wii

Pour que vous puissiez envoyer la version démo, le destinataire doit être enregistré dans vos amis Wii. Les amis Wii sont différents des amis que vous enregistrez dans le jeu, ils doivent être enregistrés via le bureau Wii. Veuillez vous référer au mode d’emploi Wii – Chaînes et paramètres pour plus d’informations.

Après vous être enregistrés mutuellement, le surnom de votre ami Wii passera de gris clair à gris foncé dans votre carnet d’adresses.

Attention

・Si vous avez du mal à vous connecter à Internet, veuillez consulter les codes d’erreur s’affichant à l’écran et vous référer à la section "En cas de problèmes" du mode d’emploi Wii – Chaînes et paramètres.
