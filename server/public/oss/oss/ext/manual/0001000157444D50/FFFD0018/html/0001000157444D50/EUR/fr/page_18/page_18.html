Paramètres ami

Pour pouvoir jouer avec des amis via la connexion Wi-Fi Nintendo, vous devez échanger et enregistrer des codes ami. Sélectionnez CWF NINTENDO dans le menu principal de Dr. Mario puis PARAMETRES AMI. Ici vous pouvez consulter votre code ami et enregistrer des codes ami dans votre liste d’amis.

LISTE D'AMIS

Consultez votre liste d'amis. Vous pouvez enregistrer jusqu'à 30 amis. Vos amis seront classés dans l'ordre dans lequel vous les avez affrontés. Si vous n’avez pas encore joué avec quelqu’un, son code ami apparaîtra à la place de son nom.
ENREGISTRER UN
CODE AMI

Entrez le code ami de vos amis pour les ajouter à votre liste d'amis. Vous aurez besoin de votre propre code ami. Il vous suffit de vous connecter à la connexion Wi-Fi Nintendo pour le recevoir automatiquement.
CONFIRMER VOTRE
CODE AMI

Consultez votre code ami.

NOTE: votre code ami est différent de votre code Wii.

Codes ami

Chaque fichier de sauvegarde se verra attribuer un code ami à 12 chiffres la première fois qu’il sera utilisé pour accéder à la connexion Wi-Fi Nintendo. Vous ne pouvez pas obtenir de code ami sans vous connecter à la connexion Wi-Fi Nintendo. Si vous ne pouvez pas saisir de code ami ou si un code ami ne s’affiche pas, essayez de vous connecter à la connexion Wi-Fi Nintendo pour voir si cela résout le problème.
