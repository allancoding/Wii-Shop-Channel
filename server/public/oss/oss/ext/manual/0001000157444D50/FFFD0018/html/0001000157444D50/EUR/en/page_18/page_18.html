Friend Settings

In order to play with friends over Nintendo Wi-Fi Connection, you must exchange and register Friend Codes. Select NINTENDO WFC from the Mode Selection Screen, then select FRIEND SETTINGS. From there, you can confirm your own Friend Code and register Friend Codes in your Friend Roster.

FRIEND ROSTER

Check your Friend Roster, where you can register up to 30 Friends. Friends will be listed in the order you last played them. Friends with whom you have only exchanged Friend Codes will have their Friend Code as their name.
REGISTER
FRIEND CODE

Enter a Friend Code to add that player to your Friend Roster. You will need your own Friend Code to complete this process. To obtain a Friend Code, connect to Nintendo Wi-Fi Connection.
CONFIRM
FRIEND CODE

View your own Friend Code.

Note: This is different from your Wii Number.

About Friend Codes

Each save file will be assigned a 12-digit Friend Code the first time it is used to access Nintendo Wi-Fi Connection. Friend Codes cannot be obtained without connecting to Nintendo Wi-Fi Connection. If you cannot enter a Friend Code or a Friend Code is not displayed, try connecting to Nintendo Wi-Fi Connection to see if this resolves the problem.
