Freundeseinstellungen

Um über die Nintendo
Wi-Fi Connection mit Freunden spielen zu können, müsst ihr Freundescodes austauschen und registrieren. Wähle NINTENDO WFC und danach FREUNDESEINSTELLUNGEN. Hier kannst du deinen eigenen Freundescode bestätigen und die Freundescodes anderer Spieler registrieren.

FREUNDESLISTE

Hier werden die Namen deiner Freunde angezeigt, und zwar in der Reihenfolge, in der du zuletzt gegen sie angetreten bist. Vor dem ersten Kampf gegen einen Freund wird nur dessen Freundescode angezeigt. Du kannst bis zu 30 Freunde gleichzeitig in deiner Freundesliste haben.
FREUNDESCODE
REGISTRIEREN

Gib einen Freundescode ein, um diesen Spieler deiner Freundesliste hinzuzufügen. Um den Prozess abzuschließen, benötigst du auch deinen eigenen Freundescode. Stelle eine Verbindung zur Nintendo Wi-Fi Connection her, um deinen Freundescode zu erhalten.
FREUNDESCODE
BESTÄTIGEN

Hier siehst du deinen Freundescode.

Hinweis: Dieser Code unterscheidet sich von deinem Wii-Code.

Über Freundescodes:

Jeder Speicherdatei wird ein zwölfstelliger Freundescode zugeteilt, wenn sie zum ersten Mal für den Zugriff auf die Nintendo Wi-Fi Connection benutzt wird. Freundescodes können nicht ohne Verbindung zur Nintendo Wi-Fi Connection erstellt werden. Wenn du keinen Freundescode eingeben kannst oder kein Freundescode dargestellt wird, versuche dich mit der Nintendo Wi-Fi Connection zu verbinden; möglicherweise wird das Problem dadurch gelöst.
