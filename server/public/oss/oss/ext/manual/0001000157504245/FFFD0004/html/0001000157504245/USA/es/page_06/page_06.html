Cómo jugar

Antes de comenzar con uno de los 5 juegos, deberás crear un perfil. Esto te permite crear un personaje a tu medida y seleccionar el diseño de la pluma para las partidas. Se puede guardar el perfil y así utilizarlo de nuevo cuando vuelves a jugar.

Cuando hayas creado un perfil podrás seleccionar el juego al que desees jugar. Se puede elegir entre 501, Cricket, Alrededor del reloj, Asesino y Béisbol.

Después de seleccionar el tipo de partida pasarás a una pantalla con ranuras de jugador vacías. Selecciona una ranura para ti y el perfil que deseas usar. Todos los jugadores que participen en el juego deberán realizar este paso. Cualquier rival controlado por ordenador que haya en el juego deberá configurarse también en una de las ranuras. Después ya estarás listo para comenzar a jugar.

Al comenzar el juego seleccionado se proporciona un resumen de las reglas donde se explica claramente cuál es el objetivo del juego. Si se necesitan repasar de nuevo las reglas, se pueden consultar durante el desarrollo del juego en el menú Pausa.

Una vez leídas las reglas, ya se está listo para comenzar a jugar. En cada tipo de partida, para determinar el orden de juego, se lanza un dardo a la diana (a esto se le llama "Diddle-for-the-Middle") y el jugador que más se acerque al centro de la diana comienza el primero.
