Cricket: Score

La grille de score affiche les cases allant de 20 à 15 dans l'ordre décroissant jusqu'à la bulle (symbolisée par un 'B'). Pour fermer une case, il faut l'atteindre avec une fléchette. Le cercle extérieur de la case compte double tandis que le cercle intérieur compte triple. Quand une fléchette compte dans le score, un slash ( \ ) est placé à côté du chiffre touché. La deuxième fléchette réussie est symbolisée par un 'X' placé à côté du chiffre touché. La troisième fléchette réussie est symbolisée par un cercle qui entoure le 'X' à côté du chiffre afin d'indiquer que cette case est fermée. En touchant trois fois un chiffre sur n'importe quelle combinaison, la case de ce chiffre est fermée.
