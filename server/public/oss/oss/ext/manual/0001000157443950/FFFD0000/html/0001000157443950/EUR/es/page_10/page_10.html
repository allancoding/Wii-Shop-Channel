●Mando de Nintendo GameCube

+ Panel de Control　Mover personaje
Botón Ｂ　Atacar
Botón Ａ　Saltar
START/PAUSE　Menú de pausa
  • Oprime izquierda o derecha en la cruz de control para desplazarte en esa dirección.
  • Oprime el botón de salto para realizar un salto vertical.
    Si saltas mientras estás en movimiento, realizarás un salto en diagonal.
  • Oprime arriba o abajo en la cruz de control junto a las escaleras para subir o bajar.
  • Oprime abajo en la cruz de control para agacharte cuando no haya escaleras cerca.
  • Oprime el botón de ataque para atacar con el látigo.
  • Si tienes un arma secundaria, puedes utilizarla consumiendo corazones obtenidos.
    Oprime el botón de ataque mientras mantienes oprimido arriba en la cruz de control para utilizar el arma secundaria.

●Asignación de botones
