Sélectionnez "Multijoueurs" depuis le menu principal pour passer à l'écran de sélection des joueurs. Sélectionnez le nombre de joueurs avec puis appuyez sur pour passer à l'écran des modes de jeu.

● Fame Game
Le mode multijoueurs
classique.

● Shame Game
Le joueur avec le score
le plus bas gagne.

● Blame Game
Jusqu'à 4 joueurs s'affrontent sur le même écran.

Sélectionnez un mode avec puis appuyez sur pour passer à l'écran des paramètres.

● Démarrer
Jouer à Same Game avec les derniers paramètres utilisés.

● Paramètres
Changer les paramètres de jeu.

Couleurs Définir le nombre de couleurs de blocs pour chaque niveau. Plus il y a de couleurs, plus le jeu est difficile.
Limite temps Définir la limite de temps pour une partie multijoueurs.
Bombes Activer ou désactiver les bombes.
Bloc coloré Activer ou désactiver les blocs colorés.
Séparation
(2 joueurs
seulement)		 Changer la disposition de l'écran.
Type de bloc Choisissez parmi 6 types de blocs.
Graphismes Choisissez parmi 5 graphismes d'arrière-plan.
Musique Choisissez parmi 6 musiques de fond.
 

Qu'est-ce que sont les blocs colorés ?

Les blocs colorés
empêchent vos adversaires d'éliminer une certaine couleur de bloc. Éliminez 10 blocs ou plus dans une partie à 3 couleurs ou 5 blocs ou plus à 4 couleurs pour faire apparaître des blocs colorés. Une icône indique la couleur de bloc qui ne peut pas être éliminée tant que les blocs colorés sont activés.
