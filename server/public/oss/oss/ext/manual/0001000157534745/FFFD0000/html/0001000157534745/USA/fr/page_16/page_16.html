Sauvegarde des données
Les données de jeu pour le mode 1 joueur sont sauvegardés automatiquement après l'écran résultats.
Si vous souhaitez supprimer ces données, rendez-vous sur l'écran de gestion des données de la console Wii.

Fonction de sauvegarde

Les données de sauvegarde peuvent être perdues si vous faites les actions suivantes pendant la sauvegarde. Attention ! Si les données sont effacées, elles ne peuvent pas être récupérées.

● Appuyer sur le bouton POWER ou sur le bouton RESET de la console Wii ou de la Wii Remote alors qu'une sauvegarde est en cours.

● Sélectionner "Menu Wii" ou "Réinitialisation" dans le Ｍenu HOME alors qu'une sauvegarde est en cours.

● Débrancher la connecteur CA alors que la console Wii est sous tension.
