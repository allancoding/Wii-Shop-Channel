Téléchargements

●Téléchargements
Il y a les cartes additionnelles et les costumes disponibles pour acheter pour «Family Pirate Party». Pour obtenir ces cartes et costumes, vous avez besoin des Wii Points, qui peuvent être achetés pour des honoraires de la chaîne boutique Wii.

●Achat de contenus ajoutés et re-téléchargement

Sélectionnez les contenus ajoutés que vous

souhaiter acheter ou re-télécharger et cliquez

sur le bouton « télécharger » ou

« re-télécharger ». Une fois que vous avez

acheté le contenu ajouté, vous pourrez le

télécharger de nouveau gratuitement, même si

vous le supprimez.

●Suppression de contenus ajoutés que vous avez téléchargés

Vous pouvez supprimer les contenus ajoutés sauvegardés sur la console Wii.
Ce que vous supprimez ici peut être re-téléchargé gratuitement.
