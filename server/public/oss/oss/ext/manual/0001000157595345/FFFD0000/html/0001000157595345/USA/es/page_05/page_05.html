Sobre el juego

¡En Yard Sale Hidden Treasures: Sunnyville usarás trastos para convertir tu casa en una maravilla! Visita las ventas de garaje abiertas en el vecindario, encuentra unos tesoros bien escondidos, y restáuralos como decoraciones y muebles bellos. ¡En cada venta de garaje, encuentra todos los artículos de tu lista tan pronto como sea posible, ya que estas ventas de garaje no estarán abiertas el día entero! Esta lista tiene tesoros realmente únicos, y cuando se renuevan, se convierten en obras maestras de diseño e ingenio. Una vez encontrados todos los artículos únicos, puedes pasar al siguiente paso de restauración con abrillantarlos, pintarlos, cortarlos y ensamblarlos para crear muebles verdaderos. ¡En Yard Sale Hidden Treasures: Sunnyville, cada trasto puede en realidad ser un tesoro único!
