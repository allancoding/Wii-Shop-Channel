Getting Started

Main Menu

On the main menu, point the Wii Remote at the screen and make a selection of "Pong Toss", "Speed Pong" or "High Scores" and press the A Button.

 

Wii Remotes

All Wii Remotes must be turned on before making player selections.

 
Choose the number of players.

Select your Scene, Table Graphic, Song, and select Next.

 

Choose your character and handedness, select Next.

 

If there are multiple players, they will have the opportunity to select their characters.

 

When all players have chosen their characters, select Start Game.
