Getting Started
On the Title Screen

On the Title Screen, press to go to the Mini-Game Selection Screen or press to go to the Region Selection Screen.

 

 

Mini-Game Selection Screen

On this screen, you can select any of the action-packed mini-games.

Use to select a mini-game and press to confirm your selection.


If you have selected a mini-game which enables you to record your high score on Nintendo Wi-Fi Connection Ranking, press to proceed to the Ranking Screen.


For more details, see 13. Nintendo Wi-Fi Connection Ranking in this manual.

Region Selection Screen

On this screen, you can select the region where you are currently located.

 

Note: When starting the game for the first time, please select your region before going to the Title Screen.
