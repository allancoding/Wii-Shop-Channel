Ecran du jeu

Lorsque vous appuyez à gauche et à droite sur la manette +, vous pouvez modifier la direction du putting. Si vous appuyez vers le haut, vous réinitialisez la direction. Si vous appuyez vers le bas, vous pouvez choisir les vues de la caméra.
Appuyez sur le bouton A et maintenez-le enfoncé pour passer du mode d'entraînement au mode putting.
Le point jaune qui se déplace indique la déviation à partir de la direction sélectionnée. Une ligne s'affiche sur le sol pour indiquer la direction obtenue.
Cette barre indique la puissance actuelle. Le point blanc indique, pour information, la puissance du dernier tir.
