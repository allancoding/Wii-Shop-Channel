Contrôles communs

Peu importe quel jeu de contrôles vous choisissez, il existe plusieurs éléments communs.

En haut de l'écran, il y a le Timeline. Le Timeline affiche le temps écoulé pour votre Jam, ainsi que le temps restant pour tout Jam chronométré.

À gauche du Timeline, on trouve le Loop Length Indicator. Cela affiche la progression de toutes les boucles. Utilisez le Loop Length Indicator pour démarrer et arrêter les boucles et déclencher les one-shots en rythme en chronométrant vos actions lorsque l'indicateur est au sein des cercles sur les marques de quart de chronométrage. Les actions de chronométrage à la huitième position (directement entre les cercles sur les marques de quart) sont également une bonne chose. Jouer des pistes en rythme permet d’obtenir de la bonne musique qui sonne bien, et de marquer le plus de XP.

Si vous voulez masquer les contrôles et juste regarder le site, vous pouvez appuyer sur pour basculer la visibilité des contrôles principaux.

Lorsque vous voulez terminer votre Jam, vous pouvez le faire diminuer progressivement. Le bouton FADE sous le bouton MENU estompe le Jam sur plus de 3 secondes et il y met fin, en vous donnant la possibilité de sauver les performances pour une lecture ultérieure et pour une édition.

Si vous avez besoin suspendre le jeu, il y a deux façons de le faire. Le premier est activer le bouton MENU à droite de Timeline à l'aide du pointeur de la Wii Remote et . Vous pouvez également utiliser . Dans le menu Pause, vous pouvez redémarrer le Jam dès le début ou terminer immédiatement, ou passer directement à l’écran de sauvegarde. Vous pouvez également sélectionner View Controls pour voir une liste de tous les contrôles spécifiques disponibles pour le mode sélectionné.
