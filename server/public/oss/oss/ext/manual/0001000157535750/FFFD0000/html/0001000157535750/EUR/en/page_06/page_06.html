How to Play

Welcome to Swords & Soldiers! The fresh side-scrolling real-time strategy game for WiiWare.

 

The goal of this game is to beat the army on the other side of the map. To do this, you can build units or cast spells.

 

Units walk by themselves and attack any enemy that comes in range. Spells have a wide variety of effects on the battlefield, see the chapter on spells for details.

 

Units cost you gold, which can be mined. Casting spells costs mana, which you'll acquire automatically. Each faction also has a trick to enhance their mana income.


Menus

To set up a game you'll need to navigate the menus. We have tried to make the menus self-explanatory, however here follow some short explanations of most menus just in case you have any problems.


Clans

First you will be greeted by the clan system. The clans are the places where all your savegame data is kept. Create a new clan if you don't have one already. You can select its name and colour.


Main Menu

After the Clans menu, you will reach the Main Menu. First you can select if you want to play alone or with a friend. If you chosen single player, you can then select from a variety of game modes.


Options

Changes to the general options can be made here. So if for instance the music is to loud you can change it here.


Campaign

Here you'll hear all about the three factions struggle for power. You will also learn about the factions' respective strengths and weaknesses, so it is a great place to start honing your skills. Unlocking new parts of the game is also done exclusively through campaigns.


Skirmish

Chose Skirmish if you want to have a single battle against an AI opponent.


Challenges

Playing the Story mode will unlock several subgames. These can then be played in the Swords & Soldiers Challenges!


Achievements

You can unlock all kinds of achievements during the game. This menu shows your progress in unlocking these.
