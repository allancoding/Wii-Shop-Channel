Modes de jeu
Cette page explique les différénts types de jeu du titre OVERTURN.
Mode STORY
Battez-vous contre l'ordinateur tout en progressant à travers chaque tournoi. Gagnez les tournois pour obtenir de nouvelles armes utilisables aussi en Mode Connexion Wi-Fi Nintendo.
Mode Connexion Wi-Fi Nintendo
Engagez vous dans des batailles en ligne de 2 à 4 joueurs à travers la Connexion Wi-Fi Nintendo. Choisissez entre les batailles 1 contre 1 ou le mode Bataille Royale (BATTLE ROYALE) opposant 4 joueurs. Selon le mode choisi, vos résultats seront pris en compte dans le classement. Vous combattrez contre des adversaires enregistrés comme amis ou des personnes inconnues.
Mode VS LOCAL
Le mode de bataille 2 joueurs en écran partagé. Les OverTurn Mech disponibles sont les mêmes que dans le Mode STORY. Une Wii Remote et un Nunchuk supplémentaires sont requis pour jouer à ce mode.
