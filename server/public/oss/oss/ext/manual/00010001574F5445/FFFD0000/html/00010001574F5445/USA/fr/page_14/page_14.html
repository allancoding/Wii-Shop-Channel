Mode Connexion Wi-Fi Nintendo

Description des Modes Connexion Wi-Fi Nintendo.

WITH ANYONE(Avec n'importe qui)
Mode VS avec classement.
Trouvez un adversaire au hasard. Choisissez de jouer en Mode VS ou en Mode BATTLE ROYALE. Notez que chaque mode à son classement séparé. Votre classement sera éffectué à partir des points que vous gagnez ou perdez.
WITH FRIEND(Avec un ami)
Combattez avec un ami qui à un code ami enregistré. Choisissez de jouer en Mode VS ou en Mode BATTLE ROYALE. Les résultats des combats entre amis ne rentreront pas en compte dans le classement.
*Seulement les amis s'ayant mutuellement enregistrés pourront combattre ensemble.
