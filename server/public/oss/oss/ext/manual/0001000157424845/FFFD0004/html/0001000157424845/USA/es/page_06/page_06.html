Preparación

Comenzar el juego

- Selecciona "Nuevo juego" para jugar desde el principio. Cuando comiences un juego tendrás la opción de elegir el nivel de dificultad: Easy, Normal o Hard. Entonces también tendrá la opción a comenzar en la más alta misión que has alcanzado por el nivel de dificultad.

Pantalla de nivel de dificultad

- Cada vez que comiences un nuevo juego podrás elegir el nivel de dificultad para el resto del juego. Puedes elegir entre Easy, Normal o Hard.

Cargar un juego

- Selecciona "Cargar juego" seleccionar de una lista de juegos guardados y reasuma del punto de guardado.

Salir del juego

- Durante el juego, pulsa el botón HOME para mostrar el Menú HOME. Después, selecciona "menú de Wii" para salir del juego sin guardar.
