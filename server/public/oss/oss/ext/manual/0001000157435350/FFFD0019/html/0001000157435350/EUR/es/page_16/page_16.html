Datos guardados
Cuando superas un puzzle o cambias las reglas de una partida, la función de autoguardado guardará automáticamente los datos de tu partida.
Para borrar los datos guardados, utiliza la pantalla
de gestión de datos del menú de Wii.

Función de guardado

Se pueden perder los datos guardados si al guardar
se lleva a cabo alguna de las siguientes acciones.
Ten en cuenta que los datos guardados perdidos
no se podrán recuperar.

● Pulsar el Botón POWER o el Botón RESET
de la consola Wii o del mando de Wii mientras
se guardan los datos.

● Seleccionar “menú de Wii” o “Reset” en el menú HOME mientras se guardan los datos.

● Desenchufar el enchufe CA de la corriente mientras la consola Wii está encendida.
