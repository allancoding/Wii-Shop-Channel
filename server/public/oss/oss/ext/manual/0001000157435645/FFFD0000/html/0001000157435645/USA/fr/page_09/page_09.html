Interface
Les informations sur votre état sont situées dans le coin haut gauche de l'écran. Vous y trouverez les indications sur l'arme sélectionnée, les munitions, et votre niveau de santé. Gardez un œil sur ces informations, cela vous évitera d'avoir à recommencer un niveau trop souvent.

1 Affiche l'arme équipée et, dans certains cas, la quantité de munition disponible.
2 Affiche le niveau de l'arme. Le niveau maximum pour toutes les armes est de 3.
3 Affiche votre santé sous forme de jauge, et également sous forme numérique.
4 Des portes et des téléporteurs sont présents dans l'île Mimiga. Pour les utiliser, tenez vous devant, et appuyez sur bas de la croix directionnelle.
5 Récoltez les cristaux pour améliorer le niveau de vos armes.
6 Vous!
