Commandes
  • Appuyez sur le bouton A pour interagir avec le monde.
  • Appuyez deux fois rapidement sur le bouton A pour courir vers l'objet avec lequel vous souhaitez interagir.
  • Appuyez sur le bouton B pour passer le dialogue.
  • Appuyez sur le bouton - pour ouvrir votre inventaire
  • Appuyez sur le bouton + pour faire apparaître le menu.
  • Appuyez et maintenez le bouton A pour marcher (ou utilisez le Nunchuk ou la manette +)
  • En marchant, maintenez enfoncé le bouton Z ou le bouton B pour courir.
  • Appuyez sur le bouton B pour remonter dans les menus.
  • Utilisez la manette + pour naviguer dans votre inventaire et dans les options de dialogue, ou pour avoir accès à des commandes spéciales (telle que la table de laboratoire de DeSinge).
  • Lorsque vous êtes dans votre inventaire, utilisez le bouton 1 pour combiner les objets (ceux-ci doivent être dans les emplacements au préalable).
  • Lorsque vous êtes dans votre inventaire, utilisez le bouton 2 pour basculer vers et hors du mode Examen.
