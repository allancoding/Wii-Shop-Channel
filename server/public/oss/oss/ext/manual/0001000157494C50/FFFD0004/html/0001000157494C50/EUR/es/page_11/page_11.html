Controles
  • Pulsa el Botón A para interactuar con el mundo.
  • Pulsa el Botón A rápido dos veces para correr hacia el objeto con el que desees interactuar.
  • Pulsa el Botón B para saltarte el diálogo.
  • Pulsa el Botón - para abrir tu inventario.
  • Pulsa el Botón + para mostrar el menú.
  • Pulsa y mantén pulsado el Botón A para caminar (o usa el Nunchuk o la cruz de control)
  • Al caminar, mantén pulsado el Botón Z o el Botón B para correr.
  • Pulsa el Botón B para salir de los menús.
  • Usa la cruz de control para desplazarte por los objetos de tu inventario y tus temas de diálogo, o para los controles especiales (como la tabla de Laboratorio de DeSinge).
  • Usa el Botón 1 cuando estés dentro del Inventario para combinar objetos (antes, los objetos deben estar colocados en los espacios).
  • Usa el Botón 2 cuando estés dentro del Inventario para entrar o salir del Modo Examinar.
