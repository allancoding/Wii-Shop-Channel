Game Modes
DRIVER PROFILES
Use this menu to load an existing driver profile or to create a new one. The game stores all player progress & best times.

RACE
Race against the AI cars on any of the tracks unlocked in the Championship mode.

TIME TRIAL
The objective here is to complete the tracks in the quickest time to win a place on the fastest time score board. This mode features a Best Time 'Replay Car', which appears after the first lap of the track.

CHAMPIONSHIP
The Championship Mode is set over four different cups, with each cup featuring four different races.

Once a cup race is over, driver points are distributed according to the driver's position. The total number of driver points determines a driver's overall ranking.

TIME CHALLENGE
In this mode each track has three reference times, each one associated with a medal - bronze, silver and gold.

Time Challenge mode is locked until all Championship cups have been completed.
