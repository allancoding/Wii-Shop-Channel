Modes De Jeu

PLAYER PROFILES (PROFILS DE JOUEUR)
Ce menu vous permet de charger un profil de pilote existant ou d'en créer un nouveau. Le jeu sauvegarde la progression du joueur et les meilleurs temps.

RACE (COURSE)
Mesurez-vous aux véhicules pilotés par l'ordinateur sur l'un des circuits débloqués dans le mode Championship (Championnat).

TIME TRIAL (CONTRE-LA-MONTRE)
Dans ce mode, vous devez réaliser le meilleur temps possible sur le circuit pour figurer dans le tableau des meilleurs temps. Après le premier tour de circuit, une "Replay Car", représentant le fantôme du meilleur temps, apparaîtra sur la piste.

CHAMPIONSHIP (CHAMPIONNAT)
Le mode Championship (Championnat) se divise en quatre coupes différentes, comprenant chacune quatre courses.
 
À la fin de chaque course, des points sont attribués au pilote selon son classement. Le total de ces points détermine le classement général du pilote dans le championnat.

TIME CHALLENGE (DÉFI CONTRE-LA-MONTRE)
Dans ce mode, chaque piste se voit attribuer trois temps de référence, correspondant à trois médailles : bronze, argent et or.
 
Vous ne pourrez accéder à ce mode qu'après avoir remporté toutes les coupes du Championnat.
