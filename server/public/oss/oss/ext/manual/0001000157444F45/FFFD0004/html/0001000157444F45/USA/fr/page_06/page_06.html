Commandes de jeu

Vous pouvez jouer à Driift Mania avec 3 types de manettes :

* La manette Wii Remote.

* Le Nunchuk
(uniquement en mode multijoueur).

* La manette Classic Controller
(uniquement en mode multijoueur).

 

Si vous préférez utiliser un Nunchuk ou une manette Classic Controller, n'oubliez pas qu'ils doivent être connectés à une manette Wii Remote.


 

 

La manette Wii Remote doit être tenue à l'horizontale.
