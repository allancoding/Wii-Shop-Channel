Controles del juego

En Driift Mania, puedes jugar usando 3 tipos diferentes de mandos:

* El Wii Remote.

* El Nunchuk (sólo en el modo multijugador).

* El Classic Controller (sólo en el modo multijugador).

 

Si quieres usar un Nunchuk o un Classic Controller, recuerda que necesitas conectarlo a un Wii Remote.


 

 

El Wii Remote debe sostenerse en posición horizontal.
