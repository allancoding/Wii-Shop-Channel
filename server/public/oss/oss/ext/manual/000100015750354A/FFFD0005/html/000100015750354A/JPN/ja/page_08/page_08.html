アイテム紹介
レース中には、４種類の
アイテムが登場します。

アイテムを使うことで
様々な効果が発揮され、
レースを有利にします。

・ミサイル
　　… 前方のライバルにミサイルをぶつけます。
　　　 爆発で他の車を巻き込むこともできます。
・パワーアップル
　　… 一定時間無敵状態となります。体当たりや
　　　 ミサイル攻撃を受けてもビクともしません。
・スピードキャンディ
　　… 一定時間ものすごい勢いで加速します。
　　　 ライバルを抜き去り大逆転も狙えます。
・スリップオイル
　　… 後方へオイルをまきちらすトラップです。
　　　 オイルにぶつかるとその場でスリップして
　　　 しまいます。
