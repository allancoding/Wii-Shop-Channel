Anfangen
Titelbildschirm

Der Titelbildschirm des Spiels wird anzeigt, wenn das Spiel geladen ist. Drücke , um das Spiel zu starten.
Eine Gerichtsakte auswählen

Um ein Spiel zu beginnen, musst du dir zuerst eine Gerichtsakte aussuchen.
Neues Spiel und Spiel laden

Wenn du eine Gerichtsakte ausgewählt hast, wird das Spielmenü angezeigt.

Hier kannst du dann das Spiel starten.
Neues Spiel

Starte vom Beginn einer Episode, die du bereits abgeschlossen hast. Beim ersten Durchgang steht dir nur eine Episode zur Verfügung.
Spiel laden

Setze ein Spiel von deinem letzten Speicherpunkt oder vom Beginn des letzten Kapitels, in dem du gespeichert hast, fort.
