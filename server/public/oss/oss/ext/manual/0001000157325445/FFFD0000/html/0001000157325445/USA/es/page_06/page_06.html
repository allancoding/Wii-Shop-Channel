 ● Llamado a Lista Infernal
Wii Remote:
Mueve el Wii Remote como ordene el Sargento Instructor.
Atención
Apunta hacia abajo con el Wii Remote y mantenlo firme por un corto periodo de tiempo.
Saluden

Imita los movimientos reales para esta posición y lleva el Wii Remote a tu espalda.

Al tiempo
Baja el Wii Remote de la posición de “saludo”.
Descansen

Imita los movimientos reales y lleva el Wii Remote a tu espalda.

 ● Juego de Castigo - Arrastrarse:
Wii Remote:
Inclina el Wii Remote a un lado y mueve el Wii Remote hacia atrás y hacia adelante para mover al personaje.
 ● Juego de Castigo - Flexiones de Pecho:
Wii Remote:
Inclina el Wii Remote a un lado y muévelo hacia arriba y hacia abajo para hacer flexiones de brazos.
 ● Juego de Castigo – Equilibrio:
Wii Remote:

Inclina el Wii Remote a un lado y sostenlo horizontalmente. Muévelo hacia la izquierda o hacia la derecha para mantener el equilibrio.
