Mode Partie Solo
Résultats

Lorsqu'une chanson se termine, vous serez noté A, B, C ou F. Il vous faudra au moins un C pour déverrouiller le niveau suivant.
Déverrouiller des Objets

En étant très performant lors d'un niveau, vous pouvez déverrouiller des objets. Certains objets peuvent être déverrouillés en obtenant un B, mais d'autres ne peuvent être déverrouillés qu'en obtenant un A !
Rejouer les Niveaux

Après avoir joué un niveau une première fois, vous pourrez sélectionner une version Courte ou Longue de la chanson. La version Courte est idéale si vous n'avez pas assez d'énergie pour une chanson entière, mais n'oubliez pas que vous ne pouvez pas déverrouiller d'objets en jouant la version Courte !
