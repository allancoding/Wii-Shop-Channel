Abspielen

Jetzt ist Ihr nächster Hit schon fast fertig und Sie wollen ihn bestimmt abspielen. Um Ihren Mix abzuspielen, drücken Sie einfach auf das Abspielen-Symbol. Sie können auch an den Zahlen der oben entlanglaufenden Timing-Bar einstellen, wo das Abspielen anfangen soll. Wenn Sie Ihren Mix schnell vom Anfang abspielen wollen, drücken Sie auf den ‘Vom Anfang Abspielen’-Knopf direkt unter dem Abspielen Hauptsymbol.

 

Hinweis: BPM oder Beats per Minute (Schläge pro Minute) setzt das Timing des Mixes fest. Mix Superstar stellt die BPM automatisch ein, je nachdem, welcher Titel-Pack ursprünglich benutzt wurde. Sie können Loops aus Packs mit unterschiedlichen BPMs verwenden, jedoch könnte es dadurch schwierig werden, die Loops passend zusammenzufügen. Sie können keine Lieder mit unterschiedlichen BPMs zum Mix hinzufügen.
