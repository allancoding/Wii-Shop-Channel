アイテムについて
LA-MULANA遺跡の探索には数多くのアイテムが必要となります。
アイテムはお店で買ったり、遺跡を探索することによって得られるものもあります。これらのアイテムを駆使して多くの謎を解き明かしましょう！
ハンディスキャナ
ノートパソコンにつなげることで動作する小型スキャナ。石版に刻まれた文字などを読み込むことが出来ます。
聖杯
記録し聖杯石碑にワープできる。ポーズ画面でアイテムボタンを押すと使う事が出来ます。
防水ケース
ノートパソコンを水から守ることが出来るケース。水中でもメニューを開くことが出来るようになります。
魚水晶
中に霊魚が住み着いている水晶。水の毒を消すことが出来るので、水からのダメージを受けなくなります。
ほら貝
大きな貝を利用した楽器。遺跡内の謎が解けると不思議なメロディーで知らせてくれます。
羽
旅立ちの村のお供え物。ジャンプ中にもう一度ジャンプすることが出来るようになります。
ヘルメスの靴
伝説の英雄がはいていたといわれる靴。歩くスピードがアップします。
かぎ爪

忍者が使っていたといわれる装備品。壁にはりつくことが出来るようになります。
古文書リーダー
スキャナで取りこんだ古代文字を翻訳するソフト。これを起動しないと古文書が読めません。
グルグルマップ
地図を表示するためのソフトです。ポーズ画面に今いるフィールドの地図が表示されます。
