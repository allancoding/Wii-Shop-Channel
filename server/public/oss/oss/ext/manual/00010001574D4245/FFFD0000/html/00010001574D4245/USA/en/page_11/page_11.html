Effect System

● What Is the Effect System?

Up to three players can play in the three separate game windows at the same time. Although each game window contains separate game action, events in each game window will have various effects on the other game windows. This is called the Effect System.

● Aspects of Game Play That Have an Effect on Other Game Windows

Circle
(○)

Defeated enemies pass through to other game windows.
Stick
(／)

The stick pokes through window borders, and defeated enemies pass through to other game windows.
Square
(□)

Fire and bomb explosions defeat enemies and destroy blocks in other game windows.

These effects also occur when playing with Replay Miis or CPU Autoplay in the other game windows.

Using the Effect System

A mysterious character might appear in one of the other game windows while you play. When this happens, use the Effect System to hit the balls in his game window and give him points. If you keep giving him points like this, something good might happen...
