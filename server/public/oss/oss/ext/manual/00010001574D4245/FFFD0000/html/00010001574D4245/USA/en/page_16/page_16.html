Downloading to a
Nintendo DS System
Maboshi's Arcade includes a special Nintendo DS version of the game that you can download to a Nintendo DS system.

● What You’ll Need

Nintendo DS handheld video game system.

Start transmission on the Wii console

Select the Nintendo DS icon from the Game Window Setup Screen.

*This option will become available as you progress through the game. This option cannot be selected during game play.

Download onto the Nintendo DS system

Turn on the Nintendo DS system, and after the Health and Safety Screen, the Nintendo DS Menu Screen (below left) will be displayed. Touch DS Download Play to display the Game List Screen (below right), then touch the Maboshi's Arcade Panel. Confirm your choice by touching Yes to begin the download.

● Game Selection Screen

The Nintendo DS system is held vertically when playing the game. Press up and down on to choose the game mode you want to play, and press right on to confirm. To change the difficulty level, select the lowermost button and press right on .

● Game Over Screen

To return to the Game Selection Screen, select the lowermost button and press right on .

CAUTION

  • The game download and high scores will be saved if you put the Nintendo DS system into Sleep Mode by closing it, but all data will be lost if you turn the power off.
  • For more details, please refer to the Nintendo DS Instruction Booklet.
