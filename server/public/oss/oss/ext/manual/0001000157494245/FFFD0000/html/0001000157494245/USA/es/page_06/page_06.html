Bits

¡Pequeños e inocentes cubos cooperativos trabajando contigo para crear Bittos! Claro, si dejados sin atender y solos, los Bits se convierten testarudos y traviesos, solidificándose y tomando control del tablero de juego. Que mal.

 

Bloques

Una colección tremenda y bien parecida de cuatro Bits, organizada en una de siete figuras geométricas posibles.



Cajas

Figuras cuadradas divertidas, gratificad amenté formadas usando Bloques. ¡Qué bien!

El número de Bits necesarios para formar una caja calificante. Hay tres tamaños de Objetivos de Cajas: 2X2, 2X3, y 3X3.
