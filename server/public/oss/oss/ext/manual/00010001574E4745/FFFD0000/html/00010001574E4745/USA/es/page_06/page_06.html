Mueve el cursor con el Wii Remote y busca grupos de 3 o más objetos iguales. Apunta a uno de estos objetos con el Wii Remote y pulsa el Botón A para destruirlo. Tu puntuación depende del número de objetos del grupo. Cuantos más objetos, mayor puntuación. En ocasiones, encontrarás una baldosa amarilla bajo un objeto. Golpéala con tu martillo para que desaparezca. Habrás superado el nivel sólo cuando hayas destruido todas las baldosas de colores.


A veces, hay sólo UN objeto en el camino que te impide alcanzar una puntuación elevada. Utiliza la mano para apartar el objeto. Puedes hacer esto 5 veces como máximo. Apunta con el cursor al objeto que deseas apartar. Atrapa el objeto con el Botón B del Wii Remote y mueve el Wii Remote para arrojarlo fuera de la zona de juego.


Si consigues destruir al menos 4 objetos a la vez, el indicador de tu martillo subirá un poco. Cuando el indicador alcance el 25%, puedes activar el martillo GRANDE. Señala con el Wii Remote el punto que deseas alcanzar, mantenlo quieto un instante y mueve el Wii Remote arriba y abajo. Despejarás una amplia zona de juego con un solo golpe. Cuanto más alto esté el indicador, más fuerte será el golpe.
