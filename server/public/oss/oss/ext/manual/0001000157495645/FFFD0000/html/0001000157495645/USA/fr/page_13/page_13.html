13
Ennemis
Présentation de quelques armes des terriens
Tanks

Ils attaquent depuis le sol.
Les petits tanks n’utilisent qu’un type de munition, mais les grands tanks en utilisent deux.
Avions de chasse

Ces avions se déplacent et attaquent rapidement. Faites attention à ne pas les percuter sous peine de subir d’importants dégâts.

 
Hélicoptères

Les hélicoptères suivent votre OVNI avec acharnement.

Même s’ils n’infligent que des dégâts modérés, vous devrez les abattre aussi vite que possible.

 
Vaisseaux de guerre

Ces redoutables ennemis envoient leurs missiles avec une précision mortelle.

Méfiez vous d’eux lorsqu’ils attaquent en groupe.

 
Sous-marins

Ils envoient leurs missiles depuis les fonds marins.
Vous ne pouvez pas les attaquer quand ils sont sous l’eau.

 
Bâtiments fortifiés

Créés pour défendre les villes terriennes, ces bâtiments constituent une grande partie du système de défense de la planète.
Ils peuvent utiliser un armement très varié.
Citernes de gaz

Elles explosent lorsqu’elles sont attaquées, envoyant voler des débris capables d’infliger de sérieux dégâts.

 
Champ de confinement

Des barrières spéciales que les humains ont créées pour capturer les OVNI.

Détruisez les générateurs énergétiques pour vous échapper.
