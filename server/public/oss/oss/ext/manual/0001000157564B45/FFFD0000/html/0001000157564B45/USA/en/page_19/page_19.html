Screen Ratios & Saving
Screen Ratios
The screen ratio can be changed at the Wii System Settings. See the Wii Operations Manual for details.
Saving
Saving is performed automatically.
If you want to delete this data, please do so via the Wii Menu's Data Management Screen.
Please understand in advance that data registered via Nintendo Wi-Fi Connection cannot be deleted.

Save Function

Save data may be lost if any of the following actions are performed while using the save feature.
Once data is erased, it cannot be restored, so please be careful.

● Pressing the Power Button or RESET on the Wii console or Wii Remote while data is being saved.

● Selecting "Wii Menu" or "Reset" at the HOME Menu while data is being saved.

● Unplugging the AC plug from an outlet while the Wii console power is on.
