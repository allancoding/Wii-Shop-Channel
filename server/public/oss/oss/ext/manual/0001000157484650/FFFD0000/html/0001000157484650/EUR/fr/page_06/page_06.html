Grades militaires

Vous pouvez vérifier vos statistiques et votre grade militaire actuel sur l'écran 'Rank' (Grades militaires).


Vous pouvez vérifier ici: le nombre d'ennemis tués, précision, coups tirés et le nombre de points obtenus.

Grades militaires disponibles:
Soldat, le rang de départ pour les nouveaux joueurs
Soldat de 1ere classe - 800 points
Caporal - 1100 points
Sergent - 4000 points
Sergent d'état major - 6000 points
Sergent-chef - 8000 points
Sergent major - 10000 points
