In Menus
Wii Remote control mode:
Point at the icon you want to select with the Wii Remote.
A Button: Select.
B Button: Return to the previous screen.
HOME: Open the HOME Menu.
Wii Zapper control mode:
Point at the icon you want to select with the Wii Zapper.
B Button: Select.
Z Button: Return to the previous screen.
HOME: Open the HOME Menu.
