En forme de nuage

L'état gazeux de l'eau — qui prend alors la forme d'un nuage — est le dernier disponible, mais ce n'est certainement pas le moins puissant! Lorsque vous êtes en forme de nuage vous n’êtes plus limité par la gravité. Vous pouvez voler dans le ciel et atteindre des endroits auparavant hors de portée. Voici les mouvements et les pouvoirs dont vous disposez sous cette forme :

● Contrôler le nuage
Vous pouvez déplacer le nuage vers la droite et vers la gauche de la même façon que vous déplacez l’eau et la glace : en inclinant la manette Wii Remote vers la direction appropriée.

Pour faire monter le nuage, inclinez la Wii Remote vers vous. Inclinez-la dans la direction opposée pour le faire descendre.

Note : Ces commandes peuvent être inversées de telle sorte qu'incliner la manette Wii Remote vers vous fera alors descendre le nuage, et vice versa. Pour cela, sélectionnez le paramètre Inverted (inversé) pour l'option Cloud up & down controls (commandes pour faire monter ou descendre le nuage) dans le menu des options (voir p. 7).

·Faire pleuvoir
Secouez la manette Wii Remote pour faire pleuvoir votre nuage... Mais faites attention à l'endroit où il tombera!

·Aspirer
Maintenez enfoncé pour aspirer des ballons et les transporter avec vous à l’intérieur du nuage. Ceci ralentit le nuage et peut être utile lorsqu’il faut parcourir un endroit avec prudence.

·Souffler une rafale
Si vous appuyez sur lorsque vous aspirez, vous expulserez une rafale de vent dans cette direction. Ce pouvoir peut être utilisé de plusieurs manières :

Poussée de
vitesse

Faites souffler une rafale dans la direction contraire à celle vers laquelle vous vous déplacez pour vous donner une bonne poussée de vitesse. Par exemple, si vous vous déplacez vers la droite, faites souffler une rafale vers la gauche et vous y serez en un rien de temps.

Déplacer des
objets

Vous pouvez faire souffler une rafale pour déplacer de petits objets, même si vous n’arrivez pas à les atteindre.

Détruire les
nuages
d'Influence

Dans certains lieux de l’Aquaticus, l’Influence s’est transformée en petits nuages pour bloquer votre progression, mais ceux-ci n’ont aucune chance contre une bonne rafale!

·Éclair
Envie d'électrocuter quelque chose? Maintenez enfoncé pour charger un éclair, puis relâchez pour le lancer. Vous pouvez vous en servir pour vous débarrasser de certains ennemis ou pour charger des machines.
