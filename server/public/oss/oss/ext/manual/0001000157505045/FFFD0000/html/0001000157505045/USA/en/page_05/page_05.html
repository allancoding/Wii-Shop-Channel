Modes
 
(1) Single Mode
Single player mode. You can select a character from a choice of four, and match off against the three remaining characters.
 
(2) Versus Mode
Two player mode. Play a single player match vs the CPU or a two player match with a friend. For two players, the screen splits in half.
 
(3) Mini Game
Enjoy three kinds of mini games: “Target Table Tennis,” “Thrilling Table Tennis” and “Matching Table Tennis”.
