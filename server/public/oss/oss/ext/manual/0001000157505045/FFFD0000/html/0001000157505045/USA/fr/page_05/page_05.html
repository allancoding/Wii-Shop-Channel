5
Modes
 
(1) Single Mode (Mode individuel)
Mode individuel. Le joueur choisit l'un des quatre personnages disponibles et dispute un match contre les trois personnages restants.
 
(2) Versus Mode (Mode affrontement)
Mode à deux joueurs. Disputez un match seul contre l'ordinateur ou à deux avec un ami. Pour les parties à deux joueurs, l'écran se divise en deux.
 
(3) Mini Game (Mini-jeux)
Jouez à trois mini-jeux palpitants :
«Target Table Tennis» (Table tennis avec cible),
«Thrilling Table Tennis» (Défi «table tennis») et «Matching Table Tennis» (Formez la paire).
