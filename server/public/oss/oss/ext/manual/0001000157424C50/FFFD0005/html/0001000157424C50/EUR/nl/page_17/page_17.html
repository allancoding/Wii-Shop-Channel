EXTEND-bellen
EXTEND-bellen zijn bellen die allemaal een letter uit het alfabet bevatten. Ze komen tijdens rondes uit luchtschachten als je aan bepaalde voorwaarden voldoet. Verzamel ze om woorden te spellen en levens te verdienen.
In de Standard Mode
Door in de Standard Mode E-X-T-E-N-D te spellen, verdien je 1 leven.
Als je dubbele bellen verzamelt, verlies je geen andere bellen.
Opmerking: Dit geldt ook voor de Super Standard Mode
In de Arrange Mode
EXTEND-bellen werken in de Arrange mode iets anders.
Er zijn vier reeksen die je kunt verzamelen. De reeksen en het bijbehorende aantal levens staan hieronder.
Opmerking: Dit geldt ook voor de Super Arrange Mode, Expert Mode 1 en Expert Mode 2.

H-E-R-O - je krijgt 1 extra leven.
E-X-T-E-N-D - je krijgt 2 extra levens.
N-E-V-E-R - je krijgt 3 extra levens.
W-I-N-N-E-R - je krijgt 5 extra levens.

Tip
Als je in Arrange Mode speelt, bedenk dan welk woord je wilt spellen voordat je letters verzamelt.
Als je met een letter komt te zitten die niet in het woord dat je wilt maken staat, begin je een nieuw woord te spellen en verlies je alle letters die niet in dat woord staan. Woorden spellen kan aan het begin moeilijk lijken, maar de beloning is de moeite waard. Volhouden!
