Single-Player Mode
This is a stage-solving mode for one player only. There are two rule variations, NORMAL and CONTINUOUS.
Normal Mode

In Normal Mode, the aim is to solve stages as fast as you can.

● Results Screen

This screen is displayed after you solve a stage.

LEVEL

The level of the solved stage.
CLEAR TIME

The time taken to solve the stage.
BEST TIME

The record time for solving the stage.
POINTS

The number of points acquired by solving the stage.
TOTAL POINTS

Your total score.
Continuous Mode

In Continuous Mode, the aim is to solve as many stages in a row as you can. You cannot move a piece once it's been placed, so be careful not to make any mistakes or the game will end.

● Results Screen

This screen is displayed when the game ends.

STAGES CLEARED

The number of stages cleared so far.
CLEAR RECORD

The record number of stages cleared.
TOTAL TIME

The total time taken so far.
POINTS

The number of points acquired by solving the stage.
TOP SCORE

The record score.
