Pantalla de juego
PLAYER
(jugador)

Indica a qué jugador le toca jugar.
LEVEL
(nivel de
dificultad)

Muestra el nivel de dificultad de este rompecabezas.
Tiempo restante

Muestra el tiempo que queda para resolver el rompecabezas. En el modo entrenamiento no hay límite de tiempo.
BEST TIME
(mejor tiempo)

Muestra el tiempo récord en que se ha resuelto este rompecabezas. Solo se muestra en el modo normal (en el modo continuo se mostrará la puntuación actual y el número de rompecabezas resueltos hasta el momento).
Pieza

Para colocarlas en el tablero, las piezas que están fuera se pueden agarrar, hacer girar y voltear.
Tablero

Para resolver el rompecabezas hay que cubrir el tablero con todas las piezas, sin dejar huecos.
Cursor

Se usa para seleccionar y mover las piezas.
