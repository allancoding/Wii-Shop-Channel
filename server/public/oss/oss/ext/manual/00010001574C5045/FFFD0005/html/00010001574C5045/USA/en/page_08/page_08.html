Game Screen
PLAYER

Indicates which player is currently playing.
Difficulty Level

Shows the difficulty of the current stage.
Remaining Time

Shows the time you have left to complete the stage. This is not displayed in Training Mode.
BEST TIME

Shows the best existing record for the current stage. It will only be displayed during Normal Mode. (In Continuous Mode, the current score and number of stages cleared are displayed instead.)
Piece

Pieces outside the box can be grabbed, rotated, and flipped, then placed in the box.
Box

Complete the stage by placing all the pieces in the box without leaving any gaps.
Cursor

Used to manipulate the pieces.
