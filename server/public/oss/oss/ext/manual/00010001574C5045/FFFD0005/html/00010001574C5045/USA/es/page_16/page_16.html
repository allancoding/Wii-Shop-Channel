Cómo guardar

En LONPOS, los datos del juego se guardan automáticamente cada vez que vuelves al menú principal después de jugar una partida o tras haber hecho cambios en la configuración. Mientras el juego se está guardando, aparece un mensaje como el que se muestra en la imagen.

Acerca del guardado de datos

Las siguientes acciones podrían provocar que no se guardaran los récords o que se perdieran los datos guardados. ¡No podrás recuperar los datos, así que ten cuidado!

  • Apagar o reiniciar la consola o el Wii Remote mientras se están guardando datos.
  • Desconectar el adaptador de corriente de tu consola Wii mientras estás jugando.
