Mode SOLO
Le mode SOLO propose trois différentes règles de jeu.
Choisis le type de course auquel tu veux participer puis appuie sur .

● BATTLE
Utilise les objets du parcours et remporte la victoire en étant le premier à terminer 3 tours de parcours. Rends-toi dans le mode BATTLE en CWF Nintendo pour plus de détails.

● STOÏQUE
Fais bon usage de tes techniques pour arriver au bout le plus vite possible. Rends-toi dans le mode STOÏQUE en CWF Nintendo pour plus de détails.

● CONTRE-LA-MONTRE
Utilise les trois objets turbo dans cette course contre-la-montre. Il est très important d'utiliser les techniques et les objets pour grappiller des secondes.

Une fois que tu as choisi un type de course, choisis un personnage et son équipement. Enfin, choisis un parcours et la course peut commencer.
Le niveau de difficulté des modes BATTLE et STOÏQUE peut être modifié dans les OPTIONS.
