Améliorations de héros

Le symbole + qui s’affiche près de l’icône de votre héros indique que vous pouvez l’améliorer ! Pour afficher le menu, pointez le symbole + et appuyez sur le bouton A.

Un héros acquiert de l’expérience puis obtient des points d’amélioration en détruisant des robots ennemis. Vous pouvez allouer vos points d’amélioration à 4 aspects de votre héros : les points de vie, les points d’énergie, les armures et les dégâts.

 

Pour quitter le menu d’amélioration, sélectionnez le bouton B ou le champ de bataille.
