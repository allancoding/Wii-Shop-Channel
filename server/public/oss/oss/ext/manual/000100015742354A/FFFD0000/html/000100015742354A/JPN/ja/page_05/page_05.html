ストーリー
 

ワイルドな夜の世界でその名を知られるパーム・オーシャン。この街に住む大金持ちが、１０人の世界最高レベルのプレイヤーを集めて、ブロックブレーカーゲームの一大トーナメントを開催するという。

名立たるブロックブレーカーチャンピオンの一人として、あなたがこのトーナメントに参加したいと望むのは当然のこと。しかしそこにはちょっとした問題があった・・・大金持ちの主催者が設定した参加費は、なんと１００万ドル！そしてあなたの財布には何にも入っていないのだ。この問題を解決する方法はただひとつ。夜の街の、様々な場所で行われるゲームに挑戦し、賞金を稼ぐしかない！しかしこれは思ったほど簡単ではなかった。手強い相手が次々とあなたの前に現れる・・・！

８０年代風の雰囲気とオシャレなキャラに囲まれて、今までにない難しいチャレンジと激しい対戦の数々に立ち向かおう。そして最高金額の賞金や、素敵なプレゼントを手にしよう！街の重要人物であるキャラ達にプレゼントを渡せば、彼らはあなたを新たな世界のドアへと導いてくれるかもしれない！
