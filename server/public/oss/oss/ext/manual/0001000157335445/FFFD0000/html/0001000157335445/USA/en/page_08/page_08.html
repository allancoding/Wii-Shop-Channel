Options & HUD

This page describes the options menu and the game's Head-Up Display (HUD).

Options Menu

MUSIC VOLUME. Change the volume of the music. The lowest setting turns off the music.

SFX VOLUME. Change the volume of the sound effects. The lowest setting turns off the sound effects.

FLIGHT SENSITIVITY. Changes the sensitivity of the currently active control type. This option is not available for the Classic Controller.

INVERT Y-AXIS. This option inverts the controls for the Y-Axis (climb and dive).

ENEMY MARKERS. Enables or disables the enemy markers. See HUD details below.
