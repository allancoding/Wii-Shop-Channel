Hoe wordt er gespeeld?

 

SPELER : Om te beginnen, richt de Wii-afstandsbediening op je speler; deze zal vervolgens de bewegingen van je aanwijzer volgen.

 

VIJANDEN : Als je een vijand van een andere kleur aanraakt verlies je een leven.

Als je vijanden van je eigen kleur aanraakt, zul je ze automatisch absorberen (mits deze absorbeerbaar zijn).

 

KLEUROMZET : Zodra je met je ruimteschip boven een kleuromzet-element vliegt, zal de kleur van je schip in de kleur van het element zelf worden omgezet.

 

EIGEN RUIMTESCHEPEN FUSEREN : Zet je ruimteschepen bij elkaar en houd Z-knop (of B-knop) ingedrukt. Laat eenvoudig de knop los om ze weer los te koppelen.

 

MULTIPLAYER FUSIE : Om jouw ruimteschepen met die van je vrienden te laten fuseren, breng ze bij elkaar en houd beiden, zowel jij als je vriend, de Z-knop (of B-knop) ingedrukt.

 

EEN PAUSE MAKEN : Druk plusknop.
