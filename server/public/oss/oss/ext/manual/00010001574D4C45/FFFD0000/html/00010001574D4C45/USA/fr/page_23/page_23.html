Astuces
N’oubliez pas d’utiliser vos nourritures bonus ! Pour battre certains mangeurs, vous devrez utiliser les nourritures bonus afin de les ralentir ou de les faire craquer.

Surveillez toujours votre adversaire ! Il est facile de se concentrer uniquement sur sa propre nourriture et de manquer les occasions de ralentir la compétition ou de se défendre contre les attaques.

Gardez un oeil attentif sur les gestes de votre personnage. Vous ne pouvez vous bâfrer que d’une chose à la fois. Attendez donc que votre personnage ait bien laissé retomber ses mains avant de réaliser le prochain mouvement avec la télécommande Wii.

Faites attention à surveiller le contenu de votre bouche. Si vous mâchez alors qu’il n’y a pas de nourriture, vous vous mordrez la langue et si vous portez des aliments à votre bouche alors qu’elle est pleine votre vomiso-mètre augmentera.

Un méga rot fonctionne mieux lorsqu’il pousse à bout le vomiso-mètre de votre adversaire, le faisant vomir. Si vous avez un méga rot, attendez le bon moment pour l’utiliser ou combinez-le à une autre attaque.
