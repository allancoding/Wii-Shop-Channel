Bevor es losgeht
Titelbildschirm
Im Titelbildschirm, drücke , um in den Modusauswahl zu gelangen.
Wenn du das erste Mal spielst, wirst du aufgefordert einen Spielernamen einzugeben. Der Name darf bis zu 8 Zeichen lang sein.
Unangebrachte Namen dürfen nicht benutzt werden.
Gibt es bereits gespeicherte Daten, kann entweder NEW GAME oder LOAD GAME ausgewählt werden. Um mit einem Spiel fortzufahren, wähle LOAD GAME. Wähle NEW GAME, um ein neues Spiel zu starten.
Modi

● STORY MODE
Spiel jedes Szenario.

● ARCADE MODE
Wähle eines von 3 Leveln und spiele um den Highscore.

● VERSUS MODE
Stelle dich der weltweiten Herausforderung mittels des Nintendo WFC.

● RANKING MODE
Du kannst diverse Highscores für STORY, ARCADE und VERSUS MODE ansehen.

● OPTION MODE
Ändere diverse Spieleinstellungen.
