ARCADE MODE
Rather than clear each
stage as in STORY MODE,
ARCADE MODE is all about going for the High Score.
There are 3 types of stages from which to choose.
When you run out of balls,
it's Game Over.
High Score records achieved in ARCADE MODE can be
registered with
Nintendo Wi-Fi Connection Ranking, allowing you to
compete with players around the world. You can access Nintendo Wi-Fi Connection Ranking by pressing while the Resul Screen is displayed.
