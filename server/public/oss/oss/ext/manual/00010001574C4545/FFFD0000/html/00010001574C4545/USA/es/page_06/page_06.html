Cómo empezar a jugar
● Empezar una partida

En la pantalla del canal apunte a COMENZAR y pulse el Botón A para ejecutar el juego y acceder a la pantalla de título.
● Pantalla de título

Pulse un botón para entrar en el universo PooYoo.
● Nivel de aprendizaje

Elija el nivel de aprendizaje adaptado al niño: PooYoo pequeño para los más jóvenes y PooYoo grande para los mayores. Seleccione una imagen con el puntero y luego pulse el Botón A para confirmar. También puede seleccionar el nivel de dificultad pulsando directamente el Botón 1 o el Botón 2: 1 para PooYoo pequeño, 2 para PooYoo grande.
● Elección de taller

Una vez seleccionada la dificultad, aparecerá el menú de selección de taller. Seleccione un taller con el puntero y pulse el Botón A para comenzar con el taller. También puede seleccionar el taller pulsando el Botón 1 o el Botón 2: 1 para comenzar con el taller del agua, 2 para comenzar con el taller del cielo.
