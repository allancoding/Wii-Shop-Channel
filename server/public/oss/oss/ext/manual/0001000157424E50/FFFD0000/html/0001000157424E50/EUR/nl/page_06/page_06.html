Je eigen kapsalon

De eerste keer dat je een profiel selecteert, kun je alleen START kiezen. Nadat je een keer gespeeld hebt en er gegevens zijn opgeslagen, kun je ook kiezen voor PLAY om verder te spelen of ERASE om het profiel te wissen.

 

Elke keer als je een nieuw profiel aanmaakt, geeft het spel geeft je een korte rondleiding door je nieuwe kapsalon: je wachtkamer, je knipstoel en je knipgereedschap.
