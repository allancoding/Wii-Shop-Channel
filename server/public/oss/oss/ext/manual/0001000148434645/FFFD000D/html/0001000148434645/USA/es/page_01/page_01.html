Introducción al Canal Wii Speak

El Canal Wii Speak™ te permite usar el accesorio Wii Speak™ para hablar con amigos de todo el mundo, enviarles mensajes de voz y muchas cosas más.

● Para usar el Canal Wii Speak cada usuario debe disponer de una conexión a internet de banda ancha, una consola Wii™, el accesorio Wii Speak y el Canal Wii Speak. Para obtener más información sobre la conexión y el uso del accesorio Wii Speak, consulta el folleto de instrucciones del accesorio Wii Speak.

● Antes de iniciar una conversación debes registrar como amigo Wii a la persona con la que quieres hablar. Un amigo Wii es cualquier usuario cuya clave Wii hayas registrado en tu agenda Wii.
