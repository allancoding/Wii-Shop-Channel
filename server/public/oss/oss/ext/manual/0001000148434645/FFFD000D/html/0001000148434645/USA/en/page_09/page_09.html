Wii Speak Settings

Select in the chat room or the lobby to display the Wii Speak Settings screen.

● Precautions
Please follow these steps to set up so you and others can enjoy using the Wii Speak Channel.

● Performance Check
Use this test to make sure the Wii Speak accessory is functioning properly. Please follow the on-screen instructions.

*Please be as quiet as possible when performing this test.

If you perform the test and the Wii Speak accessory is judged to be functioning incorrectly, please check the tutorials and the Wii Speak Operations Manual to make sure everything has been set up correctly.

If you perform the test and the Wii Speak accessory is judged to be functioning properly, yet you are still having difficulty chatting, please see the Q&A section. (see p. 18)

*The check will not test your Internet connection.

● Sensitivity Settings
You can adjust the Wii Speak microphone levels so the people you’re chatting with can hear you well. When you talk, the meter shown above will light up. Turn toward the Wii Speak microphone and speak, using and to bring the level of your voice into the green OK range.

If your voice is registering in the High range, you may experience echoing or high-frequency “howling.” If your voice is registering in the Low range, it may be too soft for your chat partners to hear.

● Headphone Settings
Select whether to use headphones or not. Follow the on-screen instructions.

*If you aren’t using headphones, please turn this setting off. If this setting is turned on when you aren’t using headphones, it may result in echoing or other audio problems. This setting will not be saved.
