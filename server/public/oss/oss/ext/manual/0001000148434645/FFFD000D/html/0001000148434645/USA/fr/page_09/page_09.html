9
Paramètres de Wii Speak

Sélectionnez dans le salon de conversation ou à l’accueil pour accéder au menu des paramètres de Wii Speak.

● Précautions d’utilisation
Consultez les précautions d’utilisation pour obtenir des conseils qui vous permettront de communiquer avec les autres utilisateurs de la Chaîne Wii Speak dans les meilleures conditions.

● Test du microphone
Utilisez ce test pour vérifier si le microphone Wii Speak fonctionne correctement. Pour ce faire, suivez les instructions affichées à l'écran.

Note: Veuillez rester aussi silencieux ou silencieuse que possible pendant ce test.

Si le résultat du test indique que le microphone Wii Speak ne fonctionne pas correctement, consultez les précautions d'utilisation et le mode d'emploi du microphone Wii Speak afin de vérifier si vous avez configuré tous les paramètres correctement.

Si le résultat du test indique que le microphone Wii Speak fonctionne correctement, mais que vous éprouvez toujours des difficultés à communiquer avec les autres utilisateurs, référez-vous à la section En cas de problèmes.

Note: Le test n'examinera pas votre connexion internet.

● Sensibilité du microphone
Ici, vous pouvez ajuster la sensibilité du microphone pour que les amis avec qui vous communiquez vous entendent plus clairement. Quand vous parlez, les voyants lumineux de l'indicateur de sensibilité s’allument. Utilisez et en parlant en direction du microphone, jusqu'à ce que le volume de votre voix se situe dans la zone verte marquée « OK ».

Si le son de votre voix atteint la zone Haute, vous risquez d'entendre de l’écho ou des sifflements aigus. Cependant, s’il ne dépasse pas la zone Faible, votre voix risque de ne pas être suffisamment audible pour vos interlocuteurs.

● Utilisation des écouteurs
Choisissez cette option pour indiquer si vous préférez utiliser la Chaîne Wii Speak avec ou sans écouteurs.

Note: Si vous n'utilisez pas d’écouteurs, assurez-vous de définir les paramètres de Wii Speak pour une utilisation sans écouteurs. Dans le cas contraire, vous risquez d'entendre de l’écho ou d’autres distorsions du son. Ces paramètres ne sont pas sauvegardés d’une session à l’autre.
