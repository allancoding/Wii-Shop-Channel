Données personnelles et Internet
Paramètres de WiiConnect24

● Pour pouvoir utiliser la Chaîne Wii Speak, vous devez activer WiiConnect24. Pour ce faire, suivez les étapes ci-dessous:

  1. Depuis le Menu Wii, accédez à l’écran des Options Wii.
  2. Sélectionnez Paramètres Wii, puis WiiConnect24. Ici, sélectionnez Activation, puis l'option Oui et validez.
