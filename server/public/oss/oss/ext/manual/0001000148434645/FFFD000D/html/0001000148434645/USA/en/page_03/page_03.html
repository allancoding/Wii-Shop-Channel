Internet Precautions
WiiConnect24 Settings

● In order to enjoy the Wii Speak Channel, you must have WiiConnect24™ set to On. To do so, please follow these steps:

  1. From the Wii Menu, go to Wii Options.
  2. Select Wii System Settings, then WiiConnect24, and select On when prompted.
About Using WiiConnect24

The Wii Network System End User License Agreement governs Wii online game play and sets forth the Wii Privacy Policy and Wii Code of Conduct.
