Sélection du Labyrinthe

Une fois que vous aurez choisi «Play» à partir du Menu Principal, une liste de tous les labyrinthes inclus dans le jeu apparaitra. Vous remarquerez qu'au début seul le premier niveau peut être joué, mais au fur et à mesure que vous compléterez des niveaux, plus d'options seront débloquées.

 

Quand vous aurez complété un labyrinthe, le jeu sauvegardera automatiquement votre position au cas où vous voudriez revenir plus tard pour rejouer le niveau.

 

Les cinq plus hauts scores s'affichent à l'écran quand vous passez votre curseur sur un niveau en particulier.

 

Mode Défi (Challenge Mode)

Sélectionner le mode «Challenge Mode» dans l'écran de sélection des labyrinthes vous permet de jouer à "L'incroyable Labyrinthe" d'une manière complètement différente. Vous devez maintenant collecter les pierres précieuses disséminées au travers du labyrinthe avant de pouvoir terminer. Regardez bien, certaines pierres peuvent être cachées ! Essayez de terminer le labyrinthe aussi vite que possible pour obtenir un "High Score".
