Lancement
Sélectionnez l'option "Confirmer" sur l'écran-titre pour accéder à l'écran de sélection du jeu. Utilisez le pointeur ou la pour choisir un mode de jeu, puis confirmez votre choix à l'aide du bouton "Confirmer". La console Wii doit être connectée à la connexion Wi-Fi Nintendo pour que vous puissiez voir et enregistrer les scores en classement connexion Wi-Fi Nintendo.

* La première fois que vous lancez le jeu, vous devez définir votre zone d'enregistrement de classement connexion Wi-Fi Nintendo. Sélectionnez la région où vous vivez dans la liste proposée.

 

Modes
Commencer
 Lancez-vous dans une partie en mode 2 minutes ou 5 minutes.
Mode tir rapide
 Voyez à quelle vitesse vous pouvez appuyer sur le bouton de tir.
Records
 Regardez les meilleurs scores ou les scores du classement.
Options
 Définissez le mode de tir de votre arme (auto ou manuel), ou changez votre zone d'enregistrement de classement connexion Wi-Fi Nintendo.
Crédits
   Regardez le nom des personnes derrière Star Soldier R.
