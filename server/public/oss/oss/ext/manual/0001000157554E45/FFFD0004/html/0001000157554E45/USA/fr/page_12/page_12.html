Règles


Le menu Règles est présent à la fois dans le mode Partie rapide et dans le mode Multijoueur. Sélectionnez l'icône Flèche pour naviguer entre les différentes règles, afin de personnaliser complètement vos parties de UNO™.

Joueurs
Choisir de jouer à 4 ou à 6.

Difficulté
Choisir de jouer en mode Facile, Moyen ou Difficile.

Score
Vous pouvez choisir un score à atteindre parmi 250, 500 et 1000.

Défi
Quand cette option est activée, dès qu'un joueur s'est débarrassé de toutes ses cartes, les autres joueurs reçoivent des points en fonction des cartes qui leur restent dans les mains. Dès qu'un joueur atteint le score désigné, appelé score de défaite, il doit quitter la partie. Quand cette option est désactivée, le premier joueur à se débarrasser de ses cartes reçoit des points en fonction des cartes qui restent dans les mains de ses adversaires. Le vainqueur est le joueur qui atteint le premier le score-objectif.

Pénalités
Quand le joueur qui vous précède joue une carte "+2" ou une carte "+4", vous pouvez jouer une carte identique si vous en avez une. Vous ne serez pas pénalisé. Par contre, le joueur qui joue juste après vous verra sa pénalité doublée. S'il possède lui aussi une carte identique, l'effet est transféré au joueur suivant.

Pioche
Il y a trois types de pioches : 1) Piochez une carte et jouez-la si elle est jouable. 2) Piochez une carte, mais ne la jouez pas. 3) Piochez jusqu'à ce que vous ayez une carte jouable.

Jeu forcé
Si vous avez une carte jouable, vous devez la jouer. Le jeu passe ensuite au joueur suivant. Vous pouvez choisir d'activer ou non cette option.

UNO™ 7-0
Vous pouvez choisir d'activer ou non cette option : chaque fois qu'une carte portant le chiffre 0 est jouée, tous les joueurs doivent donner leurs cartes au joueur suivant. Chaque fois qu'un 7 est joué, le joueur qui l'a joué doit échanger ses cartes avec le joueur de son choix.

Saute-mouton
Vous pouvez choisir d'activer ou non cette option : si vous avez exactement la même carte que celle qui se trouve au-dessus de la pile de cartes jouées, vous pouvez la poser immédiatement, sans attendre votre tour.

Contestation
Quand cette option est activée, la carte "+4" peut être contestée par le joueur suivant. Quand elle est désactivée, toute contestation de cette carte est impossible.

Défaut
Rétablir les règles du jeu par défaut.

Go !
Commencer une nouvelle partie.
