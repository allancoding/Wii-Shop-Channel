Résumé


Sur cet écran, accessible depuis le mode Tournoi, vous pouvez commencer un tournoi ou accéder à une foule de statistiques, aux trophées et aux récompenses.

Statistiques
Vous pouvez consulter un certain nombre de statistiques, de votre score total au nombre de cartes "Inversion" jouées.

Trophées
Consultez les trophées que vous avez remportés jusqu'ici. Les éléments encore bloqués seront accompagnés d'un petit texte vous expliquant comment procéder pour les débloquer.


Récompenses
Les pièces de casse-tête débloquées seront regroupées dans cette section. Après avoir gagné 9 pièces, vous pourrez essayer de les assembler pour créer un fond d'écran. Si vous y parvenez, vous débloquerez un nouveau fond d'écran, que vous pourrez utiliser dans le jeu.

Vous gagnerez une nouvelle pièce du casse-tête à chaque fois que vous remporterez une partie en mode Tournoi.
