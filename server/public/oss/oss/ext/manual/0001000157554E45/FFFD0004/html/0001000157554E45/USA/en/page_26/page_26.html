Create Game


Worldwide

This mode lets you host a game using customized settings with players from all over the world. You must first choose the rules and options you want to play with, then a game lobby will be created. You could wait for other players who wish to play according to the rules you have specified or you could invite your Friends to play. You could also start the game immediately. If you choose to do so, you will start the game with CPU-controlled opponents.

Friends Game

This mode is just like Worldwide mode, except the game is restricted to your Friends only. Just like Worldwide mode, you must create a game with your preferences and then a game lobby will be opened. You could wait for your Friends to join or invite your Friends to play with you. You could also start the game with CPU-controlled opponents.

Friend Roster


The Friend Roster is the list of Friends that you have added to the game.

At first your roster will be empty, so you must navigate to the Friend Code input screen to register a Friend. The roster will show the Mii characters of all the Friends whom you have added and who have also added you as a Friend.

A “?” indicates that your Friend has not entered your Friend Code, while a smiley face indicates a Friend who is currently hosting a game. You can point at a Mii with your Wii Remote to view a Friend Code.

You can use the “delete” icon next to a Friend’s slot to remove that Friend from your roster.

Friend Code

In the Friend Code input screen, you must enter the proper code using the number pad on the screen. Make sure you enter the code exactly as it appears otherwise your Friend will not be registered on your roster.
