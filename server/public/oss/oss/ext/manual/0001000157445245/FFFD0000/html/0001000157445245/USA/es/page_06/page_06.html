Primeros pasos

Selecciona un tipo de control y un archivo para guardar datos antes de empezar a jugar.

● Seleccionando un tipo de control
La pantalla mostrada a la derecha aparece cuando inicias el juego. Pulsa el botón de confirmación para pasar a la pantalla de títulos.

* Pulsa el botón de confirmación asignado al tipo de control elegido.

Cambio de mando

Puedes pasar del "Wii Remote horizontal" al "modo clásico" o al "modo Nunchuk" conectando el mando deseado una vez hayas desconectado el mando actual. También puedes pasar de "control de Nintendo GameCube" a otros modos pulsando los botones en los mandos respectivos.

● Selección de jugador
Pulsa el botón de confirmación en la pantalla de títulos para proceder a la pantalla de selección de jugador y selecciona un archivo para guardar datos.
Selecciona 1 de los 5 archivos disponibles o selecciona "GUEST".

 

①Archivo
Se muestra el nombre del jugador y el total de tiempo de juego.

Crear nuevos datos
Selecciona un archivo vacío, accede a la pantalla de introducción de nombre y crea nuevos datos para jugar. Registra tu nombre con un máximo de 4 caracteres.
Selecciona "Confirm" para crear los datos.

Selección de Datos Creado Anteriormente
Selecciona "Start Game" después de elegir un archivo para continuar el juego usando datos creados previamente. Selecciona "Delete Data" para eliminar los datos guardados.

*Los datos eliminados no pueden recuperarse.

Selección de "GUEST"
Selecciona "GUEST" en la parte inferior derecha de la pantalla para iniciar una partida sin guardar. No se guardarán resultados ni otros récords.

Driller Ranking

Selecciona "Driller Ranking" en la pantalla de selección de jugador para mostrar la clasificación para cada fase. Selecciona la fase y luego, la dificultad.
