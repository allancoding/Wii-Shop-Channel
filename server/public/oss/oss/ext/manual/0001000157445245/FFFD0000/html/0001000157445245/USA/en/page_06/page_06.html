Getting Started

Select a control type and a save data file before starting the game.

● Selecting a Control Type
The screen shown on the right appears upon starting the game. Press the confirm button to proceed to the title screen.

* Press the confirm button assigned to the control type chosen.

Changing the Controller

You can change between the "Horizontal Wii Remote", "Classic Style" and "Nunchuk Style" by connecting the desired controller after disconnecting the current controller. You can also change between the "Nintendo GameCube Controller" and other styles by pressing the buttons on the individual controllers.

● Player Select
Press the confirm button at the title screen to proceed to the player select screen and select a save data file.
Select 1 of the 5 files available or select "GUEST".

 

①File
The player's name and total play time is displayed.

Create New Data
Select an empty file, proceed to the name entry screen and create new data to play with. Register your name with up to 4 characters. Select "Confirm" to create the data.

Selecting Previously Created Data
Select "Start Game" after choosing a save data file to continue the game using data created previously. Select "Delete Data" to delete the save data.

*Deleted data cannot be restored

Selecting "GUEST"
Select "GUEST" on the bottom right side of the screen to start a new game without creating save data. Rankings and other records will not be saved.

Driller Ranking

Select "Driller Ranking" at the Player Select Screen to display the score ranking for each stage. Select the stage and then the difficulty.
