Screen Ratio and Saving
Screen Ratio
You can change the screen ratio from the Wii Menu.
Refer to the Wii Operations Manual for details.
Saving
This game uses an auto-save feature.
Save data may be lost if any of the following actions are performed when saving. Please note in advance that lost save data cannot be recovered.

● Pressing the Power Button or RESET on the Wii console or Wii Remote while data is being saved.

● Selecting "Wii Menu" or "Reset" at the HOME Menu while data is being saved.

● Unplugging the AC plug from an outlet while the Wii console power is on.
