Modo Diseño del acuario
Selecciona "Diseño del acuario" en el Menú principal para entrar en el modo Diseño del acuario. Escoge entre varias partes para crear tu propio acuario.
Suelo y objetos
Coloca varios objetos en tu acuario como rocas, algas, coral, arena, etc. Selecciona "Mover objeto del acuario" para mover o quitar objetos de un acuario.

● Colocar objetos
Mueve el mando de Wii en cualquier dirección para mover objetos. Cuando hayas decidido un lugar, pulsa el botón confirmar. En vez de eso también puedes utilizar del mando de Wii para mover objetos.

 
El Medidor de peso aumenta cada vez que añades un pez. Cuando esté lleno, no podrás añadir más. Entonces tendrás que quitar algo antes de poder añadir otra cosa.

● Mover o quitar objetos
Selecciona y confirma el objeto que deseas mover, y después usa el mando de Wii para moverlo. Una vez hayas decidido dónde quieres colocar el objeto, pulsa el botón confirmar de nuevo. En vez de eso también puedes utilizar del mando de Wii para mover objetos.

Selecciona el objeto que quieras quitar y después selecciona el Cubo de basura para quitarlo.
Cristal, fondo y luces
Decora el cristal, establece el tema del fondo y selecciona los ajustes de luz.
Establecer fecha especial
Puedes establecer hasta tres fechas especiales por acuario en las cuales los peces realizarán un baile especial. Pulsa "Confirmar" para aprobar esas fechas especiales.
Rehacer acuario
Rehaz el acuario mostrado en esos momentos. Hay 3 tamaños para los acuarios y sólo se pueden seleccionar en esta opción.
