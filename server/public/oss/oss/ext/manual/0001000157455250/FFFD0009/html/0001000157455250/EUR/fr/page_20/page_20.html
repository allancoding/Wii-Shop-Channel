Format de l'écran et sauvegarde
Format de l'écran
Vous pouvez changer le format de l'écran depuis le menu Wii. Veuillez vous référer au mode d'emploi Wii pour plus de détails.
Sauvegarde
Ce jeu utilise une fonction de sauvegarde automatique.
Les données sauvegardées peuvent être perdues si l'une des actions ci-dessous est effectuée au cours d'une sauvegarde. Par ailleurs, les données perdues ne peuvent être restaurées.

● Appuyer sur le bouton POWER ou sur le bouton RESET de la console Wii ou de la télécommande Wii alors qu'une sauvegarde est en cours.

● Sélectionner "Menu Wii" or "Réinitialiser" dans le menu HOME alors qu'une sauvegarde est en cours.

● Débrancher la prise CA alors que la console Wii est sous tension.
