Wi-Fiランキング
「まいにちくるりん」のスコアアタックの結果をWi-Fiランキングに登録して、全国のプレイヤーたちとハイスコアを競いましょう。
Wi-Fiランキングへの登録について
自分のスコアが自己最高記録の場合、Wi-Fiランキングに任意で登録することができます。登録画面で「はい」を選ぶと、
Wi-Fiランキングに登録されます。

Wi-Fiランキングの登録に関するご注意

● 登録できるのは「自己最高記録」のみです。
（その日の1回目のプレイで自己最高記録を出すと、ランキングを更新できます）

● 登録できるのはタイムアップまで遊び続けた時のスコアです。

● 登録するためには、セーブスロットに名前を登録しておく必要があります。
（※他の人が不快にならないような名前にしましょう）

● 状況によって登録や更新ができない場合があります。また表示に時間のかかる場合があります。
Wi-Fiランキングを見る
Wi-Fiランキングの記録は、「Wi-Fi ランキング」から見ることができます。
