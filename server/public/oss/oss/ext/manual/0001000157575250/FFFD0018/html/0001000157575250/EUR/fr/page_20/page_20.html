Editer une piste
Les nouvelles pistes ne sont au départ composées que d’une série d’éléments plats que vous pouvez modifier à votre guise. Pour sélectionner un élément lorsque vous éditez une piste, appuyez simplement sur gauche ou droite sur .
Appuyez sur haut sur pour supprimer l’élément sélectionné.
Appuyez sur bas sur pour insérer un nouvel élément avant l’élément sélectionné.
Appuyez sur pour modifier l’élément sélectionné.
Pour visualiser un à un les divers éléments disponibles, appuyez sur gauche et droite sur . Pour passer rapidement d’une catégorie d’éléments à une autre, appuyez sur haut ou bas sur . Lorsque vous avez trouvé l’élément que vous souhaitez utiliser, appuyez sur pour confirmer votre choix.
Appuyez sur pour faire un tour d’essai sur votre piste. Sélectionnez TEST DEPUIS LE DEBUT pour courir sur l’ensemble de la piste ou TEST DEPUIS CE POINT pour commencer un tour à partir de l’élément choisi.
