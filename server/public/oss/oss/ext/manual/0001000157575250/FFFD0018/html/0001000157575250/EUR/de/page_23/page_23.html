Strecken empfangen
Wenn dir jemand über WiiConnect24 eine Strecke zuschickt, erscheint im Profil-Auswahlbildschirm ein kleines Brief-Symbol links neben EMPF. STRECKEN (Empfangene Strecken). Wähle EMPF. STRECKEN aus, um deine empfangenen Strecken anzusehen.
Wähle nun jene Strecke aus, die du in einem bestehenden Profil abspeichern möchtest.
Wähle anschließend das Profil, in dem die Strecke gespeichert werden soll.
Wähle nun aus, auf welchem der Speicherplätze die Strecke gespeichert werden soll. Ist auf dem gewählten Platz bereits eine Strecke gespeichert, so wird diese überschrieben.
Hinweis: Von Wii-Freunden oder deiner eigenen Wii-Konsole empfangene Strecken können nicht bearbeitet werden.
