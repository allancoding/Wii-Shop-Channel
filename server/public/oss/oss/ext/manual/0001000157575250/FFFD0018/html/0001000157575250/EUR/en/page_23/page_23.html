Receiving Tracks
When someone sends you a track via WiiConnect24, an envelope will appear next to TRACK INBOX on the Profile Selection Screen. Select TRACK INBOX to see tracks you have received.
Choose the track you want to save from the list.
Select a profile to save the track to.
Select a slot to save the track to. If a track exists in the selected slot, it will be overwritten.
Note: Tracks received from Wii Friends cannot be edited.
