Règle et écran de jeu

Déplacer votre personnage librement, visez et éliminez vos cibles tout en évitant les attaques ennemies.

 

Si vous vous faites toucher directement par un ennemi ou par une balle ennemie vous perdez un cœur en bas de l'écran. Si vous vous faites toucher alors qu'il ne vous reste plus de cœur, le jeu s'arrête.
Appuyez sur le bouton B pour tirer.

Pour déplacer votre personnage, bougez le levier de contrôle de gauche à droite.
Appuyez sur le bouton A pour l'arme secondaire.

Vous ne pouvez plus utiliser votre arme secondaire si l'icône de l'arme secondaire disparaît au bas de l'écran.

Appuyer sur le bouton C ou balancez le Nunchuk pour faire une esquive.

Les esquives vous rendent invincible.
