バトル画面／ステータス異常

敵から特殊な攻撃を受けると、以下のようなステータス異常になることがあります。アイテムや魔法を使い、早めに治しましょう。なお、バトル後に自動的に回復するものと回復しないものがあります。

バトル後に治るもの
ねむり

眠って行動不能になる。

回復方法：白魔法「エスナ」
アイテム「目覚まし時計」
まひ

体が動かなくなり、行動不能になる。

回復方法：白魔法「エスナ」
アイテム「ユニコーンの角」
こんらん

バトルコマンドを入力できなくなり、敵・味方問わず攻撃してしまう。

回復方法：白魔法「エスナ」
“こんらん”しているキャラクターが攻撃を受ける
のろい

攻撃力や防御力が半減する。

回復方法：白魔法「エスナ」
アイテム「十字架」
バトル後も治らないもの
どく

HPが少しずつ減り続ける。

回復方法：白魔法「エスナ」
アイテム「毒消し」
くらやみ

視界が悪くなり、物理攻撃が当たりにくくなる。

回復方法：白魔法「エスナ」
アイテム「目薬」
ちんもく

魔法が使えなくなる。

回復方法：白魔法「エスナ」
アイテム「やまびこ草」
ぶた

「ポーキー」以外の魔法が使えなくなる。

回復方法：黒魔法「ポーキー」
白魔法「エスナ」
アイテム「ダイエットフード」
かえる

「トード」以外の魔法が使えず、攻撃力や防御力が下がる。

回復方法：黒魔法「トード」
白魔法「エスナ」
アイテム「乙女のキッス」
こびと

攻撃力や防御力が下がる。

回復方法：白魔法「ミニマム」「エスナ」
アイテム「うちでの小づち」
せきか

石になってしまい、行動不能になる。

回復方法：白魔法「エスナ」
アイテム「金の針」
戦闘不能

HPが0で行動不能の状態。

回復方法：白魔法「レイズ」
アイテム「フェニックスの尾」
