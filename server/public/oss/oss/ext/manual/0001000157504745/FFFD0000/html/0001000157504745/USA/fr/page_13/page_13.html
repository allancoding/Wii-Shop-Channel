Recueillir les pingouins
En utilisant les ballons pour visiter les îles et les icebergs dans le jeu, vous pouvez aller chercher d’autres pingouins et les recueillir dans le parc Snowpack.

● Comment recueillir un pingouin
Pour recueillir un pingouin, vous devez le placer à l’intérieur du cercle jaune, sous le panier, en utilisant différentes méthodes. Si vous tirez sur la corde attachée au bâton qui retient le panier dans le temps imparti, tous les pingouins situés dans le cercle jaune vous raccompagneront jusqu’au parc Snowpack.

Notes : - Si vous tombez dans l’océan ou si le temps s’écoule avant que vous ne tiriez sur la corde, vous retournerez au parc Snowpack sans avoir recueilli de pingouin. Vous garderez cependant les objets ramassés.
- L'humeur d'un pingouin qui n'a pas encore été recueilli ne s'affiche pas lorsque vous pointez dessus. Son humeur s’affichera une fois ramené au parc.
