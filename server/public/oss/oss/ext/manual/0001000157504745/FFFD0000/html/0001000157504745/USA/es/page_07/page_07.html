Cómo empezar

● Empezar un juego nuevo

Para empezar un juego nuevo oprime en la pantalla del título y después elige New Game (juego nuevo).

1. Escoge un archivo

Cuando juegues por primera vez, elige uno de los archivos disponibles para crear un archivo de datos.

Nota: Si eliges un archivo que ya contiene datos, el nuevo juego se sobrescribirá sobre esos datos. Una vez que los datos se han borrado, no podrán recuperarse ¡así que ten cuidado!

2. Elige al personaje principal

Elige entre los personajes Mii que tienes en tu consola o entre los personajes que vienen con el juego. Una vez que elijas un personaje, se creará un archivo de datos y el juego comenzará.

Notas: Una vez que comiences a jugar no podrás cambiar al personaje principal.
Si borras a un personaje Mii desde el Canal Mii y este era el personaje principal, el personaje Mii también se borrará en el juego y tendrás que elegir otro personaje principal.

● Cómo continuar

En la pantalla del título oprime , después elige Continue (continuar).

 ・Elige que archivo con datos

Para continuar con un juego guardado, elige un archivo con datos.
El número de pingüinos reunidos, los puntos de amistad ganados (Ver pág. 10), el tiempo de juego total y más, se mostrarán en cada archivo de datos.

Notas: Para más información acerca del guardado consulta la pagina 8 Cómo salir del juego.
El archivo de datos que contiene los datos de la última vez que guardaste, se mostrará con un marco amarillo.
