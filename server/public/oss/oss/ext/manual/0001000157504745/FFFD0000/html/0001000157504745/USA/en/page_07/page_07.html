Getting Started

● Starting a New Game

Press when prompted on the title screen, then select New Game.

1. Select a file

When playing for the first time, select an empty file in order to create save data.

*If you select a file that already contains save data, the new game will overwrite the existing data. Please be aware that a file cannot be retrieved once deleted.

2. Select a main character

Choose from the characters registered on your Wii console or from the characters already prepared for the game. Once you pick your character, a save file will be created, and the game will begin.

*You cannot change the main character once you begin playing.

*If you delete your selected Mii character from the Mii Channel, it will also be deleted from the game, and you will need to select a new main character.

● Continuing a Game

Press when prompted on the title screen, then select Continue.

 ・Select a Save File

Select a save file to continue a saved game.
Information, such as the number of penguins collected, friendship points earned (→P.10), total playing time, and more, are displayed for each save file.

*For more details about saving, see Exiting the Game (→P.8).

*The save file from the last time you played and saved will have a yellow frame.
