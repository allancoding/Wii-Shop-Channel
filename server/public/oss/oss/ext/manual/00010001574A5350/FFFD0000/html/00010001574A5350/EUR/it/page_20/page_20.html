Vincere la partita

Per vincere una partita a Jungle Speed devi sbarazzarti di tutte le tue carte presenti nella pila delle carte coperte e in quella delle carte scoperte.

 

Dopo aver girato la tua ultima carta, per aggiudicarti la partita devi ancora liberarti della pila di carte scoperte.

 

Puoi sbarazzarti dell'ultima carta se:

  • Vinci un duello
  • Un altro giocatore è costretto a prendere le tue carte
  • Afferri per primo il totem quando un giocatore scopre la carta AFFERRA IN FRETTA

 

Casi speciali

Se l'ultima carta che giri è una CARTA FRECCIA entrano in gioco alcune regole speciali.

 

Se l'ultima carta che giri è una carta TUTTI GIRANO, vinci la partita!

 

Se l'ultima carta che giri è una carta SCONTRO DI COLORI, tutte le carte scoperte sul tavolo vanno a finire nella tua pila di carte coperte.

 

Se l'ultima carta che giri è una carta AFFERRA IN FRETTA, per vincere la partita devi essere il primo ad afferrare il totem. Se perdi il turno AFFERRA IN FRETTA, tutte le carte scoperte sul tavolo finiscono nella tua pila di carte coperte.
