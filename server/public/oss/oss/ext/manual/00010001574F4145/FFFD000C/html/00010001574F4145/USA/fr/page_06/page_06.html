Contrôles
 

●Maillet

-Lorsque vous sélectionnez le mode «Auto», le maillet se déplace automatiquement vers le palet.

-Lorsque vous sélectionnez le mode «Manuel», vous contrôlez le maillet avec la croix directionelle.

-Si vous utilisez le Nunchuk, vous pouvez contrôler le maillet avec le levier de contrôle.

 

●Frapper avec le maillet

-Lorsque vous balancez la Wii Remote, le maillet vole en direction du palet.
-Si vous parvenez à maîtriser le palet et à le faire entrer dans les buts de votre adversaire, vous gagnez un point.
