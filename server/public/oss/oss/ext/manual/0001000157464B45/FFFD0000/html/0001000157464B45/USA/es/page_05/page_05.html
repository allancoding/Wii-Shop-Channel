Pon el mando de Wii horizontalmente cuando juegues a Family Go-Kart Racing. (Este juego también puede jugarse usando el Wii Wheel.)
 
Sujeta el mando de Wii de modo que los botones estén enfrente de ti. Gira el mando de Wii a la derecha para girar tu kart a la derecha, y a la izquierda para hacerlo girar a la izquierda.
 
Para acelerar, pulsa el botón 2, y para frenar, pulsa el botón 1. Mantén pulsado el botón 1, para dar marcha atrás.
 
Usa los objetos pulsando la cruz de control.
