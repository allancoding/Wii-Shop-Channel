Commandes
 

 
Tenez la manette Wii Remote horizontalement pour jouer à Family Go-Kart Racing. (Vous pouvez aussi jouer à ce jeu avec le Wii Wheel.)
 
Tenez la manette Wii Remote de manière à ce que les boutons se trouvent face à vous. Inclinez la manette Wii Remote vers la droite pour tourner à droite. Inclinez la manette Wii Remote vers la gauche pour tourner à gauche.
 
Pour accélérer, appuyez sur le bouton 2. Pour freiner, appuyez sur le bouton 1. Pour faire marche arrière, maintenez enfoncé le bouton 1.

 

Pour utiliser des objets, appuyez sur la croix directionnelle.
