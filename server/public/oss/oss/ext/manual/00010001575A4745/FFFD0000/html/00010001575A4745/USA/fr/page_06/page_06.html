Sélection du personnage/parcours

Pour sélectionner un personnage :

  • Remarque : La manette Wii Remote avec le témoin du joueur 1 doit être active à la fois pour le jeu à un seul joueur et le jeu multijoueurs. Dans ce mode d'emploi électronique, la personne utilisant la manette Wii Remote dont le témoin du Joueur 1 est allumé est appelée le joueur 1 ou désignée P1.
  • La cage du joueur 1 (P1) est ouverte lorsque l'écran de sélection de personnage apparait. Le joueur 1 utilise les flèches gauche et droite de la croix directionnelle pour passer en revue les personnages disponibles: Gorfou, Kangourou, Panda, Suricate.
  • Lorsque le personnage choisi est dans la cage, le Joueur 1 appuie sur le bouton A. La fenêtre Sélection gaucher/droitier apparait. Le joueur doit choisir quelle main il utilisera pour se servir de la manette Wii Remote. (Veuillez noter que le personnage animé utilise toujours son membre droit pour envoyer le disque, mais la manette Wii Remote répond à la position et à l'orientation de la main choisie ici.) Placez le curseur sur l'option désirée et appuyez sur le bouton A. Le bouton GO s'allume.

Pour jouer seul :

  • Le joueur 1 place le curseur sur GO et appuie sur le bouton A afin de passer à l'écran de sélection de parcours.

Pour jouer en mode multijoueur :

  • Jusqu'à trois joueurs supplémentaires peuvent se joindre à la partie en sélectionnant un personnage à l'aide d'autres manettes Wii Remote. Chaque manette Wii Remote utilisée dans le jeu devra être synchronisée avec la console Wii.
  • Appuyez sur le bouton B pour revenir à l'écran ou à l'étape précédente.
  • Chaque joueur supplémentaire devra appuyer sur le bouton A de sa manette Wii Remote pour ouvrir une cage et procéder à la sélection d'un personnage en suivant les étapes décrites pour le joueur 1 ci-dessus.
  • Une fois que tous les joueurs ont choisi un personnage, le Joueur 1 place le curseur sur le bouton GO et appuie sur le bouton A pour passer à l'écran de sélection du parcours. Le bouton GO ne s'allumera pas tant que tous les joueurs n'auront pas choisi un personnage.

Sélection du parcours

  • Le parcours du Zoo Disc Golf comporte 18 trous.
  • Un ensemble de trois trous, neufs trous ou comportant les 18 trous du parcours peut être choisi. Choisir « 3 » (trous) ou « 9 » (trous) fait apparaître des options supplémentaires sur la carte de parcours du Zoo Disc Golf.
  • Le joueur 1 choisit l'ensemble de trous désiré puis sélectionne Commencer la partie.
