Ordres de mission
Vous pouvez rédiger des ordres de mission ("behests") chaque matin pendant la phase de rapport. Ces ordres seront placardés sur tous les panneaux d'affichage du royaume à l'attention des aventuriers. Vous pouvez passer autant d'ordres qu'il y a de panneaux d'affichage. Autrement dit, en construisant des parcs d'affichage, vous augmenterez le nombre d'ordres que vous pouvez passer quotidiennement.

Lorsque vous sélectionnez un panneau d'affichage, vous voyez s'afficher à l'écran une carte du monde avec le château en son centre. Déplacez le curseur sur un donjon pour obtenir des informations détaillées sur la zone, le type d'ordres correspondant à ce donjon et les monstres que les aventuriers pourront y croiser. Servez-vous de ces informations pour choisir les donjons appropriés aux aventuriers de votre royaume.

Une fois que vous aurez suffisamment progressé dans le jeu, vous pourrez choisir le château pour y passer des ordres, demandant par exemple un changement de métier ou proclamant un jour férié.

Les aventuriers afflueront vers les ordres placardés sur les panneaux d'affichage. Parlez-leur pour examiner leurs statistiques et choisir vous-même l'aventurier à qui vous souhaitez confier une mission particulière. Vous pouvez envoyer ceux que vous n'avez pas sélectionnés explorer les donjons de leur choix pour leur permettre d'accumuler de l'expérience.
