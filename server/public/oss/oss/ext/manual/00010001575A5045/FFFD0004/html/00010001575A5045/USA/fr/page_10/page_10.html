Commandes

 

Un ou deux joueurs peuvent jouer simultanément.

 

 

 

Commandes

 

Se déplacer

Armes

Esquiver

Grenades

Tirer

 


1. Tirer : utilisez la télécommande Wii pour viser et ouvrir le feu sur les zombies qui vous attaquent ou pour détruire tout ce qui se dresse sur votre chemin. Appuyez sur le bouton B pour tirer avec la mitrailleuse et sur le bouton A pour lancer des grenades.

 

2. Se déplacer : utilisez le Stick directionnel pour vous déplacer vers la gauche et la droite.

 

3. Esquiver: appuyez sur le bouton Z du Nunchuk pour esquiver un zombie qui vous attaque.
