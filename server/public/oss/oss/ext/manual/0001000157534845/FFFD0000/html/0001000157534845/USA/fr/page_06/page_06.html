L'histoire
Vous êtes maintenant dans Stonekeep, la maison de vos ancêtres. Une force maléfique est entrée dans le donjon et a chassé la plupart des résidents. Seul un vrai descendant du clan peut résoudre le mystère de Stonekeep et peut finalement donner la paix aux esprits agités.

 

 

 

 

Vous commencez l'aventure Stonekeep dans les donjons. Pour survivre, vous allez avoir besoin d'acquérir un allié pour vous guider. Vous pourrez choisir Dugh (un nain), Pierre (un elfe) ou Skuzz (un Sharga) comme allié.

 

Vous choisissez votre guide en bougeant votre personnage dans l'une des trois portes quand vous commencez le jeu. Quand vous avez franchi une porte, vous ne pouvez pas revenir en arrière. Votre allié aura des indices à propos des niveaux et de l'histoire. Les alliés peuvent aussi vous aider dans les combats, mais faites attention à ne pas accidentellement ou intentionnellement attaquer votre allié ou il pourrait se tourner contre vous !

 

Lors de la sélection de votre allié, il va vous apprendre comment utiliser correctement votre poignard. Après que vous ayez montré votre capacité à utiliser le poignard, vous entrerez dans le donjon. 

Tout au long de votre avancée dans le jeu vous aurez la possibilité de sauvegarder votre progression par l’écran de gestion des données du menu Wii.

 

Lorsqu’il n’y a plus assez de mémoire disponible dans la Console Wii vous pouvez choisir de continuer à jouer sans sauvegarder. Le jeu sauvegardera temporairement votre statut mais ne le sauvera pas dans l’écran de gestion des données du menu Wii. Si vous retournez au menu Wii ou éteignez la Console Wii vous perdrez les données sauvegardées temporairement.
