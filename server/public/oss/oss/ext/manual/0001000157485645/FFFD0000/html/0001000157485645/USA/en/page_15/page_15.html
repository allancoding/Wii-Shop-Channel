Controls

Hot Rods are easy to control no matter which controller style you prefer. Below you can see the driving controls for each controller style. If you forget, you can pause the game in a race and see a list of these controls.

 

Wii Remote / Wii Wheel Style

Motion Sensor
(Tilt the Wii Remote left or right)

Steer Vehicle

2 Button

Accelerate
(release to brake)

+ Control Pad

Powerslide
(hold any direction and steer)

A Button / B Button / 1 Button

Bunny Hop

Shake Wii Remote

Perform a Stunt
(while airborne)

 

 

Nunchuk Style

Control Stick

Steer Vehicle

A Button

Accelerate
(release to brake)

Z Button

Powerslide
(hold button and steer)

B Button

Bunny Hop

Shake Wii Remote

Perform a Stunt
(while airborne)

Accelerate

Perform a Stunt
(while airborne)
