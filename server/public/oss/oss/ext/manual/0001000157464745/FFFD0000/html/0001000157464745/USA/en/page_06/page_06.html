Single-Player Modes

There are four single-player modes the player can choose from each with slight variations. Each single-player mode contains its own high score list tracked separately.

 

Arcade Mode:

This mode allows the player to experience the thrill of unlimited levels with the chance to test their skill as the difficulty gradually increases with each level passed. To increase difficulty on each progression, obstacles are increased in both frequency and speed. The timer will become a major factor as each level gets harder to navigate. The player cannot configure options in this mode and once the total number of lives is used, the player must start over from the beginning.

 

Time Trial:

Time trial is a true test of speed to see how fast the player can complete each stage or the entire game. The player can choose to challenge a single stage or the entire game. Time records are saved in the Time Trial High score list.

 

Score Attack:

The main goal of Score attack is to grade the player's efficiency for each level or full game play through. A score is awarded for the time remaining and number of lives used. High scores are recorded separately for each level and full game playthrough.

 

Free Play:

Free Play mode is a basic mode designed for casual players to play through the game with unlimited lives. Once the player reaches the end of the game there is no score, only a simple congratulation screen. As the player progresses through each level they unlock the next.
