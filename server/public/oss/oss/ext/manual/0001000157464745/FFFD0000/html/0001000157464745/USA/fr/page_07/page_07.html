Multijoueur
Modes multijoueur

Frogger Returns dispose de quatre mini-jeux et chacun d'entre eux permet de faire une partie rapide et sympathique à deux.

 

Course classique :

Ce mode oppose les joueurs lors d’une course classique. Les joueurs doivent atteindre, avant leurs adversaires, les cinq zones de but situées sur le côté de l’écran. Dès qu’un joueur atteint un but, une grenouille arborant sa couleur vient se placer à l’intérieur et le but ne peut plus être utilisé. Le vainqueur est celui qui atteint trois buts le premier.

 

Territoires :

Le joueur doit couvrir le plus de buts possible dans un temps donné. Pour couvrir un but, il doit avoir été le dernier à l’atteindre. Les buts couverts peuvent être reconquis par les autres joueurs tant que le temps n’est pas écoulé. Le vainqueur est celui qui a couvert le plus de buts à la fin du temps imparti.

 

Collecte :

Des mouches de valeur différente sont placées dans le niveau. À chaque fois que le joueur ramasse une mouche, sa valeur est ajoutée à son inventaire. Lorsqu’un joueur atteint un but et le couvre, son inventaire s’enrichit de ce but et de sa valeur. Les joueurs perdent tout leur inventaire s’ils meurent avant d’avoir atteint un but. La partie se termine lorsque tous les buts ont été emportés. Le vainqueur est celui qui a le plus de points lorsque la partie est terminée.

 

Festin de mouches :

Chronométré, ce mini-jeu oppose deux adversaires sur une carte unique. Des mouches apparaissent de manière aléatoire sur la carte et les joueurs doivent en attraper le plus possible avant la fin du temps imparti. Lorsque le temps est écoulé, le vainqueur est celui qui a le plus de mouches à son actif.
