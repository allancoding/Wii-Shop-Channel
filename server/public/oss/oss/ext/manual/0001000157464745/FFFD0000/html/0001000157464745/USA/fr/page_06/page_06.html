Modes Un joueur

Quatre modes à un joueur sont proposés, chacun étant légèrement différent des autres. À chaque mode à un joueur correspond une liste de meilleurs scores.

 

Mode Arcade :

Ce mode permet au joueur de jouer sans limite de niveaux et d’améliorer ses performances d’un niveau à l’autre. Plus il progresse dans le jeu, plus les obstacles sont rapprochés et plus le jeu se complique. Le temps passé devient alors un facteur essentiel.

 

Rapidité :

Ce mode est une véritable épreuve de rapidité. Le joueur peut choisir de relever le défi pour un seul niveau ou bien pour l’ensemble du jeu. Les records sont ensuite répertoriés dans la liste des meilleurs scores de Rapidité.

 

Attaque :

Le mode Attaque a pour principal objectif de mesurer l’efficacité du joueur, qu’il choisisse de jouer à un seul niveau ou de tous les enchainer. Le score est fonction du temps restant et du nombre de vies utilisées. Les meilleurs scores sont enregistrés séparément.

 

Loisir :

Le mode Loisir permet de jouer une fois à chaque niveau et sans limite de vies. C’est en progressant dans le jeu que le joueur déverrouille le niveau suivant.
