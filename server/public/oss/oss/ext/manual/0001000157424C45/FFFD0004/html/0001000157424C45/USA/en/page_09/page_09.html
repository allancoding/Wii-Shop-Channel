Game Screens

Standard Mode

Classic Bubble Bobble action.

Player 1 will begin each round as Bub in the lower-left, while Player 2 will start as Bob in the lower-right.

Scores are displayed at the top of the screen.

1. Displayed if you have obtained EXTEND Bubbles.
2. Player Lives (Remaining Lives)
3. Item
4. Food
5. Current Round Number
Arrange Mode
The same basic Bubble Bobble system with some new arrangements.

Offers play for up to 4 players simultaneously, with Player 1 playing as Bub, Player 2 as Bob, Player 3 as Peb, and Player 4 as Pab.

Scores are listed at the top of the screen.

 

Note: Players' default positions vary by round.

1. Player 1's EXTEND Bubbles are displayed here.
2. Player 2's EXTEND Bubbles are displayed here.
3. Player 3's EXTEND Bubbles are displayed here.
4. Player 4's EXTEND Bubbles are displayed here.
5. Player Lives (Remaining Lives)
6. Item
7. Food
8. Current Round Number
