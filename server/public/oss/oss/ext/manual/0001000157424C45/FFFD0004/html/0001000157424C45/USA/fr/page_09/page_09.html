Mode Extra
Le même système de jeu que le Bubble Bobble d'origine, avec quelques modifications. Il permet notamment à quatre participants de jouer simultanément, avec Bub (joueur 1), Bob (joueur 2), Peb (joueur 3) et Pab (joueur 4). Les scores sont indiqués en haut de l'écran.

 

Remarque : la position de départ des joueurs peut varier suivant le niveau.

1. Indique les alphabulles du joueur 1.
2. Indique les alphabulles du joueur 2.
3. Indique les alphabulles du joueur 3.
4. Indique les alphabulles du joueur 4.
