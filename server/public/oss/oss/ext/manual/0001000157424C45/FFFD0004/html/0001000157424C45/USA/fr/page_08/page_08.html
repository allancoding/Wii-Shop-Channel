Commandes
Vous pouvez utiliser deux types de commande.

Wii Remote
・・・・Déplacer le joueur / Déplacer le curseur
・・・・Projeter des bulles / Annuler
・・・・Sauter / Confirmer
・・・・Commencer à jouer / Pause
Remarque : si vous utilisez une Wii Remote, tenez-la horizontalement.
Classic Controller
・・・・・・・・Déplacer le joueur / Déplacer le curseur
・・・・Projeter des bulles / Annuler
・・・・Sauter / Confirmer
・・・・Commencer à jouer / Pause
