Controls
Tool/Hand Movement

Wii Remote Up: Forward

Wii Remote Down: Back

Wii Remote Left/Right: Left and Right

 

B Button

Hold, gesture upward, and release: Perform the scoop with the Bucket tool.

Press: Go Back

 

A Button

Press: Select

Press: Pick Up or Drop Tool

 

1 Button

Press: Pause

 

+ and - Buttons

Press: Change the scale of the selection for the Shovel tool from 1x1 to 2x2 and back.
