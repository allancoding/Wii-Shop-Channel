Controles

Mantén el Wii Remote verticalmente cuando juegues a Gene Labs.

Navegar por los menús:

Para seleccionar una opción solo debes apuntar hacia ella y pulsar el Botón A.
Si quieres cancelar la selección o volver al menú anterior, pulsa el Botón B.

Controles de juego:

Utiliza el puntero para mover el cursor.
Para seleccionar las células mantén presionado el Botón A y mueve el cursor sobre las células deseadas.

 

Para distribuir el antivirus selecciona una (o varias) células, y después mueve el cursor sobre la célula destino y suelta el Botón A.

 

Pulsa el Botón + para abrir el menú Pausa.
