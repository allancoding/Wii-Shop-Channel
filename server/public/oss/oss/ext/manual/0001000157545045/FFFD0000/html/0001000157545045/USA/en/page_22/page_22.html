Options
Region
For selecting the region to be registered for ranking.
Wii Remote Configuration
You can select one of two configurations for the Wii Remote. You cannot change the configuration for the Classic Controller.
Ghost
When Ghost is set to ON, a shadow appears to show the shape and direction of where the Tetrimino will be dropped. When Ghost is set to OFF, this shadow disappears.
Background
Changes the background displayed during the game. This setting applies to all modes of the game.
Sound
Used for changing the audio output (monaural/stereo) and SE/BGM volume levels.
Mii
Switches Mii display during games ON/OFF.
Other Player Mii Display
Used for setting whether or not to display the Miis created by other players during Nintendo Wi-Fi Connection or Tournament play.
* When "Replace" is selected, a default Mii will be displayed.
Credits
Used for viewing staff credits.

In addition to these, you can change the rule settings for each game. For details, see the game's "Options" selection.
