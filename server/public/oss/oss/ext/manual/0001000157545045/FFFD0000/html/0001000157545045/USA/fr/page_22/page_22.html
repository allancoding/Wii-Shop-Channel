Options
Région
Sélectionnez votre région pour le classement.
Configuration télécommande Wii
Choisissez un des deux types de commandes pour la télécommande Wii. Vous ne pouvez pas changer de type de commandes avec une manette classique.
Ghost
Activez ou désactivez les Ghost.
Ce paramètre s'applique à tous les modes du jeu.
Arrière-plan
Changez l'arrière-plan du jeu.
Ce paramètre s'applique à tous les modes du jeu.
Son
Changez la sortie audio (mono/stéréo) et le volume des bruitages et de la musique.
Mii
Choisissez d'afficher ou non le Mii dans le jeu.
Affichage Mii des joueurs
Choisissez d'afficher ou non le Mii des autres joueurs en connexion Wi-Fi Nintendo ou tournoi.

* Le Mii des autres joueurs sera remplacé par un Mii de défaut.

Crédits
Affiche les crédits du jeu.

Vous pouvez également changer les règles selon le mode de jeu. Pour plus de détails, référez-vous aux "Options" dans le jeu.
