Le mode Mini-jeux
Dans ce mode, vous pouvez jouer à tous les mini-jeux débloqués (jusqu'à 10) selon le niveau de difficulté de votre choix : Facile, Normal ou Difficile. Tout comme dans le mode Aventure, vous pouvez jouer jusqu'à 4 joueurs, avec des CPU qui remplacent les joueurs manquants. Alors choisissez un personnage et sélectionnez un mini-jeu !

Ecran Mini-jeux


Sur cet écran, vous pouvez sélectionner le mini-jeu auquel vous voulez jouer, ainsi que le niveau de difficulté.

Ecran Tutoriel


Voici l'écran du tutoriel du mini-jeu que vous avez choisi. Vous aurez accès à un petit texte d'aide, aux commandes et à une image du jeu. Cet écran apparaît juste avant le début du mini-jeu.

Ecran Rang


Cet écran vous indique le rang et le score de chaque joueur dans le mini-jeu auquel vous venez de jouer. Vous obtiendrez également une évaluation ('Bon', 'Nul' ou 'Excellent') en fonction de votre performance dans le mini-jeu.
