Menus
Menu principal


Le menu principal vous permet d'accéder à tous les modes de jeu. Celui-ci propose les options suivantes :

L'option Aventure vous permet de jouer en utilisant la boule de cristal pour progresser dans le manoir et vivre l'aventure au complet.

Choisissez Mini-jeux pour jouer aux mini-jeux débloqués avec différents niveaux de difficulté.

Dans le menu Options, vous pourrez régler un certain nombre de paramètres comme la luminosité et le volume de la musique et des effets sonores.


Menu de pause

Le menu de pause vous permet d'accéder au menu Options afin de modifier les paramètres. Il vous permet également de retourner au menu principal ou au menu Mini-jeux.


Menu Options


Vous pouvez régler les paramètres selon vos préférences dans ce menu : luminosité, musique et son.

Choisissez la section Crédits pour découvrir l’équipe derrière ce jeu.
