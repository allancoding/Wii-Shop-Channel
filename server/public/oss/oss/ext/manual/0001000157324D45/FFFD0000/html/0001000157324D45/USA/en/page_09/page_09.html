SAVING A GAME
BLASTER MASTER™ OVERDRIVE allows special players like you to save progress in the game in up to three different files. Don’t tell the other players or they might get jealous and that’s just tedious for everyone.

Saved files can be loaded at the title screen.

●SAVE
To SAVE the game, proceed to an area where the game is in Top View mode.

●LOAD
Select the file to load from the three slots displayed with the +Control Pad, then press the 1 Button.
