Débuter une partie

(Comment débuter une partie, Comment jouer, Qu’est ce qui est sauvegardé,…)

Commencez une partie en vous rendant dans le menu « Jouer » et en choisissant l’un des modes de jeu. Pour cela, dirigez le pointeur vers le mode choisi et pressez le bouton A.
Les modes de jeu

  • Mode Classique :

Une fois dans le jeu, les ouvriers vont construire la tour si vous ne faites rien. Avec vos pouvoirs, vous devez les éliminer.

Visez un ouvrier et appuyez sur le bouton A pour déclencher ce pouvoir. Cela permet de le lancer rapidement, fréquemment et précisément.
Placez le curseur sur le nuage sombre en haut à gauche de l’écran, maintenez le bouton A appuyé puis visez la zone dans laquelle vous allez faire tomber la foudre en relâchant le bouton !
Placez le curseur dans la zone du vent, au centre gauche de l’écran, maintenez le bouton A appuyé et faites un geste vif vers la droite à la hauteur où vous souhaitez déclencher la tempête, en relâchant le bouton A.
Placez le curseur dans la zone des météorites, en haut à droite de l’écran, maintenez le bouton A appuyé et faites un geste vif vers le bas en relâchant le bouton A.
Placez le curseur dans la zone de l’eau, au centre droit de l’écran, maintenez le bouton A appuyé et faites un geste vif vers la gauche en relâchant le bouton A.
Secouez votre télécommande Wii dans toutes les directions pour déclencher l’apocalypse chez les mortels !
Pour déclencher le pouvoir de l’œil divin, appuyez sur l’icône le représentant dès qu’elle s’illumine et qu’elle s’anime. Efficacité garantie !


Chacun de vos 7 pouvoirs de base a besoin de mana pour être lancé. Le mana se recharge avec le temps, et plus le pouvoir est puissant, plus il va en consommer. Les pouvoirs le plus efficaces sont donc les plus longs à se recharger : prenez soin de choisir le bon moment pour les utiliser.


Certain de vos pouvoirs disposent de 3 niveaux de rechargement, d’autre un seul. Ils sont visibles sur les pastilles et vous indiquent leur disponibilité. Un tiers chargé permet de lancer le pouvoir à sa puissance minimale, alors qu’un cercle complet indique la puissance maximale.

  • Mode Divinité :

Le mode divinité présente le même mode de jeu.

Il vous présente deux nouveaux pouvoirs : l’éclair rouge et le buisson ardent, qui s’utilisent de la même façon que les pouvoirs précédents.

Egalement, vous découvrirez des bonus vous permettant d’améliorer votre score, d’arrêter le temps quelques instants, ou encore de recharger instantanément l’un de vos pouvoirs les plus puissants !



Les bonus sont à activer en les visant avec télécommande Wii puis en pressant sur le bouton A.

  • Mode Campagne :

Le mode campagne vous propose une série de missions pour mettre à l’épreuve vos compétences divines.


Une carte représente les différents lieux dans lesquels les humains vont essayer de construire leur tour. Vous devez les affronter et remporter chaque bataille, avec des objectifs chaque fois différents !

Dans ces missions, vous commencez comme un dieu débutant, avec peu de pouvoir, et c’est grâce à vos performances que vous pourrez améliorer vos pouvoirs pour vous rendre invincible.


Cet écran vous présente les différentes améliorations disponibles pour vos pouvoirs.

  • Mode Fury :

Le mode fury vous permet de découvrir une variante de jeu de Babel Rising, en enfer ! Combatez les petits démons, mais attention : ceux-ci sont résistants à certains de vos pouvoirs !

Un mode de jeu à conseillez aux dieux les plus méritants.
