Choisir joueur ou Mii™
Choisir joueur


Voici l'écran sur lequel vous pouvez choisir le personnage avec lequel vous allez jouer. En mode 'Carrière', vous ne pourrez pas le modifier. Vous devrez créer un nouveau profil si vous souhaitez jouer avec un personnage différent.

Pour prévisualiser un personnage, sélectionnez son avatar. Une fois que votre choix est fait, sélectionnez l'icône 'Suivant'.

Si vous voulez jouer avec un de vos personnages Mii, sélectionnez l'icône 'Choisir un Mii'.


Choisir un Mii


Dans ce menu, vous pourrez sélectionner un des personnages Mii se trouvant dans votre console Wii ou dans votre Wii Remote.

Pour changer de Mii, rendez-vous sur la Chaîne Mii du Menu Wii.
