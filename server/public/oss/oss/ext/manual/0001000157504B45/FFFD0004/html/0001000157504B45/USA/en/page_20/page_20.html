Game Details


This menu will let you specify what kind of a game you would like to play, whether you selected Create Game or Custom Game.

The ‘Players’ parameter will determine the maximum amount of players for a table.

Some blinds parameters for cash games or buy-in parameters for a tournament will be locked if you don’t have enough money in your bankroll.

Custom Games: If you select the ‘Any’ criteria for a certain parameter, this parameter will not be taken into account when matching you with other players.
