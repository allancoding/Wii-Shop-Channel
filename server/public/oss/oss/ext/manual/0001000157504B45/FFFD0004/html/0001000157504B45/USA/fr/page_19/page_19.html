Jouez !


Choisissez de faire un Cash Game rapide, trouvez d'autres joueurs un peu partout dans le monde grâce à la section Partie personnalisée ou créez une partie avec la section Créer partie.


Cash Game rapide
Vous serez automatiquement installé à une table de Cash Game avec des petites ou des grosses blinds en fonction de vos jetons.

Si vous vous joignez à un Cash Game en cours de route, le montant de la cave est déterminé par la somme dont vous disposez.

Partie personnalisée
Allez dans le menu Infos sur le jeu, puis entrez vos critères de recherche de table de jeu. Vous serez automatiquement mis en contact avec d'autres joueurs un peu partout dans le monde.


Cash Game : en fonction des Cash Games disponibles, vous aurez la possibilité d'arriver dans une partie déjà en cours ou dans une salle de jeu avant le début de la partie.


Créer partie


A partir de ce menu, vous pouvez créer une partie mondiale, une partie entre amis, accéder à votre liste d'amis ou ajouter/supprimer des amis.


Créer une partie mondiale

Dans une partie mondiale, vous pourrez jouer avec n'importe qui partout dans le monde. Tous les joueurs peuvent inviter des amis.

Seul l'hôte a le droit de réserver des sièges et de démarrer une partie.


Créer une partie entre amis

Si vous ne voulez pas jouer avec des inconnus, pas de problème. Dans ce menu, vous pouvez choisir d'héberger une partie entre amis à laquelle seuls vos amis peuvent participer.
