Commandes

●télécommande Wii

manette +　Mouvements du joueur / Visée
bouton 1　Attaquer/Attaque fixe (*)
bouton 2　Sauter
Balancer　Changer d'arme
bouton +　Menu Pause

(*) Maintenir le bouton 1 appuyé

●manette classique

manette +　Mouvements du joueur / Visée
bouton ｘ　Changer d'arme
bouton ｙ　Attaquer
bouton ａ　－－
bouton ｂ　Sauter
bouton +/START　Menu Pause
bouton Ｒ　Attaque fixe

●manette Nintendo GameCube

manette +　Mouvements du joueur / Visée
bouton Ｘ　－－
bouton Ｙ　Changer d'arme
bouton Ａ　Sauter
bouton Ｂ　Attaquer
START/PAUSE　Menu Pause
bouton Ｒ　Attaque fixe

・Appuyez à gauche/droite sur la manette + pour vous déplacer sur les côtés.
L'arme sera pointée dans la direction où vous allez.
・Appuyez en bas sur la manette + pour vous allonger à terre.
Vous ne pouvez plus bouger quand vous êtes allongé à terre.
L'arme sera pointée dans la même direction qu'auparavant.
・Appuyez en haut sur la manette + pour vous arrêter et viser en haut.
・Appuyez en diagonale vers le haut/bas pour viser en diagonale vers le haut/bas tout en bougeant.
・Appuyez sur le bouton Sauter pour sauter en l'air.
Sautez tout en marchant pour effectuer un grand saut en avant.
・Appuyez à gauche/droite sur la manette + tout en sautant pour avancer légèrement.
La manette + vous permet aussi de viser avec votre arme.
・Sur certaines plates-formes, vous pouvez sauter lorsque vous êtes couché pour descendre d'un niveau.
・Vous pouvez vous accrocher à certaines plates-formes en les touchant pendant un saut.
Vous pouvez vous déplacer à gauche et à droite une fois accroché à une plate-forme.
Sautez à nouveau pour lâcher la plate-forme.
・Vous pouvez escalader certains murs en les touchant.
Utilisez la manette + pour monter et descendre.
Appuyez vers l'arrière sur la manette +, puis sautez pour lâcher le mur.
・Appuyez sur le bouton Attaquer pour tirer dans la direction visée avec votre arme.
・Maintenez le bouton Attaquer appuyé pour passer en mode Attaque fixe.
・En mode Attaque fixe, appuyer sur la manette + vous permet de tirer dans n'importe quelle direction plutôt que d'avancer ou de vous allonger.
・Vous pouvez modifier le bouton activant le mode Attaque fixe dans les réglages du jeu.
Dès lors, vous ne devrez plus garder le bouton Attaquer appuyé pour passer au mode Attaque fixe.
・Si vous disposez de plus de 2 armes, vous pouvez changer l'arme utilisée.

●Modifier la fonction d'un bouton
Vous pouvez modifier une commande en allant dans le menu OPTIONS / RÉGLAGES MANETTE.

●Réglages de la fonction Vibration de la manette Nintendo GameCube
Vous pouvez modifier les réglages de la fonction Vibration de la manette Nintendo GameCube dans le menu OPTIONS / RÉGLAGES MANETTE.
