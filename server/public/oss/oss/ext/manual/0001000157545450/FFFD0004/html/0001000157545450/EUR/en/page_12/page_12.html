Nunchuk Controls
When the Nunchuk is attached to the Wii Remote, you can control Toki Tori directly with the Control Stick.

The following actions are possible while using the Nunchuk Style:
Control Stick = Move Toki Tori
Z Button = Use Item
C Button = Next Item
Please note that you can combine actions of the Wii Remote and the Nunchuk control method as you please.
