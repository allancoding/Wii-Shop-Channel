Wii Remote Controls
The Wii Remote control method allows you to guide Toki Tori by simply pointing at the screen and pressing the A Button. Toki Tori will find the quickest way to this new location himself.

The following actions are possible while using the Wii Remote as your controller:
A Button = Go to Location
A Button (Off Screen) = Stop
B Button = Use Item
Shake Left or Right = Select Item
+Control Pad = Select Item
- Button = Zoom
+ Button = Pause Menu
