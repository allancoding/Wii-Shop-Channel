Wildcard
At times, the levels in Toki Tori can seem to be impossible to solve. This is never the case, of course, but sometimes it's better to take your mind off it for a while. In this case you can use the Wildcard, which allows you to pass a level without solving it.

You can select to use it from the game's Pause Menu. There is only one Wildcard available, but you can reclaim it by going back and solving the level which you've used it on.
