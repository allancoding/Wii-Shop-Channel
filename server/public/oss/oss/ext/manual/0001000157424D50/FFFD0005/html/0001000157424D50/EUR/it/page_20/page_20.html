Puoi vedere le foto scattate al Ranch o inviarle alla bacheca Wii. Nell’Album puoi salvare fino a 20 foto. Puoi anche salvare le tue foto su una scheda SD.

Nota: se scatti più di 20 foto, le più vecchie vengono automaticamente cancellate.

Note sulla scheda SD

● Nintendo raccomanda l’uso di una scheda SD con licenza ufficiale. Possono essere usate tutte le schede SD (fino a 2 Gb), ma Nintendo può garantire solo la funzionalità dei prodotti con licenza ufficiale Nintendo.

● Quando salvi le foto di My Pokémon Ranch sulla scheda SD, non cambiare il nome della cartella o il nome del file quando trasferisci le foto sul tuo computer. Quando copi le foto da una scheda SD alla console Wii, copia la cartella usata dalla console Wii. Cambiare il nome della cartella o il nome del file può rendere impossibile leggere i dati.
