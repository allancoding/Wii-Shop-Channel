Main Menu

When you point at the screen, the Main Menu will be displayed.

Select a menu option and press .

Wii MENU

After a chat with Hayley, the game will automatically be saved, and you can exit back to the Wii Menu.

CONNECT TO DS
(see "13. Connecting to a Nintendo DS System")

Deposit Pokémon to the Ranch and withdraw Pokémon from the Ranch by connecting with Pokémon Diamond or Pokémon Pearl.

BBS
(see "16. Using the Bulletin Board System (BBS)")

Pokémon Wanted
Here you can check the Wanted Pokémon.

Ranch Info
Check things like the Ranch’s current level, the maximum number of Pokémon you can deposit and the number of Pokémon you need for the next level.

OPTIONS
(see "17. Options")

Bring Mii
Bring in a Mii created in the Wii console's Mii Channel and set a Mii as a Caretaker.

Look at My Ranch!
If this setting is ON, you can exchange pictures or Ranch status info with your Wii Friends. You can invite your Wii Friends to your Ranch via WiiConnect24 (see "24. Communicating with Wii Friends").

Guest
If this setting is ON, an unexpected Guest might show up at the Ranch via WiiConnect24 (see "23. Setting up WiiConnect24").

HELP

Review how to operate the game and read an explanation of the Main Menu.

CHANGE MODE

Change between Auto Mode and Free Mode.
Note: This only becomes available when your Ranch reaches a certain level.

DEPOSIT DETAILS
(see "18. Deposit Details")

Check the current number of deposited Pokémon and view the Pokémon Parade. You can select the Pokémon on the Ranch using ALL, TRAINER or FAVORITES under DEPOSIT DETAILS.

ALBUM
(see "20. Album")

View pictures you have taken. You can also save and delete pictures on an SD Card and post pictures to the Wii Message Board.

TAKE PICTURE
(see "19. Taking Pictures")

Take a picture of the current screen. The number shown in the bottom right corner represents the number of pictures remaining.

LOOK THIS WAY (see "8. Controls")

Make Pokémon and Miis direct their attention towards you.
