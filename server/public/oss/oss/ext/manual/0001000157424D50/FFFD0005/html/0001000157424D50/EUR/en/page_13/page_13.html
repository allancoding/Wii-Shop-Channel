13
Connecting to a
Nintendo DS System

You can deposit or withdraw Pokémon you caught or traded in Pokémon Diamond or Pokémon Pearl using the Nintendo DS.

Please Note:

● You can only withdraw Pokémon using the same DS Game Card you used to deposit them.

● If you deposited Pokémon to the Ranch and started a new game on the Nintendo DS Pokémon game, you can't withdraw the Pokémon to that Game Card.

● You can't deposit Pokémon that you have in your party. You must deposit it to Bebe's PC before connecting.

● You can deposit a maximum of 1,000 Pokémon from up to eight different Game Cards (up to eight Trainers). If you want to deposit additional Pokémon from a new Pokémon Diamond or Pokémon Pearl Game Card, withdraw all of the Pokémon deposited from any one Game Card.

● If you use an external device or software to modify the game save data, the save data might be lost or become corrupted, and you might not be able to play the game. Once save data is corrupted, it cannot be restored.

● My Pokémon Ranch save data, including Pokémon data, cannot be saved to an SD Card.

● My Pokémon Ranch can only connect to Pokémon Diamond and Pokémon Pearl Game Cards that are the same language as the My Pokémon Ranch save data. Please see “9. Getting Started” for more details regarding language settings.
