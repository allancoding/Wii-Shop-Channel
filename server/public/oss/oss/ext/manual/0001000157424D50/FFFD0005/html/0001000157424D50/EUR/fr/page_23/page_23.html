Activer la fonction WiiConnect24

Vos photos et les informations relatives à votre Ranch sont automatiquement envoyées à vos amis Wii via WiiConnect24.

Note: reportez-vous au mode d’emploi Wii – Chaînes et paramètres pour activer la fonction WiiConnect24.

● Enregistrer un ami Wii

Si vous souhaitez communiquer avec vos amis, vous devez d’abord les enregistrer en tant qu’amis Wii.

Note: reportez-vous au mode d’emploi Wii – Chaînes et paramètres (Bureau Wii).

Note: les surnoms de vos amis Wii s’affichent en gris clair jusqu’à leur enregistrement.

Attention!

● Si vous rencontrez des problèmes de connexion à Internet, lisez le code d’erreur et le message associé, puis reportez-vous à la section "En cas de problèmes" du mode d’emploi Wii – Chaînes et paramètres.

● Si vous modifiez l’heure et la date de la console Wii, le contenu du jeu risque d’être affecté. Nous vous conseillons donc d’éviter ce genre de changement une fois le jeu configuré.
