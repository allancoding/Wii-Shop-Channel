街の便利屋さん
街の中には、国造りの手助けをしてくれる人々がいます。

●パブロフの情報屋
王様の後を付いて回るペンギン・パブロフは、ダンジョンの情報や冒険者のプロフィール、王国のニュースなど、様々な情報を教えてくれます。

さらに、ここではカメラ操作のオプション設定をすることもできます。

●チャイムの酒場
チャイムの酒場では、王様直属のパーティを編成することが可能です。
また、ダンジョンを探索中のパーティは、冒険の場所や内容をリアルタイムに把握することもできます。

●モグヨシの展望台
展望台の下にいるモグヨシに話しかけると、展望台にのぼって王国を一望することができます。


さらに、モグヨシは建物のことや、国造りのための様々なお役立ち情報を教えてくれます。何か困ったことがあれば、モグヨシに聞いてみましょう。
