はじめに
「Shadow Walker 影の少年と光の妖精」とは？

光の妖精「ポポ」となり不思議な柱の近くで光って影を作り、影の中を移動する木の妖精「ルクス」をゴールまで導くアクションゲームです。


●光の妖精「ポポ」
ポポはステージを自由に
移動することができます。
また、ポポはルクスを呼ぶことができます。呼ばれたルクスはポポが呼んだ位置まで移動しようとします。
（→P.９：基本ルール その１）

●木の妖精「ルクス」
ルクスは影の中にしかいることができません。
日なたに出るとミスとなりルクスマークが減って
スタート地点、またはチェックポイントからやり直しとなってしまいます。
ルクスマークがゼロになるとゲームオーバーです。
（→P.１０：基本ルール その２）

●マホマホの柱
ステージ上には不思議な紋章が描かれた「マホマホの柱」が存在します。ポポがこの柱の近くで光ると特殊な影を作ることができ、影をつなぐことができます。ルクスはつながった影を進むことができます。
（→P.１１：マホマホの柱）
