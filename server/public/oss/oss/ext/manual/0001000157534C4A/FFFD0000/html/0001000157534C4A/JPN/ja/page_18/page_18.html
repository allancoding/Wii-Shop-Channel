アドバイス その２
Ｑ．ルクスが影を渡っても、勝手に戻ってきて
　　しまいます。

Ａ．ルクスを呼んでみたり影を消してみましょう。

マホマホの柱の影を出すだけで、ほとんどの場合ルクスは自分の判断で影を渡って移動します。
ルクスを呼ぶ場合は、太陽の影がある場所で呼ぶようにしましょう。マホマホの柱の影の上でルクスを呼ぶと、ルクスは呼ばれた場所に移動しても留まることができないと判断して、元の影を戻ってしまうことがあります。

複数のマホマホの柱の影を渡る場合などは、ルクスを呼んで誘導するより、戻って欲しくないマホマホの柱の影を消してみる方が効果的です。

※必要以上にルクスを呼ばず、一度ルクスの意思に
　任せてみましょう。
