Kung Fu Funk

Votre technique sera-t-elle assez bonne?

Kung Fu Funk est un jeu un peu farfelu inspiré des films d'arts martiaux des années 1970.

Relevez le défi avec deux ou trois autres joueurs pour voir qui sera le maître au Kung Fu.

Le jeu consiste en trois modes loufoques au cours desquels il vous faudra surmonter toute sorte d'épreuves. Comme dans les films de Kung Fu, il vous faudra tout d'abord maîtriser l'art Zen: ce mode décontracté vous permettra de vous familiariser avec les commandes.

Une fois votre Kung Fu maîtrisé, vous entrerez dans le mode d'entraînement, un montage de film qui consiste en une série de mini-jeux tous plus fous les uns que les autres.

Enfin, une fois votre technique perfectionnée, vous entrerez dans le mode de combat où vous deviendrez maître de Kung Fu dans votre propre dojo discothèque.

Kung Fu Funk se joue à une ou deux mains. Pour jouer à deux mains, utilisez la télécommande Wii avec le Nunchuk.

Pour jouer d'une seule main, déconnectez simplement le Nunchuk. Kung Fu Funk vous amènera à pratiquer toute sorte de mouvements.
