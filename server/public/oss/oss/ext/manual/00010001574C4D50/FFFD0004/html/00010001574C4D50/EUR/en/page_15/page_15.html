Saving

There are two ways to save the game in La-Mulana.


"Quick-save" is done automatically when you touch a deciphered Holy Grail Tablet. If your game ends, you can continue from the the Holy Grail Tablet that you have touched last.

You need an item named "Holy Grail" for saving a game.
Quick-save file is deleted when you quit the game.

You can "Save" a game when you press the "Down arrow key" in front of a Holy Grail Tablet. Save file is not deleted even if you quit the game.

 

You must "Save" your game before quitting the game.

About Saving

Performing any of the following operations while using the save function may result in loss of save data or damage to the Wii console:
  • Pressing the POWER Button on the Wii Remote or the POWER Button or RESET on the Wii console while data is being saved.
  • Opening the HOME Menu and selecting Wii MENU or RESET while data is being saved.
  • Disconnecting the Wii Power Supply from the DC Input Connector or removing the AC Adapter from the mains.
Lost save data cannot be recovered, so please be careful.
