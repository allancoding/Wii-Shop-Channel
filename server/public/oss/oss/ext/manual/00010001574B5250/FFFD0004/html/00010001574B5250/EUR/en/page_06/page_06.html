Start Screen

To play the single player tournament mode press any Button at the startup screen

 

 

The Start screen
Character select screen

 

In the character select screen, use the pointer and press the
A Button to select your fighter.

 

The first time you play you can only select one of the five starting characters, but through winning the tournament and playing multiplayer fights you can unlock exciting new characters.

 

 

The Character select screen
Stage select screen

 

In the stage select screen, use the pointer and press the
A Button on one of the stage pictures to select the arena in which to fight.

 

 

The Stage select screen
Fight screen

 

You win a round by reducing the opponentÂ´s health to 0 or by having the most health left when the timer runs out.

The winner of a fight is the first fighter to win 2 rounds.

 

 

The Fight screen
Winning the tournament

 

To win the tournament you first have to defeat all the five starting characters to fight the final boss and then defeat him.  After you have defeated the boss he will be available for you to choose in the character selection screen.
