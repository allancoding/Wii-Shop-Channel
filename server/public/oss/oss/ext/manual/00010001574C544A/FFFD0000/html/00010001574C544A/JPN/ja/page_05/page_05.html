ゲーム内容
スクール オブ ダークネスはWiiリモコンとヌンチャク両方を使用する、ダークホラーパズルゲームです。プレイヤーは主人公ジェイクを操作して、暗い教室を入り口から出口のドアまで進み、次の部屋へと向かいます。それぞれの部屋は灯りが消えた状態からスタートします。闇に踏み込むと、ジェイクは見えないモンスターに引きずり込まれてしまいます。出口のドアへたどりつくには光源を発見し、灯りをつけてジェイクが安全に歩けるよう床を照らさなくてはなりません。光を灯すには、直接スイッチを入れるほかにも、離れた場所から物をぶつける、モーションセンサーの前を通過するなど、様々な方法があります。

 

光源のほとんどは電気製品ですので、電力を消費します。画面上の「電力ゲージ」は、現在の電力消費量を表示します。このゲージが上限を超えると、部屋の灯りもすべて消え、ステージはやり直しになります。

 

出口に到着すると、プレイヤーは次の教室へと向かいます。ゲームの進捗は、部屋を出るとき自動的にセーブされます。教室のパズルを5つクリアするごとに、ジェイクはボスと戦わなければなりません。ボスステージでは、光源を使ってボスを倒せばクリアになります。ボスステージでは、出口へ到着する必要はありません。

 

ジェイクのガールフレンド、レイチェルも学校に閉じ込められていて、ときどき校内の電話でジェイクに連絡をしてきます。プレイヤーがレイチェルの電話に応えられるかどうかで、ゲームのエンディングが変化します。追加要素の解放にも影響します。
