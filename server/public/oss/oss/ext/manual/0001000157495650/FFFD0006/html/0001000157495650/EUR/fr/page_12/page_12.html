Attaques spéciales



En cas d’urgence, appuyez sur pour appeler le
« vaisseau-mère » qui viendra vous aider avec son puissant rayon laser.
Pour utiliser cette « attaque spéciale », vous devez puiser dans la réserve spéciale affichée dans le coin inférieur gauche de l’écran. Chaque attaque spéciale consomme un élément de cette réserve.

Quand vous utilisez une attaque spéciale, votre OVNI et les envahisseurs qui l’entourent sont invulnérables aux attaques des ennemis. Par ailleurs, votre OVNI récupérera automatiquement son escorte de 100 envahisseurs une fois celle-ci effectuée.
