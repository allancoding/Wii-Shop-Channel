追加コンテンツについて

追加コンテンツを購入することで、さらに多くのステージを遊ぶことができます。

追加ステージパックは３種類あり、ショッピング画面から購入することができます。

ステージパックはAから順に購入しなくても、好きなものだけ購入することが可能です。
追加コンテンツを購入するためには、それぞれ500Wiiポイントが必要です。
ステージパック A

強力な戦艦が登場する「港湾都市」と、ジャングルに潜むチキュウ人たちと戦う「森林都市」の２ステージ６エリアのパックです。
ステージ２　港湾都市 ステージ３　森林都市
ステージパック B
ガスタンクとサーチライトが特徴的な「鋼鉄都市」と、チキュウ人の猛攻撃をしのぐ「砂漠都市」の２ステージ６エリアです。

ステージ４　鋼鉄都市 ステージ５　砂漠都市
ステージパック C
吹雪が吹き荒れる「氷結都市」と、最後のコアが眠る土地「灼熱都市」の２ステージ６エリアです。
ステージ６　氷結都市 ステージ７　灼熱都市
