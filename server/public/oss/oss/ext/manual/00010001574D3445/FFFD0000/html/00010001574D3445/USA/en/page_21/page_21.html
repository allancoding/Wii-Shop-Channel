Loop Shop

The Loop Shop can be accessed from both the Launch Menu and Mix Menu icon by selecting the 'Loop Shop'. Follow the on-screen prompts to view and download these awesome loop pack additions available for Mix Superstar and take your mixes to a new level!

To download additional loop packs start by selecting one of the four main music genres and you will be presented with the loop packs currently available for the genre. Use the A Button to select the pack you wish to download and notice the album cover will flip around with a description of the loop pack. Press the 'Download' Button and follow the on-screen steps to download the loop pack.

If you don't have enough free blocks to download a pack, you may remove an existing loop pack using the 'Remove Packs' Button. You will now have a list of loop packs that you've previously downloaded. Select the pack to remove by pressing the A Button and then select the 'Remove' Button. All removed loop packs may be re-downloaded as often as you wish for no charge.

There are a variety of music loop packs that can be downloaded within this game. Wii Points are additionally required to download any available loop packs. (This involves a fee). Free space is required in the Wii system memory to download loop packs. If there is not enough free space, either delete unnecessary data saved in Wii system memory or move it to an SD Card.

All downloaded loop packs can only be used with Mix Superstar.
