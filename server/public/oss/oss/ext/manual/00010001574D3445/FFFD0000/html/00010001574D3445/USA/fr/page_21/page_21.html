Boutique de Boucles

Vous pouvez accéder à la Boutique de Boucles à partir du Menu de Lancement et de l'icône du Menu de Mixage en sélectionnant Boutique de Boucles. Suivez les indications affichées sur l'écran pour visualiser et télécharger les lots de boucles supplémentaires fantastiques à votre disposition pour Mix Superstar, et passez à un niveau de mixage supérieur et inédit!

Pour télécharger des lots de boucles supplémentaires, commencez d'abord par sélectionner un des quatre genres musicaux pour voir les lots de boucles actuellement disponibles dans ce genre. Utilisez le bouton A pour sélectionner le lot de boucles que vous souhaitez télécharger et observez comment la couverture de l'album se retournera pour afficher une description du contenu du lot de boucles. Appuyez sur le bouton Télécharger et suivez les indications affichées sur l'écran pour le téléchargement.

Si vous ne disposez pas suffisamment de blocs libres pour télécharger un lot de boucles, vous pouvez en enlever un en utilisant le bouton Enlever les Lots. Une liste des lots de boucles déjà téléchargés sera affichée. En appuyant sur le bouton A, sélectionnez d’abord le lot à enlever et, ensuite, sélectionnez le bouton Enlever. Tous les lots de boucles enlevés pourront être téléchargés de nouveau gratuitement, autant de fois que vous le souhaitez.

Ce jeu comprend une série de lots de boucles musicales pouvant être téléchargés. De plus, tout lot de boucles disponible ne peut être téléchargé qu'avec des Points Wii. (Cela implique des frais). La mémoire de la console Wii doit disposer d’un espace libre pour le téléchargement de lots de boucles. S'il n'y a pas suffisamment d’espace libre, supprimez les données superflues sauvegardées sur la mémoire de la console Wii ou mettez-les sur une carte mémoire SD.

Tous les lots de boucles téléchargés ne peuvent être utilisés qu'avec Mix Superstar.
