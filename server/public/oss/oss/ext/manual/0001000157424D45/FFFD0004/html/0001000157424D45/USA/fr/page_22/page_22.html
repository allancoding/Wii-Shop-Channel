Préparation WiiConnect24

Tes photos et les informations sur le Ranch sont automatiquement envoyées à tes amis Wii via WiiConnect24.
* Pense à lire le mode d'emploi électronique et à activer le service WiiConnect24 (On).

Register a Wii Friend (inscrire un ami Wii)

Lorsque tu veux communiquer avec un ami, il faut d'abord l'inscrire dans tes amis Wii.
* Pense à lire le mode d'emploi électronique (Bureau Wii).

* Lorsque toi et ton ami vous êtes mutuellement inscrits en tant qu'amis, la couleur du surnom dans le carnet d'adresses du Bureau Wii passe du gris clair au noir.

Attention!

  • Si tu as un problème pour te connecter à Internet, réfère-toi aux sections Error Code (code d'erreur) et Error Message (message d'erreur) et à la section What to do if this happens? (Que faire si cela arrive?) du mode d'emploi électronique.
  • Si tu changes l'heure sur la Wii, cela peut avoir une influence dans le jeu. Il est conseillé de ne pas changer l'heure après avoir configuré le jeu.
