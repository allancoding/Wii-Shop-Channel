このゲームについて
このゲームは、自分で勇者を育てる育成パートの『シナリオ～力を求めて～』と、そのオリジナル勇者を操作して武術大会のトーナメントを勝ち抜く『武術大会モード』の2つのモードに分かれています。

■シナリオ～力を求めて～（育成パート）
「勇者になる前の若者」が主人公で、その若者の冒険を通じて勇者会にその活躍を認められ、勇者に任命されることが目標です。

１週間を１ターンとして時間が進んでいき、決められた期間内での活躍により、勇者に任命されるかどうかが左右されます。

■武術大会モード
オリジナル勇者で、様々なルールが定められた数々の大会トーナメントに挑戦することができます。また、ルールを自分で設定した大会を開催することもできます。

オリジナル勇者をパスワード化して交換する事で、他の人の作った勇者を武術大会モードに参加させる事もできます。
