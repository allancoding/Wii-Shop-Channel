大会のルールについて
武術大会では大会ごとにそれぞれルールが存在し、「主催する」を選択した場合は、これらのルールを全て自分で設定することができます。ルールの種類には以下のようなものがあります。

■ルールについて

◆大会名
大会の名前

◆チーム編成
チームの編成に関するルール
シングル・ペア・トリオが存在

◆魔法
魔法の使用に関するルール
制限なし・禁呪禁止・すべて禁止が存在
※禁呪とは、魔法「ザラストサパー」「プレラスト
　サパー」を指します

◆装備
装備に関するルール
制限なし・すべて禁止が存在

◆最大レベル制限
大会に参加出来る勇者のレベル制限に関するルール
制限なし・30・40・50が存在

◆トーナメント方式
大会トーナメントの形式に関するルール
ノーマル・パラマスが存在

◆ガード時の回復
戦闘中、防御をしたときに関するルール
あり・なしが存在
※ありの場合、防御すると少しだけHPが回復します

◆大会中での成長
大会中でのレベルアップに関するルール
あり・なしが存在
※ありの場合、戦いに勝利すると経験値を獲得し、
　必要な経験値を満たすとレベルが上がります

◆サプライズ
戦闘中に発生するサプライズに関するルール
あり・なしが存在
※味方チームが得をするのか、敵チームが得をする
　のか、何が起きるのかも分かりません

◆開始時アイテム
大会開始時に配布されるアイテムに関するルール
主に回復アイテムの中から、1個から99個まで任意の数を、最大で3種類まで設定可能

◆試合毎アイテム
試合ごとに配布されるアイテムに関するルール
主に回復アイテムの中から、1個から99個まで任意の数を、最大で3種類まで設定可能

◆おジャマ装置
戦闘中に作動する、おジャマ装置に関するルール
状態異常やステータスに影響するガスが噴出したり、モンスターが乱入する装置が存在
同じものを含め、最大6個まで設置可能
