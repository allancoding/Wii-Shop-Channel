7
Astuces

Ne manquez pas le point de contrôle!
Sur votre carte, les points de contrôle sont en couleur verte. Essayez de les toucher tous! Si vous manquez un point de contrôle, vous perdez une vie.


Accélération!
Vous pouvez utiliser votre accélérateur pour atteindre les points de contrôle difficiles.
Appuyez pour accélérer. Le nombre d'accélérations est limité, n'oubliez donc pas de ramasser les Bonus d'accélération supplémentaires lorsque vous tombez.

Tirez ou on vous tire dessus
Désarmez la défense des ennemis avant qu'ils n'aient l'occasion de vous tirer dessus! Appuyer pour tirer. Les cibles prioritaires apparaissent en rouge; abattez-les rapidement afin de détruire tous les champs de force se trouvant sur votre chemin.


Champ de force
Les Champs de force apparaissent en rouge et si vous ne les éteignez pas, ils vous tueront. Pour éteindre un champ de force, vous devez détruire un Avion de combat lourd.


Contrôle de la vitesse
Utilisez l'aérofrein pour ralentir et contrôler votre vitesse de chute. Appuyez pour activer l'aérofrein. Appuyez à nouveau pour le désactiver.

Évitez un atterrissage brutal!
Utilisez votre aérofrein pour diminuer la vitesse lors de l'approche finale à la Zone de largage. Vous serez tué pendant l'atterrissage, si vous n'utilisez pas l'aérofrein.
