Controles en pelea

Control de la cámara:

  • Botón C + Palanca de control para rotar la cámara.
  • Botón Z + Palanca de control para realizar un zoom con la cámara.

    Tomar control de una articulación:

  • Utiliza la palanca de control para seleccionar una articulación.

    Ejecutar una fuerza en la articulación:

  • El Botón A ejecuta la fuerza (contra la articulación). Presiona nuevamente el Botón A para generar una fuerza contraria (extender la articulación).
  • La cruz de control también puede ser utilizada para seleccionar fuerzas en las articulaciones.
  • Agita el Wii Remote para SOBREESTIMULAR una articulación.

    Mantener quietas todas las articulaciones:

  • Utiliza el Botón - para lograr que todas las articulaciónes se queden quietas (HOLD).

    Mueve la escena hacia adelante:

  • Con el Botón B, la escena se mueve hacia adelante, y ves tu movimiento desenvolverse.
