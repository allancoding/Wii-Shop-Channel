Trucs

●Vous n'arrivez pas à trouver le poisson que vous cherchez?
Lancez-vous votre ligne toujours au même endroit, même si vous ne voyez pas de poissons?
Si vous voulez trouver le poisson que vous cherchez, essayez de lancer votre ligne à différents endroits. Parfois, les poissons peuvent venir voir votre appât si vous attendez après avoir lancé votre ligne. Si vous n'arrivez toujours pas à trouver le poisson que vous cherchez, essayez avec un autre appât.

 

Appuyez sur le bouton - quand votre ligne est à l'eau pour changer l'angle de caméra. Vous pourrez ainsi trouver des poissons plus facilement.

 

●Choix d'appât
Attrapez-vous souvent des poissons que vous ne cherchez pas?
Comme vous le savez peut-être, certains poissons sont plus attirés par certains types d'appâts. Si vous attrapez continuellement des poissons que vous ne cherchez pas, vérifiez l'appât que vous utilisez et changez-le pour utiliser un appât plus adapté au type de poisson que vous cherchez.

 

●Indicateur de tension
Quand vous attrapez un poisson, vous pouvez voir l'indicateur de tension à la gauche de l'écran. Cet indicateur affiche la tension sur la ligne. Plus l'indicateur monte, plus la tension sur la ligne est élevée. Plus l'indicateur baisse, plus la tension sur la ligne est faible et plus le poisson risque de s'échapper.

 

Gardez un oeil sur l'indicateur pendant que vous pêchez pour ramener vos prises.

 

Si vous souhaitez vivre une expérience plus réaliste, désactive l'indicateur de tension dans le menu «Options» pour voir si vous pouvez attraper un poisson sans aide!
