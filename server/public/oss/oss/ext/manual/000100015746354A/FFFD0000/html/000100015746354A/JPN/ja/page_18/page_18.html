おきらくスピードのルール
スピードは素早い状況判断と反射能力を競うトランプゲームです。

  1. ジョーカーを除く５２枚のトランプを、赤と黒の２組に分けて、両者が１組ずつ持ちます。
  2. 自分の前に４枚のカードを表にして並べます（場札）。残りは裏のまま持ちます（手札）。
  3. 「いっせーの」で、２人の手札からカードが１枚、中央に表向きに出されます（台札）。
  4. 場札の中に、台札とひとつ違いのカード（KとAはひとつ違いとします）があれば、そのカードを台札に重ねていくことができます。
  5. 場札を台札に出して減った場札は、手札から補充されます。
  6. 両者、場札を出せなくなったら、３．に戻ります。
  7. 先に自分の手札と場札を無くしたプレイヤーが勝ちになります。
