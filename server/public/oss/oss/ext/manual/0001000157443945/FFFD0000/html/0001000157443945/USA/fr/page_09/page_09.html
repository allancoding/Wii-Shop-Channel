●Manette Nintendo GameCube

croix directionnelle　Mouvements du joueur
bouton Ｂ　Attaquer
bouton Ａ　Sauter
START/PAUSE　Pause
  • Appuyer sur gauche ou droite sur la croix directionnelle pour marcher à gauche ou à droite.
  • Appuyer sur le bouton de saut pour sauter vers le haut.
    Sauter en marchant vous permet de sauter en diagonale.
  • Appuyer sur haut ou bas sur la croix directionnelle près d'un escalier pour le monter ou le descendre.
  • Appuyer sur bas sur la croix directionnelle, quand il n'y a pas d'escalier, pour vous accroupir.
  • Appuyer sur le bouton d'attaque pour attaquer avec le fouet.
  • Si vous avez une arme secondaire, elle peut être utilisée en dépensant les cœurs récupérés.
    Appuyer sur le bouton d'attaque en maintenant la croix directionnelle vers le haut pour utiliser l'arme secondaire.

●Modifier la fonction d'un bouton

Vous pouvez modifier une commande en allant dans le menu OPTIONS / CONTROLLER SETTINGS.

●Réglages de la fonction vibration de la manette Nintendo GameCube

Vous pouvez modifier les réglages de la fonction vibration de la manette Nintendo GameCube dans le menu OPTIONS / CONTROLLER SETTINGS.
