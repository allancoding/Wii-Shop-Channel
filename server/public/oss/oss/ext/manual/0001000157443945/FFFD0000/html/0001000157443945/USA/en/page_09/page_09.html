Controls

●Wii Remote

+Control Pad　Player movement
1 Button　Attack
2 Button　Jump
＋ Button　Pause menu

●Classic Controller

+Control Pad　Player movement
ｂ Button　Attack
ａ Button　Jump
＋ Button　Pause menu

●Nunchuk Style

Control Stick　Player movement
Ａ Button　Attack
Ｂ Button　Jump
＋ Button　Pause menu
Swing Wii Remote　Attack

●Nintendo GameCube Controller

+Control Pad　Player movement
Ｂ Button　Attack
Ａ Button　Jump
START/PAUSE　Pause menu
  • Press left/right on the +Control Pad to walk left/right.
  • Press the jump button to jump vertically.
    Jumping while walking allows you to jump diagonally.
  • Press up/down on the +Control Pad near stairs to go up/down them.
  • Press down on the +Control Pad when there are no stairs around to crouch.
  • Press the attack button to attack with the Whip.
  • If you have a secondary weapon, it can be used by consuming collected hearts.
    Press the attack button while holding down up on the +Control Pad to use the secondary weapon.

●Changing Button Assignment

You can change the button assignment by going to OPTIONS and then CONTROLLER SETTINGS.

●Nintendo GameCube Controller Rumble Settings

You can change the Nintendo GameCube Controller Rumble settings by going to OPTIONS Menu and then CONTROLLER SETTINGS.
