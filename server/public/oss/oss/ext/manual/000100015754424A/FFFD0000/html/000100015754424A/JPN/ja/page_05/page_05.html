はじめに
● ファンタジックタンバリンとは

なんでも作れる不思議なタンバリンを持っている

女の子「バリンちゃん」と

そのおともの「タンバ」達と一緒に建物を作っていく音楽ゲームです。

音楽に合わせて上手にタンバリンをふり建物の完成をめざしましょう。

 

・タイミングを合わせてふろう

「うちゅうぞう」の鼻からでてくる「タンバボール」と曲に注意してタイミングよくタンバリンをふると点数がもらえます。

「タンバボール」の色にも注意しましょう。

 

・フィーバータイムで得点稼ぎ！

「うちゅうぞう」の鼻からでてくる「フィーバーボール」にあわせて３つタンバリンをたたくとフィーバータイムに突入！

通常モードより多く得点がもらえます。

曲のリズムに気をつけてどんどんタンバリンをたたきましょう。

 

・最高ランクの建物をめざせ！

得点に応じて完成する建物が変わります。マップもそれに応じて変化するので、最高ランクの建物でマップを埋め尽くしてみましょう。

ステージクリアに失敗すると建物も失敗・・・。
