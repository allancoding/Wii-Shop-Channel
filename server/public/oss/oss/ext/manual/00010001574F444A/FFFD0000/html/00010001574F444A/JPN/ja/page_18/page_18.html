外交／魔王国への使者について
魔王国に使者を送ると魔王城への登城イベントが発生して、魔王にお願いごとをすることができます。 あなたの国と魔王国の親密度が高い場合、お願いを聞いてもらえることがあります。

お願いできる内容には以下のものがあります。

◆魔物の派遣をお願いする
モンスターを指定したエリアに派遣してもらい、そこで暴れてもらいます。

◆ダンジョンの占拠をお願いする
指定したダンジョンをモンスターに支配してもらいます。

◆エリア侵攻をお願いする
指定したエリアへ魔王軍に攻め込んでもらいます。
魔王軍が勝てば、そのエリアは魔王支配になります。

◆王になりすましてもらう
指定した国の王様になりすまして、悪事をはたらいてもらいます。

◆魔物を貸してもらう
魔物を一人、部下として貸してもらいます。
借りた部下は、あなたの部下としてはたらいてくれます。

◆王女をさらってもらう
指定した国の王女をさらってもらいます。
