Objet
Utiliser les objets
Les blocs objet (à droite) apparaîtront pendant le jeu si le paramètre "Objet" est sur OUI dans les modes "Bataille VS", "Connexion Wi-Fi Nintendo" ou autres modes multijoueurs. Éliminez la ligne contenant le bloc pour obtenir l'objet.

Appuyez sur pour utiliser l'objet obtenu. Vous pouvez stocker 3 objets maximum et les utiliser dans l'ordre dans lequel vous les avez obtenus. Si vous avez déjà 3 objets et en obtenez un 4ème, l'objet le plus ancien disparaîtra.

Types d'objet
Il y a 10 types d'objet. Pour les objets avec la marque , tenez la télécommande Wii verticalement et utilisez le pointeur pour augmenter leur effet. (Ils peuvent également être utilisés sans le pointeur.)
Fumée [Effet : 1 adversaire]

Déplacez votre pointeur sur le champ de l'adversaire, puis appuyez sur pour l'envelopper de fumée. Secouez la télécommande Wii pour dissiper la fumée.
Turbo [Effet : tous les adversaires]

Utilisez cet objet pour augmenter la vitesse de chute des tétrominos de l'adversaire pendant un certain temps.

Bombardement [Effet : tous les adversaires]

Déplacez votre pointeur sur un espace vide puis appuyez sur pour faire tomber des blocs. Appuyez rapidement sur pour envoyer plus de blocs.
Effacement [Effet : tout le monde]

Utilisez cet objet pour effacer des lignes. Choisissez la ligne à effacer avec le pointeur.
Cascade [Effet : soi-même]

Utilisez cet objet pour poser un tétromino qui tombera en morceaux et remplira les espaces vides.

Verrou [Effet : tous les adversaires]

Utilisez cet objet pour empêcher les tétrominos de vos adversaires de tourner pendant un certain temps.

Escalade [Effet : tous les adversaires]

Utilisez cet objet pour faire apparaître un grimpeur dans le champ des adversaires. Si le grimpeur est écrasé, des blocs de pénalité apparaissent depuis le bas.

Téléportation [Effet : soi-même]

Déplacez votre pointeur vers l'un des cotés de votre champ, puis appuyez sur pour aligner les espaces vides vers le pointeur.
Montre [Effet : tous les adversaires]

Utilisez cet objet pour paralyser temporairement vos adversaires. Vous pouvez continuer à déplacer librement vos tétrominos.

Disparition [Effet : 1 adversaire]

Déplacez le pointeur sur le champ de votre adversaire, puis appuyez sur pour cacher temporairement leur PROCHAIN tétromino.
Vous pouvez changer les objets apparaissant pour chaque règle de jeu en sélectionnant "Objet" dans le menu "Options". Si vous jouez en mode "Connexion Wi-Fi Nintendo", vous pouvez sélectionnez OUI (Objets)/NON (Sans objet) comme condition de recherche de joueurs.
