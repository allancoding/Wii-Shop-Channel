À propos des commandes du jeu

● Tenir la télécommande Wii verticalement
Vous devez tenir la télécommande Wii
verticalement lors de l'utilisation de certains objets dans le jeu.

● Utiliser le Wii Balance Board™
Le Wii Balance Board (vendu séparément) est nécessaire pour jouer aux modes "Marathon balance", "Ultra marathon balance" et "Bataille balance ordinateur" du Wii Balance Board Tetris. Pour plus de détails, référez-vous à "Voir les règles" dans le menu des options du jeu.
