Multijoueurs
Bataille VS
Jouez avec les règles de base de bataille Tetris.
Jouez contre un rival et/ou l'ordinateur en effaçant les lignes et en rendant vos adversaires incapables de jouer (GAME OVER) pour remporter la partie.
Le nombre de lignes effacées détermine le nombre de blocs de pénalité envoyés dans le champ des adversaires.
Escalade VS
Les règles de base sont les mêmes que pour "Escalade". Vous remportez la partie si vous êtes le premier à franchir l'arrivée ou si votre adversaire ne peut plus jouer. À chaque fois qu'un joueur passe un checkpoint, il obtient un "Effacement".
Lignes VS
Effacez les lignes désignées en vert. Si vous effacez un tétromino sur une ligne, la ligne disparaîtra avec celui-ci. Vous remportez la partie si vous êtes le premier à effacer toutes les lignes ou si votre adversaire ne peut plus jouer. Si vous jouez en équipe, quand votre coéquipier efface une ligne, la même ligne disparaîtra de votre champ.
Ombres VS
Les règles de base sont les mêmes que pour "Ombre". Le gagnant est le joueur avec le plus haut pourcentage de complétion à la fin de la partie. Dans le cas où un joueur réalise 100% de complétion, la partie s'arrête.

   Affichage "Passer"

Le nombre de fois que vous pouvez passer un tétromino est affiché ici.
Course VS
Les règles de base sont les mêmes que pour "Course". Vous remportez la partie si vous êtes le premier à atteindre l'arrivée ou si vos adversaires ne peuvent plus jouer.
Tetris co-op
Un jeu à 2 joueurs où l'on joue ensemble pour effacer des lignes sur un champ deux fois plus grand (20 blocs de largeur) qu'un champ normal. Les tétrominos qui entrent dans le champ diffèrent selon le joueur. Cependant, les règles sont les mêmes que pour Tetris de base. "Garder" est partagé entre J1 et J2.
Espace
Un jeu à 2 joueurs où l'on s'affronte sur un champ deux fois plus grand (20 blocs de largeur) qu'un champ normal. Les règles diffèrent de celles d'un Tetris de base. Les joueurs s'affrontent pour voir qui peut enclore le plus d'espace avec leurs tétrominos (ceux-ci comptant également comme espace).
À tour de rôle, les joueurs placent un tétromino. Le joueur dont le tétromino bouche en dernier un espace obtient cet espace. Le joueur avec le plus d'espace à la fin de partie est le gagnant.

À propos de la sauvegarde

Pour sauvegarder vos records et résultats en mode multijoueurs, suivez les indications suivantes.

  1. À l'écran de sélection du joueur, déplacez le curseur sur la barre correspondant au numéro du joueur.
  2. Dirigez-vous vers la
    gauche ou la droite avec pour sélectionner
    "HUMAIN" et appuyez sur .
  3. Quand l'écran de sélection des données s'affiche, sélectionnez vos propres données de sauvegarde et appuyez sur .
Lorsque vous retournez à l'écran de sélection du joueur, le nom des données de sauvegarde s'affiche sur la barre.
Si vous réglez un joueur sur "INVITÉ", les données de ce joueur ne seront pas sauvegardées. Si c'est sur "ORDI", l'ordinateur jouera. Si c'est sur "NON", il ne pourra pas participer (le nombre total de joueurs sera réduit).
