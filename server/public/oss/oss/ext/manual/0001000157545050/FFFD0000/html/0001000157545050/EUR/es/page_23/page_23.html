Opciones
Región
Selección de la región que se quiere registrar para la clasificación.
Configuración del mando de Wii
Puedes elegir entre dos configuraciones del mando de Wii. No puedes cambiar la configuración del mando clásico.
Ghost
Activar/desactivar Ghost durante el juego.
Este ajuste se aplica a todos los modos de juego.
Fondo
Cambia el fondo que aparece durante el juego.
Este ajuste se aplica a todos los modos de juego.
Sonido
Se usa para cambiar la salida de audio
(monoaural/estéreo) y el volumen de FX/BGM.
Mii
Activa/desactiva que se muestre el Mii durante el juego.
Mostrar el Mii de los demás jugadores
Para elegir si se quiere o no mostrar el Mii creado por otro jugador durante la Conexión Wi-Fi de Nintendo o durante un Torneo.

* Si se selecciona "Sustituir", se mostrará un Mii
predeterminado.

Créditos
Para ver los nombres del personal.

Además de estas opciones, puedes cambiar los ajustes de las reglas para cada juego. Para más información, selecciona "Opciones" del juego.
