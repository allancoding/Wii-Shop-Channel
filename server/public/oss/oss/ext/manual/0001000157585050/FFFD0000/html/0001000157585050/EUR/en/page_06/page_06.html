Main Menu

On the Main Menu for Paint Splash you can use your Wii Remote to select which mode you want to start painting in. Point the cursor at the item you wish to select and then press the A Button.

Select Paint Mode for a regular painting program, where you can choose brush type, brush color and brush size in the tools menu, and draw exactly what you want.

Select Splash Mode for a chaotic painting experience, where random things happen when you push the buttons, point at the screen or shake the Wii Remote.

Both modes can be enjoyed by up to 4 people at the same time. Each person must have their own Wii Remote to paint.
