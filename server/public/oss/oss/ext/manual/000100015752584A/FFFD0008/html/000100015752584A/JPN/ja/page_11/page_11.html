各モードの説明


GAME START
ゲームを開始します。この項目を選択すると続けて下記のメニューが出ます。

NEW GAME

新しくゲーム本編を開始します。


LOAD

データをロードしてセーブしたところからゲームを開始します。


TIME ATTACK

各ステージのタイムアタックができます。記録はランキングに登録されます。タイムアタックのリプレイデータを保存することができます。


ENDLESS ATTACK

追加コンテンツを購入すると表示されます。無限に続くステージをどこまで進めるか挑戦します。記録はランキングに登録されます。


CHALLENGES

さまざまな条件が設定された短いステージをクリアするチャレンジゲームを行ったり、ゲーム本編で達成するチャレンジのリストを確認することができます。


RANKINGS

ランキングを見ることができます。


OPTIONS

オプション設定を変更できます。


ADD-ON CONTENT

追加コンテンツを購入することができます。
