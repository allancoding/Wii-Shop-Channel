追加コンテンツ
本ゲームには下記の追加コンテンツが用意されています。
追加コンテンツの種類

①フォルテモード

②エンドレスアタック

③スペシャルステージ１

④スペシャルステージ２

⑤スペシャルステージ３


※追加コンテンツは、ゲームメニューの「ADD-ON CONTENT」から購入することができます。

※追加コンテンツの購入には、Wiiポイントが必要です。
各追加コンテンツについて
①フォルテモード

フォルテで本編を遊ぶことができます。

弾を７方向に撃ち分けることができます。

その他、一部性能がロックマンとは異なります。

ランキングやチャレンジには対応していません。
 必要なWiiポイントは、購入画面でご確認ください。

※「フォルテモード」は、

「ロックマン１０　宇宙からの脅威！！」で使用する追加コンテンツです。
※「フォルテモード」単体では遊べません。
※「フォルテモード」の受信には、Wii本体保存メモリに空き容量が必要です。

空き容量が足りない場合は、Wii本体保存メモリに保存されている不要なデータを削除するか、SDメモリーカードに移動してください。
②エンドレスアタック

無限に続くステージで遊べます。

どこまで進められるかの記録に挑戦します。

ランキングに対応しています。

チャレンジには対応していません。
 必要なWiiポイントは、購入画面でご確認ください。

「ロックマン１０　宇宙からの脅威！！」で使用する追加コンテンツです。
