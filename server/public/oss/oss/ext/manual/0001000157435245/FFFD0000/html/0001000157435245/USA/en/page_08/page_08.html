Using the Wii Remote

In order to play Carnival King, hold the Wii Remote 8-10 feet from the television screen. A round cursor appears on screen. Move the Wii Remote to move the cursor around the screen. Point the top of the Wii Remote at the screen to select options or shoot targets.



Start/Select/Shoot



Start/Select/Shoot



Press to display or hide the on-screen cursor.



During the game pressing pauses the action and opens the Pause Menu. Shoot Calibrate to re-calibrate the shooting function or Quit to exit. Pressing again resumes game play.


Note:
Pausing games to scope the target and then shoot is not advised because pausing four times in any park prevents scores from posting to the leaderboards. A warning will appear after pausing three times.



After re-calibration, pressing restores the default calibration.
