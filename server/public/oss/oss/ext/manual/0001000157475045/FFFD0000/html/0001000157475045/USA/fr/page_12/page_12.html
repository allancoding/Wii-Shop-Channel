Faire nager un Mii

Pointez Swim with a Mii (faire nager un Mii) dans le menu de l'aquarium et appuyez sur pour faire nager un des personnages Mii sauvegardés sur la console Wii dans l'aquarium!

● Sélectionner un Mii
Pour sélectionner le Mii qui nagera dans l'aquarium, pointez un des personnages Mii apparaissant à l'écran et confirmez votre sélection en appuyant sur . Les personnages Mii sauvegardés sur la console Wii sont automatiquement inclus dans le groupe de personnages Mii que vous pouvez sélectionner.

Note : Les personnages Mii peuvent être créés dans la Chaîne Mii. Pour plus d'informations sur la façon de créer un Mii, veuillez vous référer au mode d'emploi de la Wii.

● Contrôler un Mii
En sélectionnant un Mii, vous pouvez le regarder nager dans l'aquarium. Vous pouvez également nourrir les poissons et obtenir plus d'informations sur les poissons sélectionnés ou sur l'aquarium.
