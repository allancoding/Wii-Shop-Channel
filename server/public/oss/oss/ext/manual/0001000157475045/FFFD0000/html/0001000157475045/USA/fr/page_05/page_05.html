Introduction

● AquaSpace, qu'est-ce que c'est?

Ce logiciel vous permet d'observer et d'apprécier divers poissons, ainsi que d'en apprendre un peu plus sur ceux-ci dans la section Encyclopedia, une encyclopédie illustrée dédiée aux poissons.

・Observer les poissons dans l'aquarium
Vous pouvez observer les poissons présents dans l'aquarium en choisissant votre angle de vue préféré. Vous pouvez regarder l'action du point de vue d'un poisson, pendant que les habitants de l'aquarium nagent et se nourrissent.

・Obtenir des informations sur les poissons dans l'encyclopédie
Grâce à l'encyclopédie, vous pouvez découvrir des informations sur les poissons présents dans l'aquarium telles que leur taille, leur type ou leur habitat (voir p. 7). Lorsque vous explorez un aquarium, vous pouvez en apprendre un peu plus sur un poisson en lisant sa description (voir p. 10).
