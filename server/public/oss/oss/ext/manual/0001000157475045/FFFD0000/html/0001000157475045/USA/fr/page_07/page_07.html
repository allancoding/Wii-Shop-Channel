Pour commencer

Lorsque vous démarrez le logiciel et appuyez sur à l'écran titre, le menu principal apparaît. Dans ce menu, vous pouvez sélectionner un aquarium ou consulter l'encyclopédie.

Utiliser la fonction d'aide

Vous pouvez utiliser la fonction d'aide pour vérifier les commandes et en savoir plus sur la façon d'utiliser ce logiciel. Pour utiliser la fonction d'aide, sélectionnez dans le coin supérieur droit de l'écran et appuyez sur .

● Regarder un aquarium
Pour regarder un aquarium, sélectionnez-en un parmi les huit disponibles dans le menu principal. Lorsque vous pointez un aquarium, une description de son contenu s'affiche. Pointez un aquarium et appuyez sur pour accéder à l'écran de l'aquarium (voir p. 9).

● Consulter l'encyclopédie
Pour consulter l'encyclopédie, sélectionnez Encyclopedia (encyclopédie) dans le menu principal. Une fois dans l'encyclopédie, vous pouvez choisir parmi divers types de poissons. Une fois que vous avez sélectionné un type, pointez une des photos de poisson et appuyez sur pour lire la description du poisson choisi.
