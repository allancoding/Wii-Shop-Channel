Type de partie
Jusqu'à 4 joueurs peuvent participer selon le type de tournois. Vous pouvez
paramétrer les handicaps pour les fautes (sauf lorsque vous rentrez la bille blanche) selon le niveau de chaque joueur.

● Attribuer les télécommandes Wii
Vous pouvez utiliser une seule télécommande Wii que chaque joueur utilisera à son tour.
Lorsque vous avez terminé votre coup, passez la télécommande au prochain joueur.

● Sélectionner un Mii
Lesjoueurspeuventafficher le Mii de leur choix durant le jeu. Vous pouvez aussi jouer sans sélectionner de Mii.

● Partie à plusieurs
Selon le tournoi que vous avez sélectionné, vous pouvez jouer contre 1 à 4 joueurs en jeu individuel ou jeu en équipe.

● Règles
Certains types de jeu vous permettent de paramétrer les règles. Parfois, "CueSports - Pool Revolution" utilise des règles spéciales.

● Coup en bande
Afin de déterminer qui entamera la partie, les joueurs doivent faire un coup en bande. Frappez la bande opposée depuis la mouche pour vous rapprocher le plus possible de la bande située derrière la position de départ. Le joueur le plus proche de la bande commence la partie.

● Conditions de victoire
Le vainqueur d'une partie est déterminé par le nombre de manches remportés ou les points obtenus. Vous pouvez paramétrer les conditions de victoire avant de commencer la partie.

● Ralenti
Vous pouvez sauvegarder les ralentis de vos jolis coups durant les parties et les regarder autant de fois que vous le voulez en mode ralenti. Vous pouvez sauvegarder jusqu'à 32 ralentis.
