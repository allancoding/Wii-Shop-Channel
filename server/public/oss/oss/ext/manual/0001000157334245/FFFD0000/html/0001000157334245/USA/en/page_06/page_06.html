Game Modes
 

The following modes are available:

 

  • CAREER
  • TOURNAMENT
  • LEVEL EDITOR

 

Career

Career mode is split over 10 Planet Zones, with 8 levels in each and one boss.
 

Tournament Mode

The Tournament mode allows you to create your own custom tournaments, made up from either the game levels or your created levels.

Level Editor

Fully featured editor for creating your own levels – up to 100 levels can be created and saved.

Options

Change the Sound FX & Music volumes, and view developer credits.
