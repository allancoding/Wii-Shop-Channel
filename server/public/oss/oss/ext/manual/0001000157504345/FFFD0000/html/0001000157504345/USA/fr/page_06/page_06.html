Commandes

Vous pouvez jouer en tenant la manette Wii Remote et le Nunchuk à la verticale ou en tenant la manette Wii remote à l’horizontale.

Manette Wii Remote et Nunchuk

Utilisez la manette Wii Remote et le Nunchuk pour donner des coups. Déplacez le Nunchuk pour que Little Mac donne un coup de gauche ou déplacez la manette Wii Remote pour un coup de droite.

Manette Wii Remote seule
Lorsque vous utilisez ce type de commandes, tenez la manette Wii Remote à deux mains.

Note : Si le curseur () est déplacé hors écran, vous pouvez utiliser pour naviguer dans les menus. Appuyez sur pour confirmer votre choix ou sur pour annuler.

Note : Vous pouvez passer du type de commandes « manette Wii Remote et Nunchuk » au type « manette Wii Remote seule » à tout moment durant votre partie.
