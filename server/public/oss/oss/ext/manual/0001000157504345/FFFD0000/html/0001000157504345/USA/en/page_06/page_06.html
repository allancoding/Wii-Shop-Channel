Controls

You can play holding the Wii Remote and Nunchuk controller vertically or by using the Wii Remote sideways.

Wii Remote and Nunchuk

Use the Wii Remote or Nunchuk in a punching motion to throw a punch. Move the Nunchuk to have Little Mac throw a left-handed punch or move the Wii Remote for a right-handed punch.

Wii Remote
Hold the Wii Remote with both hands when using this control style.
