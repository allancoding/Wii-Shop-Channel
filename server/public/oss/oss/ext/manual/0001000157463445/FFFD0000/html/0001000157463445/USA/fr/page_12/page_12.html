La mappemonde

● Se déplacer sur la mappemonde

Utilisez la pour déplacer votre personnage sur la mappemonde. Lorsque votre équipe compte plusieurs personnages, vous pouvez changer de meneur en appuyant sur .

● Parler / Chercher

Si vous souhaitez parler à quelqu’un ou examiner un objet, approchez-vous et appuyez sur .

● Les points de

  sauvegarde

Vous trouverez des points de sauvegarde dans les villes et dans les donjons. Sur ces points, vous pouvez sauvegarder votre progression (→ P. 10) et utiliser une tente ou un pavillon pour redonner des forces à votre équipe.

 

● Les auberges

Vous pouvez vous reposer dans les auberges moyennant un certain nombre de gils (la monnaie du jeu) : les PV et PM de tous vos personnages seront régénérés et toutes les altérations d’état seront guéries (→ P. 22). Le temps passe également lorsque vous restez à l’auberge : quand vous en sortirez, la lune sera dans une nouvelle phase. (→ P. 20).

 

● Magasins / Arsenaux /

  Armureries

Il existe trois types de magasins dans ce jeu : chacun vend différents types d’articles. Sélectionnez « Acheter » ou « Vendre » pour effectuer une transaction.

Fonctionnement des magasins

Sélectionnez l’article que vous souhaitez acheter (ou vendre) puis appuyez sur pour confirmer votre choix. Utilisez la pour choisir la quantité d’articles que vous souhaitez acheter (ou vendre) – appuyez sur gauche ou droite pour les unités et sur haut ou bas pour les dizaines – puis appuyez à nouveau sur pour confirmer. L’écran ci-dessous illustre l’achat d’une arme dans une armurerie ; l’écran sera légèrement différent dans d’autres magasins.

 

● Transport

Au fur et à mesure de votre progression, vous trouverez différents moyens de transport qui vous aideront à vous déplacer sur la mappemonde :

 

Aéronef

Appuyez sur pour monter à bord ou débarquer.
Chocobo

Parlez à un chocobo pour monter sur son dos ; appuyez sur pour descendre de son dos. Le chocobo s’enfuira dès que vous serez descendu.
