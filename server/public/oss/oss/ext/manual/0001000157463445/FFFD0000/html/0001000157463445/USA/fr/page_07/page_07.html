7
Déroulement de l’histoire

Ce jeu propose des histoires supplémentaires téléchargeables : ces nouveaux scénarios sont centrés sur les différents personnages qui apparaissent au cours de l’intrigue et éclairent d’autres aspects de l’histoire principale.

Pour jouer à ces différents scénarios, il vous faut les acheter à l’écran-titre dans « Acheter un contenu supplémentaire » (→ P. 24).
* Vous pouvez jouer à autant d’histoires supplémentaires que vous le souhaitez, dans l’ordre de votre choix.
* Pour jouer à l’histoire intitulée « Les cristaux », vous devez utiliser une sauvegarde dans laquelle vous avez terminé l’histoire principale et les histoires supplémentaires des personnages.

 

* L’icônedésigne les histoires en contenu supplémentaire que vous pouvez acheter si vous possédez l’histoire principale.
