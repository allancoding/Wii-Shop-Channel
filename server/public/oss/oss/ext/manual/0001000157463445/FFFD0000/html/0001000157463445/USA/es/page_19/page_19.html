"Settings" (Ajustes)

Aquí podrás configurar diversos aspectos del juego. Pulsa arriba o abajo en para seleccionar una opción, e izquierda o derecha para ajustar cada una de las configuraciones. Selecciona "Adjust Screen" (Ajustar pantalla) y pulsa para que se abra la ventana de ajustes correspondiente. Cuando hayas terminado, pulsa para regresar a la pantalla anterior.

 

"Battle Mode"

(Modo de combate)

Elige "Wait" (Esperar) para detener el transcurso del tiempo durante el combate, o bien "Active" (Activo) para que el tiempo transcurra con normalidad durante todas las partes del combate.

"Battle Speed"

(Velocidad de

combate)

Elige entre seis niveles distintos de velocidad para determinar lo rápido que pasará el tiempo durante los combates.

"Battle Messages"

(Mensajes de

combate)

 

Elige entre seis niveles distintos de velocidad para determinar cuánto tiempo se mostrarán los mensajes durante el combate.

"Window Color"

(Color de las

ventanas)

Cambia los colores de las ventanas ajustando los niveles de rojo, verde y azul.

"Adjust Screen"

(Ajustar pantalla)

Configura cómo se verá el juego en la pantalla de tu televisión.

"Language"

(Idioma)

Elige si los textos del juego se verán en inglés o francés.

"Controller Assignment"

(Asignación de mandos)

 

Cuando jueguen varios jugadores, podréis asignar a cada mando un personaje controlado en combate.

"Return to Title"

(Regresar al inicio)

Vuelve a la pantalla de inicio.
