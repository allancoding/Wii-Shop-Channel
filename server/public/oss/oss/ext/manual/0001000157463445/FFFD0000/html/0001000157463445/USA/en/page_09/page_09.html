The Title Screen

When you first begin the game, the "Adjust Screen" window will be shown once the game loads (→ P. 19). After this configuration is complete, the game will proceed to the title screen.

Press any button on the Wii Remote (except HOME) on the title screen to bring up the title menu. Move the cursor using and make selections by pressing .

● Adjust Screen

Change the "X / Y position" (left and right / up and down) and "Magnification" settings so that the four corners of the window displayed all fit inside the screen. Select "Done" once you're finished adjusting.
*The game may not display properly on your screen if these adjustments are not performed correctly.

 

● How to Play

Choosing "How to Play" from the title menu allows you to speak with the various people in the Training Classroom and learn the basics of the game along with assorted battle tips. It is recommended that you view them all before you start the game.
