RANKING MODE
RANKING MODE allows you to view the scores
achieved from playing the game.
Local Ranking

● Total score result for STORY MODE completion

● Top scores for each stage in ARCADE MODE

● Top scores for each stage in VERSUS MODE

Nintendo Wi-Fi Connection Ranking
(Accessed by pressing while the Ranking Screen is displayed.)

● The top score achieved by the player for each stage in ARCADE MODE and the nearest scores of 30 other players.

● The top score achieved by the player for each stage in VERSUS MODE and the nearest scores of 30 other players. While VERSUS MODE Ranking is displayed, you can switch between WORLDWIDE, REGIONAL, and FRIEND categories by pressing .

* In VERSUS MODE, you can only register with Nintendo Wi-Fi Connection Ranking when you achieve the top score.

* Registration to Nintendo Wi-Fi Connection Ranking
is voluntary.
