ARCADE MODE
Au lieu de compléter chaque niveau comme dans le
STORY MODE, vous devez essayer d'obtenir le meilleur score en ARCADE MODE.
Vous pouvez sélectionner 3 niveaux différents.
Lorsque vous n'avez plus de bille, la partie est finie.
Les meilleurs scores obtenus dans ce mode peuvent être enregistrés au classement connexion Wi-Fi Nintendo, ce qui vous permet de rivaliser avec les joueurs du monde entier. Vous pouvez accéder au classement connexion Wi-Fi Nintendo avec la à l'affichage de l'écran des résultats.
