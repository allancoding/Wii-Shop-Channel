Getting Started
Controllers
Only one Wii Remote can control the game at any time. The active controller is set at the attract screen, by pressing .

During gameplay, Bruiser and Scratch supports the Wii Remote in the horizontal position. The Nunchuk or Classic Controller may be connected to the active Wii Remote at any time. A diagram of the controls for each configuration can be seen on the large help button on the Puzzle Select screen.

Profiles
Three save game profiles are automatically created when the game starts. They are represented by distinctive doorways. By hovering the cursor over each door, you can compare the achievements of each profile player. Every few seconds a new statistic is shown. Click on a profile to advance to the Game Mode Selection screen.

To reset the statistics and progress of a single profile, place your cursor over the doorway you wish to delete, hold the button and press . A flushing sound will tell you the profile is now empty.
