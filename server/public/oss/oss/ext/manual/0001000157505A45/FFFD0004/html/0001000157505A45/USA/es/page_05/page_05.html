Controladores
Sólo un Wii Remote puede controlar el juego en cualquier momento. El Wii Remote activo se fija en la pantalla de atraer, en oprimiendo .

Durante el juego, Bruiser and Scratch apoya el Wii Remote en posición horizontal. El Nunchuk o Classic Controller puede ser conectado al Wii Remote activo en cualquier momento. Un diagrama de los controles para cada configuración se puede ver en el botón grande de ayuda en la pantalla de selección de ‘Puzzle Select’.

Perfiles
Tres perfiles de juego se crean automáticamente cuando comienza el juego. Están representados por distintas puertas. Flotando el cursor por encima de cada puerta, usted puede comparar los logros de cada jugador en su perfil. Cada cuantos segundos una nueva estadística se muestra. Seleccionar con el botón en un perfil para avanzar a la pantalla de selección Game Mode Selection.

Para restablecer las estadísticas y el progreso de un solo perfil, coloque el cursor sobre la puerta que desea borrar, mantenga presionado el y oprime el . Un sonido de lavado de agua le indicara que el perfil está vacío.
