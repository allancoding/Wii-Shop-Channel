Modalità


GAME START
Inizia a giocare. Se selezioni questa voce, verrà visualizzato il seguente menu.

NEW GAME

Inizia una nuova partita.


LOAD

Riprendi una partita precedentemente salvata.


TIME ATTACK

Gioca ogni livello nella modalità Time Attack.

I tempi verranno riportati negli elenchi classifiche.

Puoi salvare i dati del replay di Time Attack.


ENDLESS ATTACK

Verrà visualizzato dopo l'acquisto del contenuto scaricabile. Si tratta di un livello senza fine, in cui devi cercare di avanzare il più possibile. Le distanze verranno registrate nella classifica.


CHALLENGES

Livelli di gioco corti e con regole precise di completamento. Controlla la lista delle sfide da completare nel gioco principale.


RANKINGS

Visualizza i punteggi dei giocatori.


OPTIONS

Modifica le impostazioni di gioco.


ADD-ON CONTENT

Acquista contenuti aggiuntivi per MEGA MAN 10.
