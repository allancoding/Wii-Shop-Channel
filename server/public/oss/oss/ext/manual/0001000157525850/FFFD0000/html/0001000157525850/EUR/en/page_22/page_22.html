Game Over

You will lose one life every time your Energy Level reaches zero.

When you run out of lives, the game is over, and the following screen will be displayed.
CONTINUE
Re-start the game from the stage where your game ended.
STAGE SELECT

Go to the Stage Select screen.

SHOP
Go to the Shop screen.
SAVE

Go to the Save/Load screen. Save your progress in the game. You can have up to 8 Save slots.
EXIT GAME

You will return to the title screen without saving your game.
