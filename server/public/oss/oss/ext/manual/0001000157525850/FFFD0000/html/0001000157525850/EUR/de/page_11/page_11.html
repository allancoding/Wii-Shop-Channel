11
Spielmodi


GAME START
Das Spiel starten – diese Option wird das folgende Menü aufrufen.

NEW GAME

Ein neues Spiel starten.


LOAD

Ein zuvor gespeichertes Spiel fortsetzen.


TIME ATTACK

Spiele alle Abschnitte im Time Attack-Modus (Zeitangriff-Modus).

Zeiten werden in der Rangliste gespeichert. Du kannst deine Wiederholungsdaten für den Time Attack-Modus (Zeitangriff-Modus) speichern.


ENDLESS ATTACK

Wird nach dem Kauf über die zusätzlichen Inhalte angezeigt.

Ein Abschnitt, der kein Ende hat. Die Herausforderung liegt darin, so weit wie möglich zu kommen. Distanzen werden in den Rankings (der Rangliste) gespeichert.


CHALLENGES

Kurze Abschnitte mit eigenen Regeln zur Vollendung. Sieh dir die Liste der zu erfüllenden Challenges (Herausforderungen) im Hauptspiel an.


RANKINGS

Sieh dir die Spieler-Rankings (Ranglisten) an.


OPTIONS

Ändere die Spieleinstellungen.


ADD-ON CONTENT

Kaufe Add-On Content (zusätzliche Inhalte) für MEGA MAN 10.
