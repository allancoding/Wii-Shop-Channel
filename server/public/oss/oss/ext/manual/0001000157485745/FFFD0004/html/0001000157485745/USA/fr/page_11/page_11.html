Bonus

Les bonus apparaissent de façon aléatoire sur le terrain, dans une boîte mystère. Passez dessus pour les ramasser, appuyez sur le bouton A pour les utiliser.

 

Certains bonus ne sont utilisables que sur le moment, d'autres sont temporaires. Vous pouvez créer une file de deux bonus.

 

Carapace de tortue (15 secondes)

 

Votre équipe est insensible aux coups.

Survoltage (10 secondes)


Remplit votre Barre d'énergie, vous donnant un accès illimité aux coups nécessitant beaucoup d'énergie.

 

La Bombe (instantané)


Une énorme explosion renverse vos adversaires.

 

Taureau enragé (15 secondes)


Donne un bonus de force à votre équipe. Renversez vos adversaires en un coup !

 

Franc-tireur (15 secondes)


Augmente considérablement vos chances de marquer un panier.

 

Bouclier de panier (15 secondes)


La magie protège votre panier, repoussant les tirs adverses.

Ballon d'or


Vous rapporte 5 points de n'importe où. Attention - vos adversaires peuvent vous le voler. L'effet reste actif tant que personne n'a marqué.

 

Ballon fantôme


Soustrait 5 points au score de l'adversaire. Attention - vos adversaires peuvent vous le voler. L'effet reste actif tant que personne n'a marqué.
