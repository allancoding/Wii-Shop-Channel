Empezando
Pantalla de título

La pantalla de título aparecerá tras cargar el juego. Pulsa para empezar la partida.
Seleccionando un acta del juicio

Para empezar una partida, selecciona primero un archivo de acta del juicio.
Nueva partida y Continuar

Una vez hayas seleccionado el acta del juicio que desees usar, se mostrará el menú de juego.

Podrás entonces empezar la partida.
Nueva partida

Elige esta opción para jugar desde el inicio de un episodio ya terminado. Al empezar el juego solo estará disponible un episodio.
Continuar

Selecciona para continuar la partida desde el último punto de guardado, o desde el inicio del último capítulo en el que guardaste.
