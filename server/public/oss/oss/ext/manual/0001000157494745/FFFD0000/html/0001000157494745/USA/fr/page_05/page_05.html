Avant de commencer
Soyez bienvenus à Incoming (Arrivée), le jeu rapide de combat stratégique.  

 

Quand le jeu commence, vous pouvez sélectionner un jeu à un joueur ou à deux joueurs.

 

Dans un jeu d’un joueur créez ou sélectez votre profil et vous commencerez immédiatement le jeu.

 

Dans un jeu de deux joueur, vous pouvez choisir différents types de jeux à partir de chars seulement à un défi total; réglez la vitesse du jeu et commencez.

 

Bonne chance et amusez-vous !
