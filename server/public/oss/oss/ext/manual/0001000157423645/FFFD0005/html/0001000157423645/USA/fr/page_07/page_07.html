7
Quiz

A) Classic Quiz

Utilisez votre télécommande Wii pour choisir la bonne réponse parmi le choix de réponses. Pointez la réponse que vous croyez exacte avec votre télécommande Wii et appuyez sur le bouton A pour confirmer.
Vous pouvez sélectionner une autre réponse autant de fois que vous le voulez, mais plus vous répondrez rapidement, plus vous gagnerez d’argent.
Vous avez 15 secondes pour répondre à chaque question.
Répondez correctement à un maximum de questions dans chaque manche de 7 questions. Plus vous progresserez dans le quiz, plus vos gains augmenteront.


Astuce !

⇨ Répondez correctement le plus vite possible pour remporter plus d’argent que les autres candidats ayant aussi répondu correctement mais moins vite que vous.
 
B) Light Quiz

Utilisez votre télécommande Wii comme une lampe de poche pour éclairer les cases de réponses obscurcies. Déplacez le curseur sur les cases obscurcies pour révéler les réponses. Pointez la réponse que vous croyez exacte avec votre télécommande Wii et appuyez sur le bouton A pour confirmer.
Vous pouvez sélectionner une autre réponse autant de fois que vous le voulez, mais plus vous répondrez rapidement, plus vous gagnerez d’argent.
Vous avez 15 secondes pour répondre à chaque question.
Répondez correctement à un maximum de questions dans chaque manche de 7 questions. Plus vous progresserez dans le quiz, plus vos gains augmenteront.

Astuce !

⇨ Portez attention à toutes les réponses révélées en déplaçant la lampe de poche.
⇨ Vous n’êtes pas obligé d’éclairer complètement chaque case avant de choisir votre réponse.
 
C) Scratch Quiz

Utilisez votre télécommande Wii pour gratter la surface des réponses couvertes. Déplacez le curseur sur les cases de réponses pour révéler les réponses couvertes. Pointez la réponse que vous croyez exacte avec votre télécommande Wii et appuyez sur le bouton A pour confirmer.
Vous pouvez sélectionner une autre réponse autant de fois que vous le voulez, mais plus vous répondrez rapidement, plus vous gagnerez d’argent.
Vous avez 15 secondes pour répondre à chaque question.
Répondez correctement à un maximum de questions dans chaque manche de 7 questions. Plus vous progresserez dans le quiz, plus vos gains augmenteront.


Astuce !

⇨ Portez attention à toutes les réponses révélées en grattant la surface des cases couvertes.
⇨ Vous n’êtes pas obligé de gratter complètement chaque case avant de choisir votre réponse.
⇨ Vous pouvez aussi choisir une réponse grattée par un autre joueur.
