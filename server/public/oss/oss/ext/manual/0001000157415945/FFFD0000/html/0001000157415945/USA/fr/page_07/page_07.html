7
Des adieux déchirants

Comme vous êtes fait de papier, vous ne pouvez pas mourir, mais vous vous déchirez si vous heurtez le sol ou les objets trop vite ou si vous vous trouvez écrasé entre deux obstacles. Soyez prudent quand vous êtes dans un espace confiné, si vous faites tourner le monde, vous risquez également de vous faire écraser.

Ne vous inquiétez pas si vous vous déchirez en deux, car vous ne perdrez pas de vie et ne subissez aucune pénalité. Vous vous reconstituez un nombre illimité de fois.

Une chose à remarquer est que vous survivrez à une longue chute rapide si vous atterrissez sur une pente descendante, alors que vous vous déchirerez si vous tombez sur un sol plat. Servez-vous de cet effet en essayant de faire pivoter le monde pour atterrir sur une pente !

Pour mieux apprécier la vitesse de votre chute, observez la barre de vitesse qui est affichée (voir illustration) tant que vous faites pivoter le monde. Plus la barre est longue, plus vous tombez vite et plus vous avez de risques de vous déchirer en touchant le sol. La barre vous indique aussi dans quelle direction vous allez heurter le sol si vous cessez de faire tourner le monde.
