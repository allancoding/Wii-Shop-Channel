はじめに

「ソーサリーブレイド」とは？
ソーサリーブレイドは、RPG（ロールプレイングゲーム）と呼ばれるジャンルのゲームです。
移動シーンで情報を集めたり、戦いの際に使用する武器を入手したり強化し、戦闘シーンで敵を倒すことで経験を積み、主人公たちを成長させ強くしていきます。
これらを繰り返しながら、物語を進める内容となっています。
ソーサリーブレイドの世界は30世紀の火星。
この時代、火星には「カーネリア」という小国家が存在しています。
地球は複数の隕石の落下という大災害により、文明が崩壊してしまいました。
一方、火星に移住した移民の子孫（カーネリアン）は、ほぼ原始の時代からの出発であった開拓時代を乗り越え、技術を発展させ、今度は地球を目指すことができる宇宙船を作り上げるに至ります。
カーネリアは、我々が生きる現代社会がもう少し進歩した世界、近未来のこの地球を凝縮したような世界と言うこともできるでしょう。
