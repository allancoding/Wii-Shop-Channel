スーパーシュート
スーパーシュート


ダッシュ中やジャンプ中タイミング、キャッチのタイミングによって発動するシュートです。
4色の「すーぱーしゅーと」があります。


通常のシュートとは異なり、威力やスピードがアップして強烈爽快に相手をふっ飛ばします。
キャラクターのステータスによって使い分け、
緩急をつけることによって勝負を有利に進めることができます。


4色の「すーぱーしゅーと」は組み合わせて効果を倍増させることができます。

必殺シュート


「ひっさつしゅーと」は、ダッシュ中、ジャンプ中タイミングよくボタンを押すと発動するシュートです。


ボールが曲がったり消えたり、形が変わったりと特殊効果のある威力の高いシュートです。


発動するとボールが黄色に光ります。


キャラクターによってダッシュ、ジャンプ別に発動するシュートが設定され、多彩な種類があります。

ナイスシュート


「ないすしゅーと」は、ダッシュ中、ジャンプ中タイミングよくボタンを押すと発動するパワー5割増しの超破壊力シュートです。


さまざまな体勢から投げ出す種類があります。


発動するとボールが青色に光ります。

リターンシュート


「りたーんしゅーと」は、相手のシュートをタイミングよくキャッチすることで発動するシュートです。


至近距離での返り討ちシュートやダッシュ、ダッシュジャンプで相手に飛び込むキャッチカウンターとして、攻撃のパターンを広げることができます。


発動するとボールが緑色に光ります。


「りたーんしゅーと」に「ひっさつしゅーと」または「ないすしゅーと」を組み合わせるとダブル効果となり
「りたーんひっさつ」または「りたーんないす」に変化します。

アリウプシュート／レシーブシュート


「ありうぷしゅーと」は、バスケットボールのアリウプシュートのように、味方のパスをジャンプした状態でタイミングよく捕球することで発動するシュートです。


「だっしゅまん」による連携攻撃をより強力に使うことができます。


発動するとボールが赤色に光ります。


「ありうぷしゅーと」に「ひっさつしゅーと」または「ないすしゅーと」を組み合わせるとダブル効果となり
「ありうぷひっさつ」または「ありうぷないす」に変化します。


味方のパスをダッシュした状態でタイミングよく捕球すると
「れしーぶしゅーと」になります。
