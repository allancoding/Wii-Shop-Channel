Cómo se juega
El objetivo de cada nivel es conseguir introducir el máximo de Goo Balls en la tubería.

Agarra y suelta las Goo Balls para construir torres, zepelines, lenguas gigantes y demás estructuras montadas con cariño. Tienes que ir con cuidado porque tu manipulación de las Goo puede ser desastrosa si no construyes una base sólida de Goo y de aprendender.

Para pillar una Goo Ball, apúntala con el cursor y agárrala con . Puedes mover la Goo Ball hacia donde quieras y después soltarla. Para conectar la Goo Ball a la estructura, agárrala, acércala a la estructura y suéltala. La Goo Ball se une automáticamente a las demás. Así puedes construir desde una bonita estructura sencilla a una complicadísima.

Recuerda que es un mundo entero. Experimentar y descubrir forman parte del juego. Ni el juego ni el mundo se pueden estropear, así que prueba todo lo que quieras. Que te diviertas, querido amigo.
