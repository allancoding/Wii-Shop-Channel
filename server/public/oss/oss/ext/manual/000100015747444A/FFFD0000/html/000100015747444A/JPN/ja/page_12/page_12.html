OPTIONS
ゲームの難易度や残機数、画面表示の調整など、以下の設定を変更することができます。

●BGM VOLUME
　ゲーム中に使用されている曲の音量を
　設定できます。

●SE VOLUME
　効果音の音量を設定できます。

●BGM TEST
　ゲーム中に使用されている曲をテスト再生
　できます。

●SE TEST
　ゲーム中に使用されている効果音をテスト再生
　できます。

●DEFICULTY
　ゲームの難易度を設定できます。

●PLAYER STOCK
　ゲーム開始時の機体数を設定できます。

●EXTEND
　ゲーム中、何点で残機体が増えるかを
　設定できます。

●DEFAULT NAME
　ランキングの名前を設定を設定できます。

●SCREEN ADJUST
　画面の位置の変更などの設定ができます。

●BUTTON CONFIG
　ボタンの割り当ての設定ができます。

●DEFAULT SETTINGS
　オプションでの各設定を初期値に戻します。

●EXIT
　オプションモードを終了します。
