Controles
Solo se requiere el mando de Wii para jugar Flowerworks. El Nunchuk™ y el mando clásico™ no son soportados por el juego.

Controles para el crecimiento de las flores
Apunta el mando de Wii hacia la pantalla para mover a Follie. Ella se moverá hacia donde has apuntando. Colecta el polen tocándolo. Presiona el Botón A para lanzar el polen.

Debes mantener presionado el Botón A para seleccionar la flor más cercana. Mientras presionas el Botón A, Follie podrá moverse alrededor de la pantalla, sin perder su conexión con la flor. Rápidamente suelta y presiona el Botón A para lanzar el polen, esto sucede mientras mantienes activa tu conexión con la flor. Presiona el Botón B para cambiar tu conexión hacia la flor más cercana (esto es útil solo cuando aún se está presionando el Botón A). Después que Follie ha ganado la habilidad de fuegos artificiales, presiona hacia abajo en la cruz de control para lanzarlos. Presiona el Botón - o el Botón + para ingresar al menú.

Controles para Navegar el Mundo
Apunta el mando de Wii hacia la pantalla y presiona el Botón A para mover a Follie. Camina sobre las monedas o semillas para recolectarlas. Párate al lado de un gnomo para calmarlo. Párate al frente de un local para hablarle. La cruz de control puede ser usada para moverse. Presiona el Botón - o el Botón + para ingresar al menú y navegar por las páginas. Agita el mando de Wii cuando se indique para así calmar a un gnomo.
