7
Navigation du menu

Il suffit de pointer la Wii Remote vers le téléviseur pour naviguer dans le menu principal. Un faisceau de lumière apparaît et permet de découvrir et sélectionner les options du menu principal. Lorsque le faisceau illumine une option, celle-ci devient rouge. Le Bouton A permet de sélectionner cette option. Le Bouton B permet à tout moment d'annuler une sélection ou de retourner le joueur à l'écran du menu précédent.

 

La lampe n'a pas d'effet sur certaines parties des instructions qui apparaissent dans le menu principal. Ces instructions permettent de commencer une nouvelle partie ou de continuer une partie sauvegardée. Pour commencer une nouvelle partie, suivez les instructions en sélectionnant un des fichiers nommé NEW GAME (NOUVELLE PARTIE). Les parties sauvegardées peuvent être continuées en sélectionnant le fichier correspondant. Les progrès accomplis dans le jeu sont automatiquement sauvegardés et le dernier niveau atteint par le joueur est toujours chargé lors de la sélection d'un fichier.

 

Ce jeu offre des surprises à débloquer qui sont des récompenses gagnées après avoir terminé le jeu ou décroché tous les appels de Rachael. Le joueur peut choisir au début d’une nouvelle partie s’il veut ou non utiliser les surprises débloquées. Des instructions dans le menu principal expliquent comment faire les sélections pour chacune des surprises débloquées.

 

Pour supprimer un fichier, il suffit de sélectionner DELETE sous les trois fichiers sauvegardés, puis d'indiquer le fichier à supprimer. La suppression d'un fichier efface définitivement tous le progrès associé au fichier et le remplace par l’option de commencer une nouvelle partie.

 

Dans la partie supérieure gauche du menu principal s'affiche le mot BRIGHTNESS (LUMINOSITÉ). Cette option permet au joueur de régler la luminosité dans le jeu et de trouver le réglage idéal pour son écran. La luminosité augmente en appuyant sur le Bouton + de la Wii Remote, et baisse en appuyant sur le Bouton –. Le Bouton A permet de confirmer la luminosité et de retourner au menu principal. La luminosité est alors sauvegardée. Pour annuler le réglage de la luminosité et revenir au réglage par défaut, il suffit d'accéder au menu BRIGHTNESS et d'appuyer sur le Bouton B.

 

Dans la partie inférieure droite du menu principal se trouve un plan du lycée. Sélectionnez-le pour le voir en version plein-écran. Ce plan montre l'emplacement de toutes les salles de classe. Lorsque le joueur atteint une nouvelle salle de classe, son numéro ou son nom s’affiche sur le plan. Le joueur peut sélectionner n’importe quelle salle indiquée sur le plan pour rejouer le niveau. Le fait de rejouer un niveau n'a aucun effet sur le progrès sauvegardé et n’avance pas le joueur au niveau suivant une fois ce niveau terminé. Par contre, cela permet au joueur d’apprécier un défi déjà maîtrisé, ou d'exercer sa dextérité pour découvrir la manière la plus rapide d’achever le niveau. Après avoir débloqué le mode Obscurité, le joueur peut passer du Dark Mode (mode Obscurité) au Light Mode (mode Illuminé) grâce au bouton qui s'affiche en haut du plan du lycée. Les salles réussies en mode Obscurité doivent être ajoutées au plan indépendamment des salles réussies en mode Illuminé.

 

Dans la partie inférieure gauche du menu principal, le mot CREDITS est gravé sur le mur au-dessous du tableau. Cette option fait défiler le générique de développement du jeu. Il suffit d'appuyer sur le Bouton A pour retourner au menu principal.
