Modes de jeu

Dans LIT, le joueur dispose de trois modes de jeu :

 

Light Mode (mode Illuminé)
Principal mode de jeu. Le joueur guide Jake au travers de 25 niveaux d'énigmes et 5 niveaux avec boss. Dans ce mode, Rachael appelle Jake sur le système téléphonique du lycée et si le joueur peut répondre à tous les appels, il obtient la fin heureuse du jeu et une surprise à débloquer. Ce mode est disponible dès le début du jeu.

 

Dark Mode (mode Obscurité)
Ce mode de jeu est semblable au mode Illuminé et utilise les mêmes salles de classe, énigmes et boss. Toutefois, la lumière faiblit de plus en plus dans chaque salle et le joueur doit atteindre la sortie ou vaincre le boss avant qu'il ne fasse complètement noir, sinon, il faut recommencer le niveau. L'intensité lumineuse faiblit plus ou moins rapidement selon la vitesse à laquelle le niveau peut être accompli. En outre en mode Obscurité, le personnage de Jake est remplacé par une silhouette, ce qui rend la traversée de chaque salle encore plus difficile. Ce mode exige de la rapidité mais il n'y a pas d'appel téléphonique.

 

Le mode Obscurité est débloqué quand le joueur a battu le jeu une fois. Il peut alors choisir entre le mode Illuminé ou le mode Obscurité quand il commence une nouvelle partie.

 

Replay Mode (mode Répétition)
Le plan dans le coin inférieur droit du menu principal permet de rejouer n'importe quel niveau du jeu. Au fur et à mesure que les niveaux sont atteints en mode Illuminé, des numéros de salles s'affichent sur le plan. Ces numéros permettent de rejouer le niveau. Le mode Répétition permet de rejouer des niveaux précédemment gagnés ou de s'entraîner pour pouvoir réussir le niveau en mode Obscurité.

 

Ce mode est disponible dès le début du jeu, mais le plan n'affiche aucune salle tant que le niveau n'a pas été atteint en mode Illuminé ou Obscurité. Lorsque le mode Obscurité est débloqué, le joueur peut passer entre les plans du mode Illuminé et du mode Obscurité dans le coin supérieur droit du plan pour rejouer des niveaux précédemment atteints dans ces modes.
