Navigating the Menu

The Main Menu can be navigated by aiming the Wii Remote at the television screen. This causes a flashlight beam to appear, which is used to reveal and select options from the Main Menu. When an option is centered in the flashlight’s beam, it will turn red. When an option is red, pressing the A Button will select that option. Pressing the B Button at any time will cancel a selection, or back the player out to the previous menu screen.

 

Certain instructional text that appears at the Main Menu is not affected by the flashlight. This text should be followed in order to begin a new game, or continue an existing one. To begin a new game, follow these instructions by selecting one of the files labeled NEW GAME. Existing games can be continued by selecting their corresponding save file. Progress in this game is saved automatically, so selecting a save file will always load up the last level the player reached the last time they played the game under that save file.

 

This game offers unlockable extras, earned by completing the game or answering all of Rachael’s phone calls. These extras can be toggled on and off when beginning a new game after unlocking them. Instructional text at the Main Menu will inform the player how to make selections in each of these extras.

 

If a file needs to be deleted, this can be done by selecting the word DELETE underneath the three save files, then selecting the file that is to be deleted. Deleting a file permanently erases all progress associated with that file, and replaces it with the option to begin a new game.

 

At the top-left of the Main Menu is the word BRIGHTNESS. Selecting this option allows the player to adjust the brightness in-game to what is ideal for their television display. Brightness is increased by pressing the + Button on the Wii Remote, and decreased by pressing the – Button. Pressing the A Button confirms the adjusted brightness setting, and returns the player to the Main Menu. This brightness setting is saved from then on. To undo the adjusted brightness setting and return the game to its default setting, the player can access the brightness menu then press the B Button to return the game’s brightness to its default setting.

 

At the bottom-right of the Main Menu is a map. Selecting this map will bring up a full-screen map view. The map displays a layout of all rooms in the school. When a new classroom is reached, its number or label is added to the map. By selecting a labeled room from the map, the player can replay that level. Replaying a level does not affect save progress or advance the player to the next level upon completion. Replaying a level is useful to either enjoy a previous challenge, or to practice a previously-beaten level to discover the quickest way possible through it. After unlocking Dark Mode, a toggle between Dark and Light modes is added to the top of the map. Dark Mode rooms must be added to the maps independently from Light Mode rooms.

 

At the bottom-left of the Main Menu, etched on the wall below the chalkboard is the word CREDITS. Selecting this option plays the game’s development credits. Credits can be exited by pressing the A Button.
