Stratégie

La stratégie de LIT est simple : l'obscurité est dangereuse alors que les zones éclairées ne le sont pas. Si Jake s'aventure dans l'obscurité ou laisse l'obscurité gagner l'endroit où il se trouve, les créatures de l'ombre l'emportent et il faut rejouer le niveau. Pour allumer une source de lumière, il suffit de l'activer directement ou de lui lancer un projectile pour la déclencher. Si Jake se trouve près d'un objet qu'il peut utiliser ou ramasser, cet objet émet une lueur rouge. Il suffit alors d'appuyer sur le Bouton A pour l'utiliser ou le ramasser.

 

Il est possible d'allumer et d'éteindre les sources de lumière tout au long du jeu. La plupart des objets lumineux remplissent le compteur électrique et doivent être utilisés avec parcimonie pour pouvoir passer les niveaux plus difficiles. Les fenêtres sont la seule exception ; elles ne modifient pas le niveau du compteur et, une fois brisées, elles ne peuvent pas être refermées. Jake ne peut pas toujours atteindre les objets lumineux qui doivent être éteints mais il peut les détruire à distance avec le lance-pierre ou un pétard. Cette action permet de réduire le niveau du compteur mais les sources ainsi détruites ne pourront plus jamais servir.

 

Jake commence chaque niveau à la porte d'entrée de la salle de classe dans un petit carré de lumière à partir duquel il peut toujours atteindre une source de lumière ou un objet qui peut avoir un effet sur une source de lumière. Si le joueur doit recommencer un niveau, il commence toujours à partir de cet endroit.

 

Dans l'image suivante, Jake peut suivre un rayon de lumière provenant d'une fenêtre brisée mais une table l'empêche de suivre le chemin illuminé. Il se trouve près d'une lampe qui émet une lueur rouge, il peut donc l'utiliser en appuyant sur le Bouton A.

 

 

En appuyant sur le Bouton A, la lampe s'allume et crée suffisamment de lumière pour que Jake puisse contourner la table et suivre le rayon de lumière de la fenêtre.

 

 

Lorsque Jake a passé cet obstacle, la lampe ne sert plus à rien sauf s'il doit revenir pour une raison quelconque. Il peut réduire le compteur électrique en éteignant cet objet une fois qu'il est passé de l'autre côté de la table.

 

 

Si le compteur électrique est plein, il se met à clignoter pour prévenir le joueur que si une autre source de lumière est allumée, il éclatera. Chaque objet allumé remplit plus ou moins le compteur. La liste ci-dessous est basée sur un compteur électrique de 100 unités :

 

Fenêtre : aucune

Lampe : 20 unités

Écran d'ordinateur : 10 unités (par écran)

Lumière clignotante : 20 unités

Télévision : 25 unités

Détecteur de mouvement : 30 unités

 

Dans les niveaux avec boss, Jake peut non seulement être attrapé par les créatures de l'ombre mais il peut également être assommé si un boss le frappe ou lui lance un projectile. Lorsque Jake est assommé, il tombe et le joueur doit recommencer le niveau. Le jeu ne contient pas de compteur de vie et il suffit que Jake soit frappé une seule fois pour qu'il s'effondre.
