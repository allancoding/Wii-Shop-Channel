The Basics

LIT is a dark horror puzzle game that uses both the Wii Remote and the Nunchuk attachment. The player must guide Jake, the teenage hero of the game, across each dimly-lit classroom from the entrance door to the exit door, in order to move on to the next room. Each room begins unlit, and stepping into the darkness will cause Jake to get pulled into the shadows by unseen creatures. To reach the exit door, light sources must be discovered and turned on in order to create bridges of light that Jake can safely walk across. Light sources are turned on by accessing them directly, by firing a projectile at them, or by walking past them in the case of the motion sensor.

 

Most light sources are electrically-powered, so using them consumes electricity. A Light Meter at the top of the screen displays how much electricity is currently being used. If this meter is over-filled, the Light Meter will burst, along with all lights in the room, causing the current level to restart.

 

Once the exit door is reached and accessed, the player is taken to the next classroom in the game. Game progress is saved automatically upon exiting each room. After every 5 puzzle classrooms, Jake must fight a boss character. These levels are completed by defeating the boss creature with the available light sources, and Jake is not required to reach an exit door in these levels.

 

Jake’s girlfriend, Rachael, is also trapped in the school and occasionally calls Jake on the school’s phone system. Whether the player is able to answer all of Rachael’s phone calls through the game affects the game’s ending and unlockables.
