6
Disposition de l'écran

Voici une capture d'écran du jeu avec des explications sur tout ce qui apparaît dans la perspective informatique :

 

 

1 - Compteur électrique
Ce compteur se remplit en fonction de la quantité de sources de lumière électrique allumées dans le niveau actuel. Si ce compteur est trop plein, il éclate avec toutes les sources de lumière et le joueur doit recommencer le niveau. Le compteur peut être vidé en éteignant les sources de lumière électrique qui ne sont plus nécessaires pour que Jake avance.

 

2 - Objet actuel
Cette illustration indique l'objet que Jake peut utiliser en appuyant sur le Bouton A. Par défaut, l'illustration affiche l'objet actuellement sélectionné, mais quand Jake se trouve à côté d'une source de lumière ou d'un autre objet interactif, cet objet-là remplace l'objet actuellement sélectionné. Le joueur doit éloigner Jake des objets interactifs et des sources de lumière pour pouvoir utiliser l'objet sélectionné. Cette illustration indique également combien Jake possède d'objets identiques à celui sélectionné. Dans l'illustration ci-dessus, le joueur possède deux pierres. Si la réserve d'un objet est épuisée, il est impossible d'utiliser cet objet tant qu'elle n'a pas été renouvelée. Lorsque Jake ne possède plus d'objets supplémentaires, la lampe est sélectionnée par défaut.

 

3 - Animation pour remuer la Wii Remote
Lorsque le joueur doit éffectuer un movement avec la Wii Remote, un dessin animé s'affichera pour lui montrer comment l'effectuer.

 

4 - Texte indiquant le ramassage d'un objet
Lorsque Jake ramasse un objet, un texte s'affiche ici brièvement pour expliquer le type d'objet ramassé.

 

5 - Porte de sortie
La porte de sortie de chaque salle de classe est signalée par sa lueur rouge. Lorsque Jake est suffisamment près de la porte de sortie pour l'ouvrir, la lueur s'agrandit. La sortie se trouve toujours sur le mur du fond de la salle. Les salles où se trouvent les boss ne contiennent pas de sortie.

 

6 - Réticule
Le réticule permet de viser et de diriger l'orientation de Jake. Le réticule change de forme lorsque le joueur vise avec un projectile afin de mieux atteindre la cible.

 

7 - Étincelle
Tout objet à ramasser pris dans la lumière se met à étinceler afin d'indiquer au joueur qu'il peut le ramasser dès que Jake peut l'atteindre.

 

8 - Jake
Le héros de LIT. Il est toujours à l'écran.

 

9 - Bouton A
Le fait d'appuyer sur le Bouton A peut avoir un effet sur un objet proche, une illustration du Bouton A apparaîtra pour enforcer cette idée.
