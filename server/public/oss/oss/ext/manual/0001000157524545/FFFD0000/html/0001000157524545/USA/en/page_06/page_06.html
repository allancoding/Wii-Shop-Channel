Controls

These are the controls for the game Racers' Islands - Crazy Arenas.
This game must be played with the Wii Remote and the Nunchuk.

Wii Remote controls

Pointer : Target the vehicles, and wave the Wii Remote to load the weapon.

Shoot.

Use a bonus.

Pause the game.

Nunchuk controls

Turn.

Brake, reverse.

Accelerate.
