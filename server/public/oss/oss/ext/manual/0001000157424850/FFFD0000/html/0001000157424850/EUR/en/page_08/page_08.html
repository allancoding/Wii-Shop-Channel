8
Options
  • Reset to Defaults
    The default control type is the Wii Remote in the vertical position.
  • Control Type

    Wii Remote: The Wii Remote is aimed at the screen and moving the Wii Remote forward and back will Zoom-in and Zoom-out.

    Nunchuk style: This control type uses the Control Stick to control the zoom.

    Classic style: This game can be played with the Classic Controller. The Classic Controller Pro is also supported.

    Nintendo GameCube Controller: This game can be played with the Nintendo GameCube Controller.
  • Aim with

    Select an alternate control for aiming. This option is not available with the default Wii Remote in the vertical position.

  • Zoom with

    Select an alternate control for zooming.

  • Aim Grenade with

    Select an alternate control for changing the Grenade angle.

  • Aiming Sensitivity

    Aiming Sensitivity adjusts the range of motion required to move the aim with the Pointer.

  • Zooming Sensitivity

    Zooming Sensitivity adjusts the range of motion required to zoom the view.

  • Control Sensitivity

    Control Sensitivity adjusts the range of motion required to move the aim with the Control Stick.

  • Control Acceleration

    Control Acceleration adjusts the response speed of aiming.

  • Invert Y Axis

    Flip the Up / Down for the aiming controls.

  • Sound Volume

    Adjust the volume of the sounds in the game.

  • Music Volume

    Adjust the volume of the music in the game.
