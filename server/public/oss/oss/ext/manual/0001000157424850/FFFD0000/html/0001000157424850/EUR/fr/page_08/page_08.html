8
Options
  • Réinitialisez les paramètres

    Le type de contrôle par défaut est la télécommande Wii tenue à la verticale.

  • Type de contrôle

    Télécommande Wii. C'est le contrôle par défaut avec la télécommande Wii.

    Style Nunchuk. Ce type de contrôle utilise le Stick directionnel pour Zoomer / Dézoomer.

    Style classique. Pour ce jeu, on peut utiliser la manette classique. La manette classique pro est également compatible.

    Manette Nintendo GameCube. Pour ce jeu, on peut utiliser la manette Nintendo GameCube.
  • Viser avec

    Choisis un autre type de contrôle pour viser. Cette option n'est pas disponible avec le contrôle par défaut utilisant la télécommande Wii tenue à la verticale.

  • Zoomer avec

    Choisis un autre type de contrôle pour zoomer.

  • Visée grenade avec

    Choisis un autre type de contrôle pour changer l'angle de la grenade.

  • Précision de la visée

    La précision de la visée ajuste les mouvements requis pour viser avec le pointeur.

  • Précision du zoom

    La précision du zoom ajuste les mouvements requis pour zoomer.

  • Précision du contrôle

    La précision du contrôle ajuste les mouvements requis pour viser avec le Stick directionnel.

  • Accélération contrôle

    L'accélération du contrôle ajuste la vitesse de réaction du viseur.

  • Inverser axe Y

    Intervertit Haut / Bas pour les commandes de visée.

  • Volume effets sonores

    Ajuste le volume des sons dans le jeu.

  • Volume musique

    Ajuste le volume de la musique dans le jeu.
