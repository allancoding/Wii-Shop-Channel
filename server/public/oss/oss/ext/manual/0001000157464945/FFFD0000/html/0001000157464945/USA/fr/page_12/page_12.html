Laisser Mordre L'hameçon

Lorsque les poissons seront attirés par votre leurre, ils commenceront à le mordiller. La Wii Remote se mettra alors à vibrer. N'essayez pas de prendre un poisson à ce moment-là : attendez jusqu'à ce qu'il morde bien le leurre. Lorsque le poisson l'aura bien mordu, vous verrez qu'il le mord fermement et la Wii Remote vibrera plus intensément. C'est à ce moment-là que vous voudrez le prendre! Dès que le poisson aura mordu le leurre, soulevez la Wii Remote rapidement pour le prendre! Notes à propos de la prise :

* Si vous tentez une prise trop tôt, vous ferez peur aux poissons.

* Si　vous tentez une　prise trop tard, le poisson pourra               se  déprendre de l'hameçon facilement.

* Le temps de prise est différent pour chaque type de poisson. Ça    peut être compliqué au début, mais c'est en vous pratiquant que    vous deviendrez expert pêcheur!
