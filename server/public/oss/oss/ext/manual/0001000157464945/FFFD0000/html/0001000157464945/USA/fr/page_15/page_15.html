Mode Défi

Le but du mode Défi est de réussir une série de missions. Le but de chaque mission est d'attraper le poisson désigné dans le temps alloué. Si vous ne réussissez pas la mission dans le temps alloué, vous aurez échoué. Lorsque vous échouez, vous aurez la chance de réessayer ou de retourner à l'écran d'introduction. Si vous avez des données enregistrées de « Reel Fishing : Angler's Dream, » vous déverrouillerez des missions en prime qui seront disponibles dès que les missions normales auront été complétées.

●Menu de Pêche
Appuyez sur le bouton - pour afficher le menu de pêche. Vous verrez les options suivantes :
Changer de matériel de pêche : Changer de grosseur de ligneet de leurre.
Comment jouer : Afficher des informations sur les commandes de jeu.
Mission : Voir les poissons qui vous reste à attraper afin de réussir la mission en cours.
Réglages : Changer les réglages du jeu.
Retour à l'écran d'introduction : Quitter et retourner à l'écran d'introduction.
Nota : Vous ne pouvez pas accéder au menu de pêche alors que votre ligne est à l'eau.

● Réessayer
Si vous échouez une mission, vous aurez la chance de réessayer. Sélectionnez « Oui » si vous voulez réessayer. Sélectionnez « Non » pour voir les résultats et pour retourner à l'écran d'introduction.
