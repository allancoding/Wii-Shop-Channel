Objets
Dans le décor se trouvent des objets qui aideront ou ralentiront le joueur.

Pièges mortels
Pics: La balle explose si elle touche les pics (sauf la balle en acier).
Bain d’acide: La balle se détruit si elle touche l’acide.

Objets du décor
Canons: Les canons envoient la balle dans la direction dans laquelle ils sont tournés. Pour utiliser un canon, il suffit de s’en approcher. Tous les canons ont la même puissance.
Trampolines: Font rebondir les balles qui tombent sur eux.
Matelas: Permettent d’amortir la chute d’une balle.
Ventilateurs: Permettent de faire voler les balles légères.
Téléporteurs: Cet objet permet de téléporter une balle d’un point vers un autre. Tous les téléporteurs se ressemblent. Ils fonctionnent par paire. Pour utiliser un téléporteur, il suffit de s’en approcher. Ils fonctionnent dans les deux sens. Pour utiliser un téléporteur qui vient d’être utilisé, il faut s’en éloigner un peu et revenir.

Faux murs: Ils ressemblent à de vrais murs mais les balles passent à travers (une sorte de passage secret).
Bonus: Ajoute des points ou du temps.
