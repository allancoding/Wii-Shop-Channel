Contrôles
Le jeu se joue avec la télécommande Wii tenue horizontalement ou avec la Wii Balance Board™.

Télécommande Wii
Le joueur penche la télécommande Wii pour pencher le décor.
Le bouton 2 est utilisé pour valider les choix dans les menus. Le bouton 1 permet de revenir au menu précédent.
La télécommande Wii peut être tournée de 90° dans les deux directions.

Wii Balance Board
Le joueur se penche à droite ou à gauche pour pencher le décor.
Le bouton 2 de la télécommande Wii est utilisé pour valider les choix dans les menus. Le bouton 1 permet de revenir au menu précèdent.

Quand le joueur sélectionne la Wii Balance Board, le jeu vérifie qu’elle fonctionne correctement. Si un message d’erreur vous informe que la Wii Balance Board ne fonctionne pas correctement, visitez le site support.nintendo.com.
