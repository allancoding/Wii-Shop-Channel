Modes de jeux
Conquête
C’est le mode principal où le joueur doit terminer les 64 niveaux.
Les niveaux sont joués dans l’ordre. Dans ce mode, une carte montre la progression du joueur. Le joueur peut choisir de passer un niveau qui sera rejoué plus tard dans la séquence.

Niveau aléatoire
Jouer un niveau tiré au hasard (partie rapide).

Challenge
Le mode Challenge est un mode ou le joueur doit jouer 5 niveaux (tirés au hasard). Chaque niveau a des règles spéciales :
- Temps limité
- Nombre de vies limité
- Faire la course avec un adversaire

Jeu à plusieurs : Mode Rallie
Les joueurs doivent rapidement terminer 5 niveaux (tirés au hasard). Chaque joueur peut aller à son propre rythme. Quand un joueur a terminé un niveau, il passe au niveau suivant sans attendre les autres. Le vainqueur est le premier joueur à atteindre la dernière sortie.

Jeu à plusieurs : Mode Match
Les joueurs doivent terminer 5 niveaux (tirés au hasard). Quand tous les joueurs ont terminé un niveau, ils reçoivent des points. Ensuite la partie reprend sur le niveau suivant. Le vainqueur est celui qui a le plus de points au classement général.
