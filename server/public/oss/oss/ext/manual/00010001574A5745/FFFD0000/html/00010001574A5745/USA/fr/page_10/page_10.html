Modes de jeu

Bejeweled 2 comprend quatre manières différentes de jouer. Que vous cherchiez à améliorer vos talents en assortiment avec le mode Action ou vous relaxer avec le mode Infini, nous avons ce qu'il vous faut.

Classique : le mode Classique est la super sensation de puzzles que Bejeweled 2 devrait être. Faites des assortiments de 3, 4 ou 5 gemmes pour progresser dans les niveaux. Le jeu se termine quand il n'y a plus aucun assortiment de disponible.

Action : faites une course contre la montre dans ce mode chronométré de Bejeweled 2. Le temps se réduit quand vous progressez dans les niveaux plus élevés... créez donc vos assortiments au plus vite. Le jeu se termine à la fin de votre chrono.

Infini : relaxez-vous et faites des assortiments dans ce mode de jeu zen. Il n'y a aucun danger de perdre alors prenez votre temps et faites autant d'assortiments que vous le voulez.

Puzzle : utilisez votre connaissance de Bejeweled 2 pour résoudre plus de 70 puzzles différents.

CONSEIL : Utilisez les conseils à petite dose ! Les conseils réduisent votre score en mode Classique et Puzzle et annulent la progression de niveau en mode Action.
