Démarrer

Comment jouer

Dirigez les avions vers les zones d'atterrissage de même couleur qui leur sont attribuées à l'aide de votre télécommande Wii. Il vous suffit de pointer votre télécommande Wii vers l'écran, de sélectionner un avion avec le curseur et de maintenir le bouton A enfoncé pour tirer un trait jusqu'à une piste d'atterrissage.

L'avion deviendra blanc en entamant l'atterrissage.

Des avertissements de proximité vous indiquent que vous devez rediriger les avions pour éviter d'éventuelles collisions.

Si des avions entrent en collision, la partie est terminée. Une personne peut jouer seule, ou deux à quatre personnes peuvent se répartir les tâches d'un contrôleur aérien.

Appuyez sur le bouton - pour activer ou désactiver l'avance rapide. Ce mode se coupera automatiquement et le jeu reprendra une vitesse normale si vous recevez un avertissement de collision. Pour activer l'avance rapide sans arrêt automatique, appuyez deux fois sur le bouton -.

Faire atterrir un avion

Attention aux différentes vitesses des avions. Utilisez la télécommande Wii pour redessiner les trajectoires d'atterrissage des avions autant de fois que vous le désirez.

Commandes

Ce jeu nécessite la télécommande Wii. Les boutons peuvent être utilisés dans les menus.
