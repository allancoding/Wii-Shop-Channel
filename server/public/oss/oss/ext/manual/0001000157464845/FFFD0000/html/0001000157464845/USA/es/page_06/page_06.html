Introducción

Cómo jugar

Dirige el avión a las zonas de aterrizaje del mismo color con tu mando de Wii. Simplemente apunta a la pantalla con el mando de Wii, selecciona un avión con el cursor y mantén pulsado el Botón A para dibujar una ruta de vuelo hasta una pista de aterrizaje.

Los aviones se pondrán de color blanco cuando se aproximen para aterrizar.

Mantente alerta a las advertencias de proximidad y redirige los aviones para evitar colisiones.

Si choca algún avión se acaba el juego. Puede jugar una persona sola o hasta cuatro jugadores en colaboración.

Pulsa el Botón - para activar y desactivar el avance rápido. Si se activa una alerta de colisión, se detendrá automáticamente el avance rápido y la velocidad del juego volverá a la normalidad. Para activar el avance rápido sin detención automática, pulsa dos veces el Botón - rápidamente.

Aterrizar

Vigila las diferentes velocidades del avión. Usa el mando de Wii para trazar rutas de vuelo tantas veces como quieras.

Controles

Este juego usa el mando de Wii. Los botones se pueden usar en los menús.
