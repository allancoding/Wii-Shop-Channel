Moulinet

Quand un leurre est à l’eau, vous devez utiliser le moulinet pour remonter la ligne, tout d’abord pour attraper le poisson puis pour le ferrer lorsqu’il mordra.

  • Motion Controls (Contrôles de mouvement): Déplace le Nunchuk en mouvement circulaire. Plus le mouvement est rapide, plus le moulinet est rapide.
  • Standard Controls (Contrôles standard): Appuyez sur pour un moulinet lent, pour un moulinet moyen, ou + pour un moulinet rapide.

Une vitesse modérée de moulinet produira la meilleure action sur le leurre et fera le meilleur travail pour attirer les poissons.

En plus d’un jeu de moulinet constant, vous peut également balancer la canne à pêche pour tirer le leurre avec des mouvements brefs et saccadés lorsque la ligne est tendue.
