Lancer

Une fois que vous avez sélectionné un leurre, il est temps de lancer.

Tout d’abord, vous devez démarrer le Casting Meter (Mètre de coulée), puis déclencher l'action de lancer quand le mètre a la puissance désirée. Un timing précis est nécessaire pour les lancers longs.

  • Motion Controls (Contrôles de mouvement): Balancer la Wii Remote vers le haut pour lancer le mètre, puis balancer la Wii Remote vers le bas pour lancer.
  • Standard Controls (Contrôles standard): Appuyez et maintenez pour lancer le mètre puis appuyez sur pour lancer.

Une fois que le leurre est à l’eau, vous pouvez utiliser pour regarder aux alentours. Il s'agit d'une technique précieuse pour trouver les poissons qui sont au moins à quelques-uns pieds de votre leurre.
