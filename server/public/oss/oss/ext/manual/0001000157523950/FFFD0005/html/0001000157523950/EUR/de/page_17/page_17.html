17
Spezialwaffen
Wenn Sie einen Boss besiegt haben, können Sie dessen Waffe benutzen. Wenn Sie eine Spezialwaffe benutzen, senkt das die Waffenenergie dieser Waffe.
Magma Bazooka
Feuert Magma-Kugeln in drei Richtungen. Wird diese Waffe aufgeladen, erhöht das deren Angriffkraft.
Laser Trident
Feuert einen Laser, der Objekte durchbohren kann.
Black Hole Bomb

Drücken Sie den Schießen-Knopf, um die Bombe zu werfen. Drücken Sie ihn erneut, um sie zur Explosion zu bringen und ein schwarzes Loch zu erschaffen. Das schwarze Loch wird Projektilattacken und Gegner aufsaugen. Drücken Sie auf dem Steuerkreuz nach oben oder unten, um den Weg der Bombe zu beeinflussen, bevor sie explodiert.

Hornet Chaser

Feuert mechanisierte Hornissen auf ein Ziel ab.
Concrete Shot
Feuert einen Schwall Beton. Sie können diesen Beton als Tritthilfe nutzen.
Jewel Satellite

Ein Diamant wird um den Spieler kreisen und ihn so beschützen. Drücken Sie den Schießen-Knopf erneut, um den Diamant wegzuschleudern.
Plug Ball
Feuert einen Plasmaball, der sich über jede Oberfläche bewegt.
Tornado Blow
Feuert einen Tornado, der Gegner zurückschleudert.
Rush Coil
Springen Sie auf Rushs Rücken, um höhergelegene Plätze zu erreichen. Vom Beginn des Spiels an verfügbar.
Rush Jet

Springen Sie auf Rushs Rücken und fliegen Sie durch die Luft. Ist im Laufe des Spiels verfügbar.
