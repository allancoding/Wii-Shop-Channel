Spielmodi

Die folgenden Modi können im Spielmenü ausgewählt werden.

SPIELSTART
Das Spiel starten - diese Option wird das folgende Menü aufrufen.
NEUES SPIEL

Ein neues Spiel starten.

LADEN

Ein zuvor gespeichertes Spiel fortsetzen.

ZEITANGRIFF
Spielen Sie alle Abschnitte im Zeitangriff-Modus. Zeiten werden in der Rangliste gespeichert.
ENDLOSANGRIFF

Wird nach dem Kauf über die herunterladbaren Inhalte angezeigt. Ein Abschnitt, der kein Ende hat. Die Herausforderung liegt darin, so weit wie möglich zu kommen. Distanzen werden in der Rangliste gespeichert.

SPEZIALSPIELE

Wird nach dem Kauf über die herunterladbaren Inhalte angezeigt. Eine Auswahl einzigartiger und spaßiger Spiele. Auf der Seite für herunterladbare Inhalte gibt es mehr Informationen.

CHALLENGE-LISTE
Regeln für die Challenges.
RANGLISTEN
Zeigt die Spieler-Ranglisten.
OPTIONEN
Ändern Sie die Spieleinstellungen.
HERUNTERLADBARE INHALTE
Kaufen Sie zusätzliche Inhalte für MEGA MAN 9.
