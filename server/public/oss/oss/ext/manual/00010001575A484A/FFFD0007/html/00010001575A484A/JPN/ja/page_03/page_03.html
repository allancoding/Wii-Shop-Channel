インターネット接続時のご注意
追加コンテンツの購入に関するご注意
追加コンテンツの購入に関しては、以下の点に注意してください。

● 追加コンテンツは、任天堂を通じて購入していただけます。

● 追加コンテンツを購入するには、別途Wiiポイントが必要です（※有料）。

● Wiiポイントの購入方法は、Wiiショッピングチャンネル、または任天堂ホームページをご覧ください。（ニンテンドーポイント プリペイド番号をWii本体に登録すると、Wiiポイントとして扱われます。）

● Wiiショッピングチャンネルの「ご利用記録」では、追加コンテンツを購入したソフトの名前や購入日、使用したWiiポイントを確認することができます。

● 保有しているWiiポイントは、Wiiショッピングチャンネルで確認することができます。

● Wiiポイントが不足している場合、追加コンテンツを購入することはできません。Wiiショッピングチャンネル内でWiiポイントを追加してください。

● 購入された追加コンテンツは、Wii本体保存メモリに保存されます。Wii本体保存メモリが不足している場合は、購入することはできません。Wii本体に保存されているデータを削除するか、SDメモリーカードに移動してください。詳しくは、Wii本体取扱説明書 機能編「Wiiオプション（データ管理）」をご覧ください。

● 購入された追加コンテンツの返品・返金・交換はできません。

● 一度購入した追加コンテンツを削除しても、無料で再受信することができます。なお、諸般の事情により、追加コンテンツの提供を予告なく中止または終了する場合があります。その際、再受信もできなくなることがあります。あらかじめご了承ください。

● 追加コンテンツのデータをコピーしたSDメモリーカードを他のWii本体に差し込んでも使用できません。
