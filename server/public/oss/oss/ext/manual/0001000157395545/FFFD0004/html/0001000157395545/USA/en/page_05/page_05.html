Welcome on board!

Just for you! Four fabulous games to play based on gambling:

  • Slot Machine
  • Roulette
  • Black Jack
  • Video Poker

Once on the deck in the game you will be able to choose:

  • Competition mode
  • Quick Play mode
  • Top Scores
  • Credits

Competition Mode

    Select the number of players and play through all four games in order to win the "Champion Trophy".


Quick Play

    After selecting the number of players, you will then be able to select any one of the four games.


Top Scores

    If you beat a record, you will be able to register your name in the high score table.
