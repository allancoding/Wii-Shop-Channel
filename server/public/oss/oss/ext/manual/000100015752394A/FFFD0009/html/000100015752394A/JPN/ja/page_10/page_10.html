各モードの説明

ゲームメニューでは、次のようなモードを選ぶことができます。
GAME START
ゲームを開始します。これを選択すると続けて下記のメニューが出ます。
NEW GAME

新しくゲーム本編を開始します。
LOAD

データをロードしてセーブしたところからゲームを再開します。
TIME ATTACK
クリアしたステージのタイムアタックができます。記録はランキングに登録されます。
ENDLESS ATTACK

追加コンテンツを購入すると表示されます。無限に続くステージをどこまで進めるか挑戦します。記録はランキングに登録されます。
SPECIAL GAMES

追加コンテンツを購入すると表示されます。様々なゲームが楽しめます。詳しくは追加コンテンツのページを参照して下さい。
CHALLENGES LIST
チャレンジの達成状況を確認できます。
RANKINGS
ランキングを見ることができます。
OPTIONS
オプション設定を変更できます。
DOWNLOADABLE CONTENT
追加コンテンツを購入することができます。
