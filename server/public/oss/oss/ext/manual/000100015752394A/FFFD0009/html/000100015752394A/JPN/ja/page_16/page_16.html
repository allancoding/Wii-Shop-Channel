特殊武器
ボスを倒すとボスの武器を入手できます。これらの特殊武器を使うと、武器エネルギーを消費します。
マグマバズーカ
３方向にマグマ弾を発射します。チャージする事で、より強力なマグマ弾が発射できます。
レーザートライデント
貫通力のあるレーザーを発射します。
ブラックホールボム

発射後にもう一度ボタンを押すと爆発し、ブラックホールが発生します。ブラックホールは弾や敵を吸い込みます。爆発前ならボタンの上下で軌道を操作できます。
ホーネットチェイサー
追尾性能のあるハチ型のメカを発射します。
コンクリートショット
コンクリートのかたまりを発射します。コンクリートは足場にすることもできます。
ジュエルサテライト
ダイヤモンドが回転して身を守ります。もう一度ボタンを押すと前方に発射されます。
プラグボール
地面に沿って進むプラズマ球を発射します。
トルネードブロー
竜巻を起こして敵を吹き飛ばします。
ラッシュコイル
上に乗ると大きなジャンプができます。最初から所持しています。
ラッシュジェット

上に乗って空中を移動することができます。ゲーム中に入手します。
