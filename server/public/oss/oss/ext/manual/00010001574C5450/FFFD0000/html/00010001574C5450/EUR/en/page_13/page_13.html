Game Modes

There are three game modes available to the player in LIT:

 

Light Mode
The primary game mode has the player guiding Jake through 25 puzzle levels and 5 boss levels. Rachael calls Jake on the school’s phone system in this mode, and if the player is able to answer every phone call, they will receive the game’s Good Ending and an extra unlockable. Light Mode is available from the start of the game.

 

Dark Mode
This game mode works similarly to Light Mode, and uses the same rooms, puzzles, and bosses. In Dark Mode, however, each room is constantly dimming and the player must reach the exit door or defeat the boss before everything goes completely black. Once everything goes completely black, the level is restarted. Each level’s rate of dimming is based on how quickly that level can be completed. Additionally, Jake’s character model has been replaced with a dark silhouette of himself in Dark Mode, which makes navigating through each room even more challenging. Because levels in Dark Mode require quick timing, there are no phone calls to answer in this mode.

 

Dark Mode is unlocked once the player has beaten the game once. After doing this, the player will be able to select either Light Mode or Dark Mode when beginning a new game.

 

Replay Mode
Selecting the map at the bottom-right of the Main Menu allows the player to replay any previously-reached level in the game. As levels are reached in Light Mode, room numbers are added to the map. These numbers can then be selected to replay that level. Replay Mode is useful for revisiting previously-beaten levels, or practicing a level in order to be able to complete it in Dark Mode.

 

Although Replay Mode is available from the start of the game, no levels can be accessed from the map until they are reached in Light or Dark Mode first. Once Dark Mode has been unlocked, the player can toggle between light and dark maps at the top-right of the map, to replay previously-reached levels in either mode.
