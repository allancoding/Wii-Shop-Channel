¡Bienvenido!

● ¿Qué es Actionloop Twist?
Actionloop Twist es un juego de inteligencia y reflejos en el que deberás juntar tres o más gemas del mismo color para hacerlas desaparecer. Intenta conseguir la mayor puntuación posible, o superar la prueba correspondiente, antes de que las gemas caigan por la tronera.

Cañón

● ¡Gira, apunta y dispara!
Gira el mando de Wii para hacer rotar el cañón y pulsa para lanzar tus gemas. Tu rapidez y destreza a la hora de lanzar las gemas serán clave para lograr superar las pruebas.

Se pondrá a prueba tu capacidad para tomar decisiones rápidas.
