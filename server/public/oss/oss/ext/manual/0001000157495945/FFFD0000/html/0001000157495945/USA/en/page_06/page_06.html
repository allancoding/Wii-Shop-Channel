Starting the Game

Select the game from your Wii Menu. The Channel Preview screen will load. To launch the episode, aim the Pointer at Start and press the A Button. This brings up the main menu.


Main Menu

The main menu gives you the following options:

  • Save & Load: Save a game in progress, or open a saved game.
  • New Game: Start the game from the beginning.
  • Settings: Change the hint level, turn subtitles or pop-up text on/off and adjust the audio settings.
  • Quit: Exit the game and return to the Wii Menu.
  • Resume Game: Return to your current gaming session.
To do any of these things, point at the appropriate heading and press the A Button.



New Game

Point at New Game and press the A Button. When prompted, select and press the A Button on Yes, I am! to start your adventure, or Oops, no! if you wish to return to the main menu.


Save & Load

Point at Save & Load and press the A Button. It's always important to save your progress. The game has an auto save feature that kicks in at important points during the game. Each auto save rewrites the one before it, so if you want to be able to go back and replay different parts of the game later on, remember to save in one of the other slots.


To manually save or load your game, access the Save & Load menu, press the A Button to scroll left or right with the arrows, select a number slot, and press the A Button on either Save Game, or Load Game. Select Back or press the B Button to return to the main menu.


Settings

Point at Settings and press the A Button. In this screen you can change your game settings. You can alternate between Gameplay Settings and Audio Settings by pointing at the arrows and pressing the A Button. Select Back or press the B Button to return to the main menu.


Gameplay Settings
  • Hint Frequency: Tales of Monkey Island has a built-in hint feature that kicks in if you get stuck. Using the A Button, move the hint slider to the right for maximum hints, or leave it closer to the left for fewer hints. You can change the hint level at any point while playing.
  • Pop-up Text: Descriptive pop-up text displays on screen when you move the pointer over an object or person that Guybrush can interact with. Use the A Button to turn this off if you don't want to see pop-up text displayed.
  • Subtitles: Switch between having the subtitles on or off, by using the A Button.

    • Audio Settings

    Point at the arrows and press the A Button to individually control the audio settings.

    • Music: Sets the volume of the game's background music.
    • Voice: Sets the volume of the spoken dialogue.
    • Sounds: Sets the volume of the game's sound effects (such as footsteps).

    Note: Certain game puzzles use sound effects, so turning the sounds off completely is not recommended.


    Quit

    Point at Quit and press the A Button. When prompted, select and press the A Button on Yes, quit! to exit to the Wii Menu, or No, don't! if you wish to return to the main menu.
