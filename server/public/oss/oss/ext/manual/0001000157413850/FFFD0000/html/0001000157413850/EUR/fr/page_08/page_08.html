Menu principal
  • Ecran du menu principal
    Appuyez sur à l’écran titre pour afficher le menu principal. Choisissez un des modes de jeu disponibles.
    Note: le nombre de stages et de modes de jeu disponibles peut augmenter selon votre progression. Les éléments venant d’être débloqués sont indiqués par le symbole .
  • STAGES
    En mode stages, la taille maximale requise pour tous les tentacules varie en fonction du stage. Utilisez pour passer en revue les stages disponibles. Chaque colonne correspond à un niveau, déterminé en fonction du nombre de tentacules de l’organisme. Pour chaque niveau, les stages disponibles apparaissent les uns en dessous des autres. Sélectionnez un stage, puis appuyez sur pour le commencer.
  • INFINI
    Faites grandir autant que possible deux tentacules ou plus pour obtenir un score élevé. Sélectionnez le nombre de tentacules de votre choix, puis appuyez sur pour commencer à jouer.
  • SERPENT
    Faites grandir autant que possible votre unique tentacule pour obtenir un score élevé. Appuyez sur pour commencer à jouer.
    Note: progressez dans le mode stages pour débloquer des images de fond (thèmes) pour le mode serpent.
  • TUTORIEL
     Contient les instructions des modes stages, serpent et infini.
  • OPTIONS
    Appuyez sur gauche sur pour diminuer la luminosité de l’écran, ou sur droite sur pour l’augmenter.
