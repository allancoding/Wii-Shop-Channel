Introduction
  • Penta Tentacles est un jeu dans lequel vous contrôlez un organisme devant absorber d’autres organismes se trouvant dans son environnement afin d’augmenter la taille de ses tentacules.
  • Modes de jeu
    Il existe trois modes de jeu différents.

Attention

Vous risquez de perdre vos données ou d’endommager votre console Wii si vous effectuez les actions suivantes pendant une sauvegarde automatique:
  • En appuyant sur le bouton POWER de la console ou de la télécommande Wii, ou sur le bouton RESET pendant que les données sont sauvegardées.
  • En ouvrant le menu HOME et en sélectionnant MENU Wii ou REINITIALISATION pendant que les données sont sauvegardées.
  • En débranchant le bloc d'alimentation alors que la console Wii est allumée.
Faites attention, les données effacées ne peuvent pas être récupérées.
