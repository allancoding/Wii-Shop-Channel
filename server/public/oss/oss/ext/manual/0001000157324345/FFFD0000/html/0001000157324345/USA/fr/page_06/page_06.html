Commandes

Dans les menus
Télécommande  Wii :

Pointez l'icône que vous voulez sélectionner avec la télécommande Wii.
Sélectionner.
Retourner à l'écran précédent.
Ouvrir le Menu HOME.
 

Menus de pause
Télécommande  Wii :

Pointez l'icône que vous voulez sélectionner avec la télécommande Wii.
Sélectionner.
Retourner au jeu.
Ouvrir le Menu HOME.
 

Ecran en jeu

Télécommande  Wii ou

Choisissez votre réponse avec le curseur ou la manette + sur la télécommande Wii.
Sélectionner.
Menu de pause.
Ouvrir le Menu HOME.
 

Sur les écrans de bilans cérébraux
Page suivante.
Retourner à l'écran précédent.
Changer de tableau.
Ouvrir le Menu HOME.
 

Sur l'écran du jeu Tigre à vélo
Télécommande  Wii :

Tenez la télécommande Wii horizontalement et secouez-la pour faire avancer le tigre.
Choisir une réponse.
Sélectionner.
Menu de pause.
Ouvrir le Menu HOME.
 

Sur l'écran du jeu Panda funambule
Télécommande  Wii :

Tenez la télécommande Wii horizontalement et inclinez-la pour redresser le panda.
Choisir une réponse.
Sélectionner.
Menu de pause.
Ouvrir le Menu HOME.
