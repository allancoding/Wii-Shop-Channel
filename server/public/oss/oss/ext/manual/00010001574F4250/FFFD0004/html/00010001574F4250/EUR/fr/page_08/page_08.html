Commencer

Sélectionnez le titre Orbient dans le menu Wii, puis choisissez DEMARRER sur l’écran de présentation de la chaîne. L’écran de sélection des fichiers apparaîtra alors. Choisissez un des trois fichiers en pointant dessus avec la télécommande Wii et appuyez sur . On vous demandera alors si vous souhaitez créer des données de sauvegarde. Acceptez en sélectionnant YES (oui). Les bases du jeu vous seront alors expliquées, puis vous passerez à l’écran titre.

Si vous appuyez sur sur l’écran titre, vous vous retrouverez sur l’écran de sélection des galaxies, où vous pourrez choisir la galaxie dans laquelle vous souhaitez jouer. La première fois que vous jouez, vous ne pouvez sélectionner que la galaxie située en haut de l’écran.

Une fois la galaxie sélectionnée, vous vous rendrez sur l’écran de choix du niveau. Faites votre choix parmi les cinq niveaux de la galaxie. La première fois que vous jouez, vous ne pouvez sélectionner que le niveau situé dans le coin supérieur gauche.

Le jeu commence après la sélection du niveau!

Terminer tous les niveaux

Terminez tous les niveaux afin d’en débloquer d’autres. Si vous remplissez certains objectifs, vous aurez accès à encore plus de niveaux. Essayez de tous les réussir!

Sauvegarder

Vous risquez de perdre vos données ou d’endommager votre console Wii si vous effectuez les actions suivantes lorsque vous utilisez la fonction de sauvegarde.

● En appuyant sur le bouton POWER de la console ou de la télécommande Wii, ou sur le bouton RESET pendant que les données sont sauvegardées.

● En ouvrant le menu HOME et en sélectionnant MENU Wii ou REINITIALISATION pendant que les données sont sauvegardées.

● En débranchant le bloc d'alimentation alors que la console Wii est allumée.

Faites attention, les données effacées ne peuvent pas être récupérées.
