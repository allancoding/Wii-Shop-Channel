Wii Balance Board Setup

This software supports the Wii Balance Board. For setup and registration instructions, as well as important health and safety precautions, please see the Wii Balance Board Operations Manual.

 

Playing using the Wii Balance Board

 

To play using the Wii Balance Board, you will need to register your Wii Balance Board to your Wii console. In this software, you can register a Wii Balance Board on the Control Selection Screen, accessible by selecting OPTIONS from the Title Menu.

 

If the Wii Balance Board is not synchronised to your Wii console:

 

Wii Balance Board
Wii console

You will now need to carry out a Standard Mode synchronisation for the Wii Balance Board.
Remove the battery cover on the underside of the Wii Balance Board and open the SD Card slot cover of the Wii console.

When removing the battery cover, be careful not to drop the Wii Balance Board, or trap your fingers or hands in any way.

Make sure the batteries are set properly in the Wii Balance Board.

 

Wii Balance Board

Press  on the Wii Balance Board. The Wii Balance Board’s Power LED will blink.

 

Once the Wii Balance Board’s Power LED has started blinking, stop pressing  on the Wii Balance Board.

 

Wii console

Press  on the Wii console while the Wii Balance Board’s Power LED is blinking. The Wii Balance Board’s Power LED will light up once the Wii Balance Board has been synchronised with the Wii console.

 

IMPORTANT: Do not hold down  on the Wii console for more than 10 seconds.
This will erase the registration of all Wii Remotes that have been registered using Standard Mode synchronisation.

 

Be sure to close the SD Card slot cover of the Wii console and replace the battery cover of the Wii Balance Board.

 

Only one Wii Balance Board can be synchronised with a given Wii console using Standard Mode synchronisation and, including Wii Remotes, up to 10 controllers / accessories can be synchronised. If 11 or more Standard Mode synchronisations are performed, the oldest Wii Remote registration data will be deleted, but the Wii Balance Board data will not be deleted.

Only one Wii console can register a Wii Balance Board using Standard Mode synchronisation. If a registered Wii Balance Board is then synchronised with another Wii console using Standard Mode synchronisation, the settings of the Wii Balance Board will be overwritten. To re-use it with the initial Wii console, it will be necessary to perform a Standard Mode synchronisation again.

The Wii Balance Board will be connected as Player 4 (P4).

 

Note: If a Wii Remote is connected to P4, that Wii Remote will be disconnected.
