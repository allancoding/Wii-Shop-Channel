Contrôle du Jeu

bouton A : Bouton d’action. Utilisé pour valider toutes les actions, comme faire exploser une bombe, mettre en place un piège ou bien séparer des connecteurs. On peut aussi utiliser le bouton A pour manipuler des objets dans le niveau ou directement en plaçant le curseur sur l’élément et en maintenant enfoncé le bouton pour le placer dans le niveau puis en appuyant à nouveau sur le bouton A pour valider la position.
bouton B : Bouton de connexion. Ce bouton permet de parcourir les différents connecteurs disponibles pouvant être utilisés pour créer des pièges ou déplacer des éléments.
bouton C : Bouton d'utilisation d'ampoules. Ce bouton permet de sélectionner les ampoules (à jeter ou les objets bonus), ou la main. Jeter une ampoules sur un bloc, un monstre ou un élément en utilisant le bouton A.
bouton Z : Bouton d’objet. On peut utiliser ce bouton pour choisir entre les objets disponibles du téléporteur et les objets du niveau. Les objets sélectionnés peuvent être placés dans le niveau en utilisant le bouton A.
bouton HOME : Ce bouton ouvre et ferme le menu HOME.
bouton - : Permet de Zoomer / De-zoomer. Ce bouton zoom sur le niveau et permet d’avoir plus de précision pour des actions ou juste de voir de plus près certains objets et/ou montres.
bouton + : Bouton de Pause. Ce bouton met le jeu en pause ou quitte la pause.
manette + : Appuyer haut pour sélectionner les ampoules, Gauche pour sélectionner les objets, Droite pour sélectionner les connecteurs et Bas pour sélectionner les objets du niveau.
Contrôle de la Camera : La camera peut être contrôlée soit en déplaçant le curseur sur les bords de l’écran soit en utilisant le Stick directionnel sur le Nunchuk.
