Sauvegarder
Le jeu est doté d'une fonction de sauvegarde automatique, ce qui signifie que les données sont automatiquement sauvegardées à chaque fois qu'un combat se termine ou que les paramètres sont modifiés. Si tu veux supprimer des données, vas dans l'écran de gestion des données de la console. Mais attention : ces données seront irrécupérables. Les données sauvegardées sur la carte SD ne peuvent pas être copiées.

Informations sur la sauvegarde

Les actions suivantes en cours de sauvegarde peuvent entraîner une perte définitive des données. Alors fais bien attention !

● Appuyer sur le bouton RESET ou POWER de la console Wii ou de la télécommande Wii pendant la sauvegarde.

● Sélectionner « Menu Wii » ou « Réinitialisation » dans le menu Home pendant la sauvegarde.

● Débrancher le cordon AC alors que la console Wii est sous tension.
