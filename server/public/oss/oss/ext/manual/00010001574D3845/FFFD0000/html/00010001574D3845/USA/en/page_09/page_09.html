Normal Battle Setup
Normal Battle is played as follows.

 

Players
Choose how many people will play, and set the level of difficulty for COM characters to "Easy," "Normal," or "Hard." Choose "Empty" for 1 less character to participate.

 

Characters
Press right or left to select users registered via "User Information." If you do not select any users, the following default characters will be used.
             

* You can also select Guest Miis, but battle results will not be saved when using them.

* A Mii saved to the Wii Remote in use can only be selected when a Guest Mii is selected.

 

Rules
Choose the rules of battle from among 6 different types.

* When playing a Ranked Battle in Nintendo Wi-Fi Connection Battle mode, you cannot select "TEAMS" or "SURPRISE."

 

Teams
You can only make teams when selecting "TEAMS" at the Rules screen.

 

Stages
Select a stage you'd like to play in, and press up or down to select a map size.

 

Settings
Press up, down, left, or right to select various battle settings, then press to confirm your choices.
Battle Type

● Trophies
Whoever wins the most rounds out of the designated number of rounds gets the victory.

● Score
Victory goes to whoever earns the most points by performing various actions during battle.
Rounds
(1-9)		 When playing Trophies, select the number of rounds to be won for final victory. When playing Score, select the total number of battles.
Minutes
(1-9)		 Select a time limit of 1 to 9 minutes per battle.
Start Point
(Fixed/Random)		 Select the area your character starts out in for each battle. Choosing "Fixed" will designate a fixed starting point for Players 1-8. Choosing "Random" will have each player start out in a different area for each battle.
Pressure Blocks
(ON/OFF)		 When time has almost run out, Pressure Blocks will start falling from the sky to make the stage smaller and smaller.
Revenge
(ON/OFF)		 Toggle the Revenge function ON or OFF.
Item Panels
(Burnable/Fireproof)		 Decide whether or not items can be destroyed by bomb blasts.
Skulls
(ON/OFF)		 Decide whether or not Skulls appear during battle.

 

Final Check
Run a final check of the settings you've made, then set items and handicaps.
Start Start playing a battle.
Handi Set various handicaps to be in effect once the battle starts. Press to return to default settings. Your character can be hit by a blast once for each heart in possession before dying.
Items Set the types of items to appear during battle. When selecting SAVE, the designated types of items will appear every time a battle is played. Press to return to default settings.

 



 

Results & Winner Announced
Once a battle ends the Results screen will be displayed. Total wins are shown when playing Trophies, and total points are shown when playing Score. A final winner is announced once the set number of rounds have all been played.
Quit
 Exit Normal Battle mode.
Settings Go back to the Rules screen.
Try Again Play another battle with the same settings.
