Información de usuario
Los usuarios pueden registrarse para las Batallas Conexión Wi-Fi de Nintendo y consultar los récords de batalla.

 

Usuarios
Selecciona una ranura vacía para registrar a un nuevo usuario o cambiar la configuración de un usuario ya registrado.
Ver/Editar Comprueba o edita datos de usuarios registrados.
Borrar Elimina datos de un usuario registrado.

 

Datos usuario
Al seleccionar Ver/Editar en la pantalla Usuarios, aparece el menú siguiente.
Personaje Se puede seleccionar cuando la función Mii está desbloqueada. Selecciona si usarás un Mii.
Apodo Crea o cambia el apodo de usuario.
Pose de acción Establece una pose de acción para hacerla en batalla al pulsar .
Aviso de acción Establece los avisos de acción. Estos mensajes aparecen durante las batallas de amigos en el modo Batalla Conexión Wi-Fi de Nintendo.
Pueden fijarse cuatro mensajes diferentes, que se mostrarán en tres situaciones distintas: durante la batalla, o cuando ganas o pierdes una batalla. Pulsad urante la partida para que aparezca un aviso de acción, o mantén pulsado y cambia a un mensaje diferente.
Récords Comprueba tu récord de victorias/derrotas en los modos Combate normal o Batalla Conexión Wi-Fi de Nintendo.
Tabla Consulta tu clasificación en el modo Batalla Conexión Wi-Fi de Nintendo.
