Pantallas
Esta pantalla muestra una batalla en curso.
 Minutos
  El tiempo que queda para terminar el combate.

 Victorias
  Las victorias se muestran en el modo "Trofeos" y la clasificación en el modo "Puntuación".

 Bomberman
  Este es el personaje del jugador.

 Venganza
  Cuando esté activada la Venganza, puedes seguir lanzando bombas después de explotar. Si le das a alguien, recuperarás la vida. Pulsa para moverte y para tirar bombas. Mantén pulsado mientras usas para moverte más deprisa. Si sueltas después de mantenerlo pulsado un rato, lanzarás una bomba más lejos de lo normal.

 Bomba
  Esta bomba ha sido activada y explotará cuando pase un tiempo determinado.

 Explosión de bomba
  Las llamas de una bomba que explota.
Recibir el impacto de la explosión de una bomba significa la muerte.

 Bloque duro
  No se puede romper con la explosión de una bomba.

 Bloque blando
  Se puede romper con la explosión de una bomba. Al destruirlo es posible que aparezca un objeto.

 Obj.
  Coge un objeto para usar sus efectos.
