Paramètre ami
Lorsque tu sélectionnes Paramètre ami, tu accèdes aux deux menus de code d'ami suivants.
Code d'ami Sélectionne cette option pour afficher ton propre code d'ami. Celui-ci n'est disponible que quand tu es connecté à la connexion Wi-Fi Nintendo.
Sélectionne Combat Connexion Wi-Fi Nintendo pour te connecter.
Liste d'amis Tu peux ici sélectionner des emplacements vides pour enregistrer les codes amis communiqués par tes amis.

Qu'est-ce qu'un « code d'ami » ?

Un « code d'ami » est un code à 12 chiffres automatiquement créé quand tu te connectes à la connexion Wi-Fi Nintendo. Échange ton code d'ami avec des joueurs du monde entier puis enregistre-les dans le menu Paramètre ami.
Tu pourras alors les affronter dans des combats Connexion Wi-Fi Nintendo.
