Pause Menu

Press to display the Pause Menu. From the Pause Menu you can return to the Title Menu or Terminal, or take a picture. Of course, while the Pause Menu is open, the game is paused.

The Pause Menu that appears from the Terminal is different from the Pause Menu that appears in the Stages and the Battle Royale.

If you’re in a Stage and you use the Pause Menu to return to the Terminal, any befriended Pokémon and you collected in that Stage will be lost.

You cannot display the Pause Menu using the Nunchuk.

 

CONTINUE

Continue the game.

GO TO TITLE MENU

Go to the Title Menu. This option only appears in the Terminal.

TO THE TERMINAL

Return to the Terminal. This option only appears in a Stage or in the Battle Royale.

Caution: If you return to the Terminal from the Pause Menu, you will forfeit all befriended Pokémon and  you’ve earned in the Stage so far.

TAKE A PICTURE

Take a picture of the game screen. You can save pictures you’ve taken to an SD Memory Card, but not to the Wii system memory.

SD CARD

You can delete all Pokémon Rumble pictures saved to the SD Memory Card.

MULTIPLAYER

Play co-operatively with up to 4 players (see p. 18: Facilities for Co-op Play).

 

 

More about Taking Pictures

If you select TAKE A PICTURE, a picture will be taken of the game screen. Once you’ve checked the picture, you can select YES to save the picture to an SD Memory Card (as a JPEG file) and the filename and file folder will be displayed.

The Pause Menu will not appear in pictures.

You can view your pictures in the Photo Channel or on your home PC.

 

The copyrights to all pictures taken in Pokémon Rumble belong to The Pokémon Company and other parties. Please do not use them outside of copyright laws.

 

Deleting Pictures

To delete the pictures saved to the SD Memory Card, insert the SD Memory Card into the Wii console and select SD CARD → DELETE PICTURES.

If you select DELETE PICTURES, all images saved to the SD Card will be deleted, not just the pictures from this game. It is not possible to delete images one by one.

 

 

Caution Regarding SD Memory Cards

 

You can use the following storage media: SD Memory Card (includes SDHC Memory Card, miniSD Card, microSD Card, miniSDHC Card, and microSDHC Card).

The above-listed miniSD Card, microSD Card, miniSDHC Card, and microSDHC Card must be used with an appropriate Card Adapter. Failure to use an appropriate Adapter with these cards will make it impossible to remove them from the Wii console, and is a cause of system failure.

Nintendo recommends the use of an officially licensed SD Card. While any SD Card can be used, Nintendo only guarantees the functionality of officially licensed Nintendo products.

When saving the Pokémon Rumble pictures you have taken to your home PC or to the Wii console, copy the pictures without changing the directory names or filenames on the SD Memory Card. Changing the file structure or filenames on the SD Memory Card will make the images unrecognisable to the Wii console.
