Le Menu Principal

Lorsque le jeu commence, l’écran titre et le Menu Principal s’affichent. Naviguez dans le Menu Principal avec  et validez avec .

 

NOUVELLE PARTIE

Pour commencer le jeu depuis le début, choisissez un fichier de sauvegarde indiquant (AUCUNE) pour créer un nouveau fichier. La partie commence immédiatement après.

 

CONTINUER

Cela vous permet de continuer une partie sauvegardée. Choisissez le fichier de sauvegarde avec lequel vous désirez jouer.

 

OPTIONS

Régler les vibrations vous permet de régler les vibrations de la manette Nintendo GameCube.

Effacer la sauvegarde vous permet d’effacer le fichier de sauvegarde de votre choix.

 

Note: pour régler les vibrations de la télécommande Wii, consultez l’écran des paramètres de la télécommande Wii dans l’écran du menu HOME.

 

Attention: il n’est pas possible de récupérer un fichier de sauvegarde qui a été effacé.
