Title Menu (titelmenu)

Als je het spel opstart, wordt het Title Menu (titelmenu) weergegeven. Met  kun je OPTIONS (opties) selecteren. Druk op om te bevestigen.

 

NEW GAME (nieuw spel)

Als je een spel vanaf het begin wilt spelen, selecteer je een vakje voor SAVE DATA (spelgegevens opslaan) waar NONE (geen) staat. Er wordt een nieuw bestand met opgeslagen spelgegevens aangemaakt. Zodra het spelbestand is aangemaakt, begint het spel.

 

CONTINUE (verdergaan)

Laad het spelbestand om verder te gaan met het spel. Selecteer het spelbestand dat je wilt gebruiken.

 

OPTIONS (opties)

De RUMBLE FEATURE SETTING (instelling trilfunctie) regelt de trilfunctie voor de Nintendo GameCube Controller. Je kunt deze ON/OFF (aan/uit) zetten. Selecteer DELETE SAVED DATA (opgeslagen spelgegevens wissen) om de opgeslagen spelgegevens te wissen.

Opmerking: als je de trilfunctie op de Wii-afstandsbediening wilt instellen, selecteer je INSTELLINGEN Wii-AFSTANDSBEDIENING in het HOME-menu.

Let op: eenmaal gewiste gegevens kunnen niet worden teruggehaald, dus wees hier voorzichtig mee.
