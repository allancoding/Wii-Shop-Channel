Facilities for Pokémon

Recruit Point

  Use , passwords, or Tickets to recruit a Pokémon. To exit from this facility, select QUIT.

 

Recruit Using

Use  to befriend a Pokémon. Select RECRUIT USING and a Pokémon will appear. Touch this Pokémon to befriend it.

Caution: The Pokémon will disappear after a period of time.

Note: The amount of  required to befriend a Pokémon is different for each Rank.

 

Recruit with Passwords

You can recruit a Pokémon using a password. Enter the 8-character password using the numbers 0–9, and then select OK. If the password is correct, you will recruit a Pokémon. Each password can be used only once per Save Data.

 

Recruit with Tickets

When you release Pokémon at the RELEASE POINT, you may sometimes receive a Ticket with a Pokémon’s name written on it. If you have a Ticket, this menu will appear.

You can recruit the Pokémon whose name appears on the Ticket. The Ticket will disappear after you’ve used it.

 

Release Point

Release a befriended Pokémon. The screen is the same as the Switch Menu (see p. 11: Switch Menu). Press to select and to confirm. You may select more than one Pokémon. Select YES at the end to release the selected Pokémon. Released Pokémon will leave behind   as a Goodbye Gift.

Caution: Released Pokémon cannot be returned.

Note: The amount of  in the Goodbye Gift depends on the number of released Pokémon and their Power.

Tickets

Sometimes, depending on the type and number of Pokémon released, a Ticket with a Pokémon’s name written on it may be left as a Goodbye Gift. Use the RECRUIT USING TICKET option at the RECRUIT POINT, and you can recruit the Pokémon whose name appears on the Ticket.

Note: You may hold only one Ticket at a time.

Training Point

Use  to teach new moves to the Pokémon you’ve befriended. Moves are selected at random.

Pokémon can remember only two moves at a time. If your Pokémon already knows two moves, you will need to exchange moves.

As you advance in Rank, training costs more , but the types of moves available will increase, and you can learn stronger moves that have more icons.
