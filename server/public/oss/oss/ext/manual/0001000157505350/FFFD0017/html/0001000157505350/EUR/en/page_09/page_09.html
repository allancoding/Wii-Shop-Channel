How to Save Data

Your game progress will automatically be saved to the Wii system memory when you move from a Stage or Battle Royale to the Terminal or use some of the Terminal’s special facilities. While the data is being saved, the message SAVING will appear in the lower left corner of the screen.

Note: Only three Save Data files can be created.

 

Erasing All Save Data

If you erase the Save Data file saved to the Wii system memory, all Save Data files will be erased. For more information, please see the “Wii Settings and Data Management” section of your Wii Operations Manual: Channels and Settings.

Note: To erase only one Save Data file at a time, select OPTIONS (see p. 8: Options) from the Title Menu.  

 

Caution: Erased Save Data files cannot be restored.

 

Quitting the Game

Press to open the HOME Menu, then select Wii MENU.

Warning

Performing any of the following operations can prevent your Save Data file from saving or cause it to become corrupted. If that happens, the Save Data file cannot be restored, so please take care.

Pressing the POWER Button or the RESET Button on the Wii console or Wii Remote while the data is saving.

Pulling the AC plug from the electrical outlet while playing the game.
