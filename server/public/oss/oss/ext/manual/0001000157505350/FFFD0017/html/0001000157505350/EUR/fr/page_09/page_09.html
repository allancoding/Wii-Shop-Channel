 Les fichiers de sauvegarde

Votre progression dans le jeu est automatiquement sauvegardée dans la mémoire de la console Wii chaque fois que vous rentrez au Terminal depuis un Stage ou depuis la Bataille Royale, ou encore lorsque vous effectuez certaines actions dans le Terminal. Pendant la sauvegarde, SAUVEGARDE s’affiche en bas à gauche de l’écran.

Note: vous pouvez créer jusqu’à 3 fichiers de sauvegarde.

 

Effacer tous les fichiers de sauvegarde

Si vous effacez toutes les données depuis l’écran de gestion des données de la console Wii, vous effacez également tous les fichiers de sauvegarde. Pour plus de renseignements, consultez la partie « Paramètres Wii et gestion des données » du mode d’emploi Wii – Chaînes et paramètres.

Note: pour effacer vos fichiers un à un, choisissez Options dans le Menu Principal (voir p. 8: Options). Attention: il n’est pas possible de récupérer un fichier de sauvegarde qui a été effacé.

 

Quitter le jeu

Affichez l’écran du menu HOME avec  puis sélectionnez MENU Wii.

 

Attention!

Il est possible que votre progression dans le jeu ne soit pas enregistrée ou qu’un fichier de sauvegarde soit perdu si vous faites l’une des actions suivantes. Les données perdues ne peuvent pas être récupérées.  

Appuyer sur le bouton RESET ou le bouton POWER de la console Wii ou de la télécommande Wii pendant qu’une sauvegarde est en cours.

Débrancher la prise CA pendant que vous jouez.
