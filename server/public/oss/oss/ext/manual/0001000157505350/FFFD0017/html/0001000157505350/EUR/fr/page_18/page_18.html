Installations relatives au Mode Coop.

Vous pouvez commencer le Mode Coop. en utilisant l’installation MULTIJOUEUR située dans le Terminal, ou en choisissant l’option MULTIJOUEUR dans le Menu de pause (voir p. 16: Menu de pause). Le Mode Coop. est identique dans les deux cas.

 

MULTIJOUEUR

Jusqu’à 4 joueurs peuvent jouer simultanément en Mode Coop.

Toutefois, vous avez besoin d’avoir suffisamment de Pokémon et de manettes pour tous les joueurs. Il faut donc que vous ayez réuni suffisamment de Pokémon dans les Stages ou que l’un des joueurs utilise la fonction IMPORT/EXPORT pour copier l’un de ses Pokémon dans sa télécommande Wii.

 

En Mode Coop., si vous branchez un Nunchuk ou une manette classique sur une télécommande Wii, un joueur peut utiliser la télécommande Wii tandis qu’un autre peut utiliser la manette d’extension, vous permettant ainsi de jouer à 2 en même temps.

 

Commencer le Mode Coop.

Approchez-vous de l’installation MULTIJOUEUR, appuyez sur puis choisissez OUI, ou choisissez MULTIJOUEUR dans le Menu de pause.

 

Synchronisez une télécommande Wii avec la console Wii, branchez un Nunchuk ou une manette classique sur une télécommande Wii déjà synchronisée, ou encore branchez une manette Nintendo GameCube dans la console Wii. Vous pourrez alors vérifier à l’écran le nombre de manettes utilisables.

 

Pour participer, appuyez sur (pour une télécommande Wii), (pour un Nunchuk), (pour une manette classique), ou pour une manette Nintendo GameCube.

 

Note: si vous branchez un Nunchuk ou une manette classique à une télécommande Wii, l’écran de confirmation des participants s’affiche.

Un Menu de Changement s’affiche pour chaque joueur. Choisissez le Pokémon que vous voulez contrôler. Attention: vous ne pouvez pas choisir un Pokémon choisi par un autre joueur.

 

Lorsque vous avez tous choisi votre Pokémon, l’écran de confirmation s’affiche. Un joueur n’utilisant pas de Nunchuk doit alors appuyer sur  (ou ) pour finir les préparatifs du Mode Coop.

 

Lorsque le Mode Coop. est prêt, les Pokémon de chaque joueur apparaissent au Terminal. Chaque Pokémon est indiqué par le chiffre et la couleur du joueur qui le dirige.

 

Arrêter le Mode Coop.

Si vous voulez arrêter le Mode Coop. pour jouer seul, ou si l’un des joueurs veut arrêter, les joueurs qui veulent continuer à jouer doivent choisir à nouveau leur Pokémon depuis l’écran MULTIJOUEUR.

Le joueur qui veut arrêter de jouer doit appuyer sur le bouton d'annulation dans le Menu de Changement depuis l’écran MULTIJOUEUR (voir p. 7: Les commandes). Son Menu de Changement disparaît.

Les joueurs qui veulent continuer à jouer doivent choisir leur Pokémon à nouveau. Lorsque l’un des joueurs appuiera sur (ou ), le joueur voulant quitter le Mode Coop. sera retiré de la partie.

 

IMPORT/EXPORT

Cette installation vous permet d’envoyer un de vos Pokémon sauvegardés dans la console Wii vers la télécommande Wii, ou vice-versa.

Grâce à cela, vous pouvez importer l’un de vos Pokémon dans votre télécommande Wii, et l’emmener chez un ami pour jouer en Mode Coop. chez lui en lui donnant votre Pokémon.

Toutefois, il n’est pas possible d’importer ou d’exporter un Pokémon dont la Puissance est limitée par la Clé Mystérieuse.

Une télécommande Wii peut contenir jusqu’à 12 Pokémon.

 

S’il existe d’autres données dans la télécommande Wii (à l’exception des Mii™), vous perdrez ces données.

Les seuls Pokémon que vous pouvez importer ou exporter sont ceux issus de Pokémon Rumble.

 

Attention: les Pokémon exportés vers une télécommande Wii disparaissent de la console Wii dont ils sont issus.
