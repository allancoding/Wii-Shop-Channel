Formato dello schermo e salvataggio
Formato dello schermo
Il formato dello schermo può essere modificato dal menu di sistema della console Wii.
Consulta il manuale di istruzioni della console Wii per maggiori informazioni.
Salvataggio
Il salvataggio è automatico.
Se desideri eliminare questi dati, puoi procedere tramite lo schermo Gestione dati della console Wii. Ricorda che i dati registrati con la Nintendo Wi-Fi Connection non possono essere eliminati.

Salvataggio

Se, durante il salvataggio, compi una delle azioni descritte di seguito, potresti perdere i dati di gioco o danneggiare la console Wii.

● Se premi il pulsante POWER sul telecomando Wii o il pulsante POWER o RESET sulla console Wii durante la scrittura dei dati.

● Se apri il Menu HOME e poi selezioni MENU Wii o RIAVVIA.

● Se scolleghi il blocco alimentatore per Wii dalla console Wii.

Non è possibile recuperare i dati persi, quindi fai attenzione.
