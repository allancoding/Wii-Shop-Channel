Écran d'état
Pointez un animal dans le site et appuyez sur pour activer la caméra travelling.

●Type d'animal
Type de l'animal que vous élevez.　

●Nom de l'animal
Nom que vous avez donné à l'animal.　

●Sexe
Indique si l'animal est un mâle ou une femelle.

●Élevé depuis
Indique depuis combien de jours vous élevez cet animal.

●Caractère
Caractère actuel de l'animal.

●Attachement
Il augmente tous les jours un peu si les caractéristiques Appétit et Énergie de l'animal restent constamment élevées.
Cependant, l'attachement n'augmente pas les jours où le site est sale, même si les caractéristiques Appétit et Énergie sont élevées.

●Énergie
Elle augmente chaque fois que vous caressez ou que vous appelez l'animal.
Mais si son site est sale, l'énergie diminue plus vite.

●Appétit
Cette caractéristique diminue progressivement avec le temps, indiquant que l'animal commence à avoir faim.
N'oubliez pas de le nourrir avant qu'il n'ait le ventre complètement vide.

●Nourri
Indique combien de fois vous avez nourri l'animal.

●Caressé
Indique combien de fois vous avez caressé votre animal.

●Appelé
Indique combien de fois vous avez appelé l'animal.

●Enceinte
Ne s'affiche que pour les femelles qui attendent un petit.


●Père
Indique le nom du père de l'animal. (Cette information ne s'affiche pas pour les animaux qui viennent de l'animalerie.)

●Mère
Indique le nom de la mère de l'animal. (Cette information ne s'affiche pas pour les animaux qui viennent de l'animalerie.)
