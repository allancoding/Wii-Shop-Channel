Qu’est-ce que le systeme PEGI?

 Le systeme de classification par age PEGI (Pan European Game Information, systeme europeen d'information sur les jeux) permet aux parents d'Europe de prendre des decisions eclairees lors de l'achat de jeux video. Le systeme PEGI a ete lance au printemps 2003, remplacant les differents systemes nationaux de classification par un systeme unique desormais utilise dans la plupart des pays d'Europe.

Pour les jeux en vente dans le commerce, la classification est indiquee clairement sur l'emballage. Pour les jeux vendus via la chaine boutique Wii, la classification est affichee sur l'ecran de details du jeu.

Pictogrammes de classification et descripteurs
A chacun des jeux classes selon le systeme PEGI correspond un pictogramme de classification par age et, dans certains cas, un ou plusieurs descripteurs de contenu.

- Classes d'age



Les classes d'age du systeme PEGI sont les suivantes: 3, 7, 12, 16 et 18. Le pictogramme donne une indication de l'age minimal recommande pour utiliser le jeu. La classe d'age n'indique pas le niveau de difficulte du jeu.

- Types de contenu




Les descripteurs sont des pictogrammes indiquant le type de contenu propose dans le jeu. L'importance du contenu dans le jeu depend de la classe d'age: ainsi, le contenu d'un jeu classe "16" associe au descripteur "Violence" peut etre plus violent qu'un jeu classe "7" egalement associe au descripteur "Violence".



 Violence Ce jeu contient des scenes violentes.

Sexe Ce jeu montre des scenes de nudite et/ou des comportements ou des allusions de nature sexuelle.

Langage grossier Ce jeu contient des expressions grossieres.

Drogue Ce jeu illustre ou se refere a la consommation de drogues.

Peur Ce jeu risque de faire peur aux jeunes enfants.


Discrimination Ce jeu contient des images ou des elements susceptibles d’inciter a la discrimination.

Jeux de hasard Ce jeu incite a jouer aux jeux de hasard ou enseigne leurs regles.

Jeu en ligne Ce jeu permet de jouer en ligne avec d'autres personnes.
